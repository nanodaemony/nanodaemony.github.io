## 字符串题目

### 字符串最后一个单词的长度(58-简单) 💯

一个字符串 s 由若干单词组成，单词前后用一些空格字符隔开。返回字符串中 最后一个单词的长度。

```
输入: s = "   fly me   to   the moon  "  输出: 4
解释: 最后一个单词是 moon, 长度为 4.
```

[https://leetcode.cn/problems/length-of-last-word/description/](https://leetcode.cn/problems/length-of-last-word/description/)

**从后往前遍历法**

**从后往前遍历，找到第一个不为空格的字符**，然后持续遍历计数，找到第一个为空格的字符为止。

```
public int lengthOfLastWord(String s) {
    if (s == null) {
        return 0;
    }

    // 从后往前找到第一个非空格字符
    int index = s.length() - 1;
    while (index >= 0 && s.charAt(index) == ' ') {
        index--;
    }

    // 从非空格字符开始计数, 直到遇到空格或者到头
    int count = 0;
    while (index >= 0 && s.charAt(index) != ' ') {
        count++;
        index--;
    }

    return count;
}
```

### 翻转字符串里的单词(151-简单) 💯

给定一个字符串，逐个翻转字符串中的每个单词。反转后的字符串中不能存在前导空格和尾随空格，如果两个单词间有多余的空格，反转后的字符串需要将单词间的空格减少到仅有一个。

```
示例 1:
输入: s = "the sky is blue"  输出: "blue is sky the"
示例 2:
输入: s = "  hello       world  "  输出: "world hello"  解释: 去掉前后空格和中间的多余空格.
```

[https://leetcode.cn/problems/reverse-words-in-a-string/](https://leetcode.cn/problems/reverse-words-in-a-string/)

**切分空格逆序法**

按照空格切分字符串，倒序遍历字符串，遇到空单词则跳过，否则将结果和一个空格添加到 StringBuilder。

```
public String reverseWords(String s) {
    // 删除首尾空格, 按照空格分割字符串
    String[] strs = s.trim().split(" ");
    StringBuilder res = new StringBuilder();

    // 倒序遍历单词列表
    for (int i = strs.length - 1; i >= 0; i--) {
        if (strs[i].isEmpty()) {
            continue;        // 遇到空单词则跳过
        }
        // 添加单词和空格
        res.append(strs[i]).append(" ");
    }
    // 删除尾部空格并转化为字符串返回
    return res.toString().trim();
}
```

**从后往前遍历法**

使用 StringBuilder，从后往前找，找到一个单词就添加到 StringBuilder 中，然后**持续过滤中间的空格**。

```
public String reverseWords(String s) {
    // 移除两端的空格
    s = s.trim();
    // 从后往前遍历找到所有的单词
    int j = s.length() - 1, i = j;
    StringBuilder res = new StringBuilder();
    while (i >= 0) {
        // 寻找一个单词
        while (i >= 0 && s.charAt(i) != ' ') {
            i--;
        }
        // 添加找到的单词&加空格
        res.append(s, i + 1, j + 1).append(" ");

        // 寻找下一个单词的开头 注意这里有可能存在多个空格 所以用while
        while (i >= 0 && s.charAt(i) == ' ') {
            i--;
        }
        // 重置单词的末尾
        j = i;
    }

    return res.toString().trim();
}
```

### 翻转单词序列 🔑

输入一个英文句子，翻转句子中单词的顺序，但**单词内字符的顺序不变**。标点符号需要和普通字母一样处理。

```
输入: "I am a student."   输出: "student. a am I"
```

**单词逐个反转法**

题目应该有一个隐含条件就是不能用额外的空间。虽然 Java 的题目输入参数为 String 类型，需要先创建一个字符数组使得空间复杂度为 `O(N)`，但是正确的参数类型应该和原书一样，为字符数组，并且只能使用该字符数组的空间。

思路是**先旋转每个单词，再旋转整个字符串**。

```java
public String reverseWords(String str) {
    int len = str.length();
    char[] chars = str.toCharArray();
    int i = 0, j = 0;

    // 反转所有单词
    while (j <= len) {
        // 寻找一个单词, 也就是遇到空格或者到达字符串末尾
        if (j == len || chars[j] == ' ') {
            // 反转单词
            reverse(chars, i, j - 1);
            // 更新i指向下一个单词的起始位置
            i = j + 1;
        }
        j++;
    }

    // 翻转整个字符串
    reverse(chars, 0, len - 1);

    return new String(chars);
}

// 仅仅反转一个单词
private void reverse(char[] c, int i, int j) {
    while (i < j) swap(c, i++, j--);
}

private void swap(char[] nums, int i, int j) {
    char temp = nums[i];
    nums[i] = nums[j];
    nums[j] = temp;
}
```

### 左旋转字符串 🔑

字符串的左旋转操作是把字符串前面的若干个字符转移到字符串的尾部。定义一个函数实现字符串左旋转操作的功能。

```
输入: S="abcXYZdef"  K=3    输出: "XYZdefabc"
```

**原地分段反转法**

类似题目：[**轮转数组(189-中等)**](https://www.yuque.com/daerwendecha/qwuv92/gdn6l3zi8fmvfnqx#cliwf)**。**

先将 "`abc`" 和 "`XYZdef`" 分别翻转，得到 "`cbafedZYX`"，然后再把整个字符串翻转得到 "`XYZdefabc`"。

```
public String reverseLeftWords(String str, int n) {
    if (n >= str.length())
        return str;
    char[] chars = str.toCharArray();
    // 先反转前半部分字符串abc
    reverse(chars, 0, n - 1);
    // 再反转后半部分字符串XYZdef
    reverse(chars, n, chars.length - 1);
    // 最后整个字符串一起反转
    reverse(chars, 0, chars.length - 1);
    return new String(chars);
}

private void reverse(char[] chars, int i, int j) {
    while (i < j)
        swap(chars, i++, j--);
}

private void swap(char[] chars, int i, int j) {
    char t = chars[i];
    chars[i] = chars[j];
    chars[j] = t;
}
```

### 重新格式化字符串(1417-简单)

给一个混合了数字和字母的字符串 s。将该字符串重新格式化，使得**任意两个相邻字符的类型都不同**。也就是说，字母后面应该跟着数字，而数字后面应该跟着字母。返回重新格式化后的字符串。如果无法按要求重新格式化，则返回一个空字符串。

```
输入: s = "a0b1c2" 输出: "0a1b2c"
解释: "0a1b2c" 中任意两个相邻字符的类型都不同.  "a0b1c2", "0a1b2c", "0c2a1b" 也是满足题目要求的答案.
输入: s = "leetcode" 输出: ""
解释: "leetcode" 中只有字母, 所以无法满足重新格式化的条件.
输入: s = "ab123" 输出: "1a2b3"
```

[https://leetcode-cn.com/problems/reformat-the-string](https://leetcode-cn.com/problems/reformat-the-string)

**双指针填充法**

**定义两个变量，对数字字符和字母字符计数，如果数量相差 2 及以上，直接不算了**。初始化一个结果数组，使用双指针，**哪种类型的字符数量多哪个就从 0 开始，另一个从 1 开始**，遍历字符串中的字符，根据对应的字符类型填充到对应类型的指针位置，由于字符与数字需要间隔，所以**每次步长加 2**。反正数字字符和字母字符最终都要放进去。

```
public String reformat(String str) {
    // 统计字母与数字的个数
    int letterCnt = 0, numberCnt = 0;
    char[] chars = str.toCharArray();
    // 统计数字与字母的个数
    for (char c : chars) {
        int a = Character.isDigit(c) ? numberCnt++ : letterCnt++;
    }

    // 字母和数字的数量相差超过1则不合法
    if (Math.abs(letterCnt - numberCnt) > 1) {
        return "";
    }

    // 哪个数量多则放到第一个位置
    int letterIndex = 0, numberIndex = 0;
    if (letterCnt > numberCnt) {
        numberIndex = 1;
    } else {
        letterIndex = 1;
    }

    // 结果数组
    char[] res = new char[chars.length];

    // 遍历字符串重新填充对应位置的字符, 填充一个后对应的索引+2
    for (char c : chars) {
        if (Character.isLetter(c)) {
            res[letterIndex] = c;
            letterIndex += 2;
        } else {
            res[numberIndex] = c;
            numberIndex += 2;
        }
    }

    return new String(res);
}
```

### 数组中的字符串匹配(1408-简单)

给你一个字符串数组 words，数组中的每个字符串都可以看作是一个单词。请按任意顺序**返回 words 中是其他单词的子字符串的所有单词**。

```
输入: words = ["mass","as","hero","superhero"]  输出: ["as","hero"]
解释: "as" 是 "mass" 的子字符串, "hero" 是 "superhero" 的子字符串. ["hero","as"] 也是有效的答案.
```

[https://leetcode-cn.com/problems/string-matching-in-an-array](https://leetcode-cn.com/problems/string-matching-in-an-array)

**拼接求解法**

用 StringBuffer **把所有的单词都拼接起来，寻找单词在拼接字符串的前后位置索引，如果两个索引不相同，则说明重复出现了**，也就这个单词是其他字符串的子字符串。

```
public List<String> stringMatching(String[] words) {
    if (words == null) return null;
    List<String> resList = new ArrayList<>();

    // 将所有字符串拼接成一个字符串
    String allStr = Arrays.toString(words);

    // 遍历每个字符串, 如果在拼接的字符串中出现了两次则说明是子串
    for (String word : words) {
        if (allStr.indexOf(word) != allStr.lastIndexOf(word)) {
            resList.add(word);
        }
    }

    return resList;
}
```

### 字符串中的连续字符(1446-简单)

找出字符串中只**包含一种字符**的最长非空子字符串的最大长度。

```
输入: s = "leetcode"  输出: 2  解释: 子字符串 "ee" 长度为 2, 只包含字符 'e'.
```

[https://leetcode-cn.com/problems/consecutive-characters](https://leetcode-cn.com/problems/consecutive-characters)

**遍历计数法**

直接从左到右变遍历记录各个连续字符出现的次数，**遇到不同的字符就更新计数器**。

```
public int maxPower(String str) {
    if (str == null) {
        return 0;
    }
    // 至少有1个字符
    int res = 1;
    // 字符计数器
    int counter = 1;
    // 记录当前的字符
    char temp = str.charAt(0);

    for (int i = 1; i < str.length(); i++) {
        // 如果当前的字符与上一个相同
        if (str.charAt(i) == temp) {
            counter++;
        } else {
            // 不同就更新结果并重置计数器
            res = Math.max(res, counter);
            counter = 1;
            temp = str.charAt(i);
        }
    }

    // 最后一定要更新结果, 防止出现aaaaa这种记录
    res = Math.max(res, counter);
    return res;
}
```

### Z 字形变换(6-中等) 💯

将一个给定字符串根据给定的行数 numRows，以从上往下，从左到右进行 Z 字形排列。比如输入字符串为 "`PAYPALISHIRING`" 行数为 3 时，排列如下：

```
P   A   H   N
A P L S I I G
Y   I   R
```

排列后的输出需要从左往右逐行读取，产生出一个新的字符串，比如："`PAHNAPLSIIGYIR`"。

[https://leetcode.cn/problems/zigzag-conversion/description/](https://leetcode.cn/problems/zigzag-conversion/description/)

**上下循环填充法**

**使用 numRows 个列表来分别存储各个行的字符，然后按照从上到下，再从下到上的方式，持续不断的将字符填充到对应的列表中**。

![](/zimg/yuque-1777880829877-1.png)

另一个示例图：

![](/zimg/yuque-1777880829904-2.gif)

注意这里的编码方式，**使用了一个 flag 来表示遇到了上或者下边界，遇到之后就反向填充**，非常巧妙。

```
public String convert(String s, int numRows) {
    if(numRows < 2) return s;

    // 🌟构造一个List用于存储每一行的所有字符
    List<StringBuilder> resList = new ArrayList<>();
    for(int i = 0; i < numRows; i++) {
        resList.add(new StringBuilder());
    }

    int index = 0, flag = -1;

    // 将字符逐一填充到对应的行
    for(char c : s.toCharArray()) {
        resList.get(index).append(c);
        // 🌟到达边界则反转进行另一个方向的填充
        if(index == 0 || index == numRows -1) flag = - flag;
        index += flag;
    }

    // 将所有行的字符拼接起来
    StringBuilder res = new StringBuilder();
    for(StringBuilder row : resList) {
        res.append(row);
    }

    return res.toString();
}
```

### 替换空格(简单) 🔑

将一个字符串中的空格替换成 "`%20`"。

```
输入: "A B"  输出: "A%20B"
```

**直接替换法**

在字符串尾部填充任意字符，使得字符串的长度等于替换之后的长度。因为一个空格要替换成三个字符 `%20`，因此当遍历到一个空格时，需要在尾部填充两个任意字符。令 P1 指向字符串原来的末尾位置，P2 指向字符串现在的末尾位置。P1 和 P2 从后向前遍历，当 P1 遍历到一个空格时，就需要令 P2 指向的位置依次填充 `02%`(注意是逆序的)，否则就填充上 P1 指向字符的值。

从后向前遍历是为了在改变 P2 所指向的内容时，不会影响到 P1 遍历原来字符串的内容。

![](/zimg/yuque-1777880829968-3.png)

这里用一个 StringBuilder 来进行最终答案的记录。

```
public String replaceSpace(String str) {
    StringBuilder res = new StringBuilder();
    for (int i = 0; i < str.length(); i++) {
        if (str.charAt(i) == ' ') {
            res.append("%20");
        } else {
            res.append(str.charAt(i));
        }
    }
    return res.toString();
}
```

写法 2：

```
public String replaceSpace(String str) {
    int length = str.length();
    // 直接初始化一个原始长度3倍的数组
    char[] array = new char[length * 3];
    // 记录最后需要多少个字符
    int size = 0;
    for (int i = 0; i < length; i++) {
        char c = str.charAt(i);
        if (c == ' ') {
            array[size++] = '%';
            array[size++] = '2';
            array[size++] = '0';
        } else {
            array[size++] = c;
        }
    }
    return new String(array, 0, size);
}
```

### 字符串的最大公因子(1071-简单)

对于字符串 S 和 T，只有在 `S = T + ....+ T`(T 与自身连接 1 次或多次)时，才认定 "T 能除尽 S"。返回最长字符串 X，要求满足 X 能除尽 str1 且 X 能除尽 str2。

```
输入: str1 = "ABCABC", str2 = "ABC" 输出: "ABC"
输入: str1 = "ABABAB", str2 = "ABAB" 输出: "AB"
输入: str1 = "LEET", str2 = "CODE" 输出: ""
```

[https://leetcode-cn.com/problems/greatest-common-divisor-of-strings](https://leetcode-cn.com/problems/greatest-common-divisor-of-strings)

**辗转相除法**

使用辗转相除法求最大公约数。如果两个字符串有公因子 abc，那么 str1 就是 m 个 abc 的重复，str2 是 n 个 abc 的重复，连起来就是 `m+n` 个 abc，因此 `m+n` 个 abc 跟 `n+m` 个 abc 是一样的。

```
public String gcdOfStrings(String str1, String str2) {
    // 假设str1是N个x, str2是M个x, 那么str1+str2是等于str2+str1的. 不满足则没有最大公约数
    if (!(str1 + str2).equals(str2 + str1)) {
        return "";
    }
    // 辗转相除法求最大公约数
    return str1.substring(0, gcd(str1.length(), str2.length()));
}

private int gcd(int a, int b) {
    if (b == 0) {
        return a;
    } else {
        return gcd(b, a % b);
    }
}
```

### 最长公共前缀(14-简单) 💯

编写一个函数来查找字符串数组中的最长公共前缀。如果不存在公共前缀，返回空字符串。

```
输入：strs = ["flower","flow","flight"]  输出："fl"
输入：strs = ["dog","racecar","car"]  输出：""
```

[https://leetcode.cn/problems/longest-common-prefix/description/](https://leetcode.cn/problems/longest-common-prefix/description/)

**逐一对比字符法**

从左到右逐一对比每个字符串在相同位置的字符是否相同，如果不同，或者已经超出某个字符串的范围，则直接结束寻找即可。

```
public String longestCommonPrefix(String[] strs) {
    if (strs == null || strs.length == 0) return "";

    int index = 0;

    // 🌟取第一个字符串的长度进行遍历即可
    for (int i = 0; i < strs[0].length(); i++) {
        // 找到这一轮需要比较的字符
        char c = strs[0].charAt(i);
        for (int j = 1; j < strs.length; j++) {
            // 如果index超过了某个字符串的长度, 或者某个字符串的index位置的字符不等于c, 则直接返回了
            if (index >= strs[j].length() || strs[j].charAt(index) != c) {
                return strs[0].substring(0, index);
            }
        }
        index++;
    }

    return strs[0].substring(0, index);
}
```

### 版本数字比较(165-中等)

比较两个版本号 version1 和 version2。如果 `version1 > version2` 返回 1，如果 `version1 < version2` 返回 -1，除此之外返回 0。

```
输入：version1 = "1.2", version2 = "1.10"    输出：-1
输入：version1 = "1.0", version2 = "1.0.0.0" 输出：0
```

[https://leetcode.cn/problems/compare-version-numbers/description/](https://leetcode.cn/problems/compare-version-numbers/description/)

**逐一比较法**

如果要用 split 函数，需要转义点号。**注意 split 之后，两个数组的长度可能是不一样的，如果已经遍历到头了，则直接用 0 占位表示即可**。

```
public int compareVersion(String version1, String version2) {

    // 注意这里split需要转义.号
    String[] v1 = version1.split("\\.");
    String[] v2 = version2.split("\\.");

    // ⭐两个都遍历完才算完成
    for (int i = 0; i < Math.max(v1.length, v2.length); i++) {
        // ⭐技巧: 缺少位数时, 缺少的就用0表示!!
        int num1 = i < v1.length ? Integer.parseInt(v1[i]) : 0;
        int num2 = i < v2.length ? Integer.parseInt(v2[i]) : 0;

        if (num1 > num2) {
            return 1;
        } else if (num1 < num2) {
            return -1;
        }
    }

    // 全部相等则版本号相同
    return 0;
}
```

### 构造最长回文串(409-简单)

给定一个包含大写字母和小写字母的字符串，找到通过这些字母可以构造成的最长的回文串。请注意区分大小写。比如 "`Aa`" 不能当做一个回文字符串。

```
输入: "abccccdd" 输出: 7 解释:可以构造的最长的回文串是"dccaccd", 它的长度是 7.
```

[https://leetcode.cn/problems/longest-palindrome/description/](https://leetcode.cn/problems/longest-palindrome/description/)

**字符数统计法**

**统计字符串中每个字符出现的次数即可，只要是偶数就能构成回文**。除此之外，中间允许一个数单独出现，比如 "abcba"，所以**如果最后有字母落单，总长度可以加 1**。

首先将字符串转变为字符数组。然后遍历该数组，判断对应字符是否在 Set 中，如果不在就加进去，如果在就让 `count++`，然后移除该字符！这样就能找到出现次数为双数的字符个数。

```
public int longestPalindrome(String s) {
    if (s.isEmpty()) {
        return 0;
    }
    // 用于存放出现过的字符
    HashSet<Character> set = new HashSet<>();
    char[] chars = s.toCharArray();
    int count = 0;

    for (int i = 0; i < chars.length; i++) {
        // 如果hashset没有该字符就保存进去
        if (!set.contains(chars[i])) {
            set.add(chars[i]);
            // 如果有,就让count+2(说明找到了一个成对的字符), 然后把该字符移除
        } else {
            set.remove(chars[i]);
            count += 2;
        }
    }

    // 如果set为空,说明所有字符都是成对的, 直接返回count*2
    // 如果set不为空,说明至少有一个字符是单个的, 那么最后结果就是count*2+1
    return set.isEmpty() ? count : count + 1;
}
```

由于字母是可以穷举的，所以也可以用**数组统计**每个字母出现的次数，另一种写法如下。

```
public int longestPalindrome(String s) {
    if (s.isEmpty()) {
        return 0;
    }

    // 使用数组存放字符出现的次数, 注意这里区分大小写所以不能只用26个数组
    int[] count = new int[128];
    for (char c : s.toCharArray()) {
        count[c - '0']++;
    }

    int res = 0;
    // 是否有奇数个的字符
    boolean hasOdd = false;

    // 遍历数组, 如果是偶数个字符则直接加上, 如果是奇数个字符则加上偶数个字符
    for (int i : count) {
        res += i / 2 * 2;
        // 存在奇数个的字符
        if (i % 2 == 1) {
            hasOdd = true;
        }
    }

    // 如果有奇数个字符则结果+1
    return hasOdd ? res + 1 : res;
}
```

### 最长回文子串(5-中等) 💯

给定一个字符串 s，找到 s 中最长的回文子串。

```
输入: "babad"   输出: "bab"  这里 "aba" 也是一个有效答案
输入: "cbbd"    输出: "bb"
```

[https://leetcode-cn.com/problems/longest-palindromic-substring](https://leetcode-cn.com/problems/longest-palindromic-substring)

**中心扩展法**

面试记住这个是好理解的。

**遍历整个字符串，以各个字符为中心，先找到与当前字符相同的左右区间的范围，因为字符一样所以整个范围一定是回文的，然后以该回文范围向左右两边分别扩展，寻找到最大的回文串**。遍历过程需要维护一下最大回文串的起始位置和最大长度。注意如果当前字符与前一个字符相同，则可以忽略当前字符，因为它们的最长回文串是一样的，这是剪枝操作。

![](/zimg/yuque-1777880829982-4.png)

这里理解这个窗口很重要，**对于一个字符串来说，只要是相同的字符，无论是奇数还是偶数个，都是回文串，所以这个就可以构成一个窗口**。

```
public String longestPalindrome(String str) {

    if (str == null || str.isEmpty()) return "";
    int strLen = str.length();

    // 记录最长回文串
    String resStr = "";

    // 从左到右以每个元素为中心找到其最大的回文串
    for (int center = 0; center < strLen; center++) {

        // 剪枝: 如果当前字符和前一个字符相同则跳过, 因为这种情况已经在前一个字符的时候处理过了
        if (center > 0 && str.charAt(center) == str.charAt(center - 1)) {
            continue;
        }

        // 起初左右都指向center
        int left = center;
        int right = center;

        // 先进行左右扩展, 找到与当前字符相同的字符作为窗口, 然后逐一向两边扩散
        // 往左扩散-找到左边与当前中心字符相同的所有字符
        while (left >= 0 && str.charAt(left) == str.charAt(center)) {
            left--;
        }
        // 右边扩散-找到右边与当前中心字符相同的所有字符
        while (right < strLen && str.charAt(right) == str.charAt(center)) {
            right++;
        }

        // 以当前回文区间为中心向两边扩散
        while (left >= 0 && right < strLen && str.charAt(right) == str.charAt(left)) {
            left--;
            right++;
        }

        // 如果发现最大长度就更新, 这里的区间已经变成了[left + 1, right + 1], 需要注意区间的取值
        if (right - left - 1 > resStr.length()) {
            resStr = str.substring(left + 1, right);
        }
    }

    return resStr;
}
```

对于中心扩展法，还有一种思路就是**分别以一个字符为中心和以两个字符为中心进行中心拓展**，然后寻找最长的回文串。写法如下：

```
public String longestPalindrome(String s) {
    int length = s.length();
    String resStr = "";

    for (int center = 0; center < length; center++) {
        // 这里进行两次中心扩展, 一次是中心节点只有一个字符, 一次是中心节点有两个字符
        for (int j = 0; j <= 1; j++) {
            int left = center;
            int right = center + j;

            // 从中心向两边扩展
            while (left >= 0 && right < length && s.charAt(left) == s.charAt(right)) {
                left--;
                right++;
            }

            // 上一步左右指针已经超出回文范围, 需要各自回溯到回文串的起始和结束位置
            left++;
            right--;

            // 比较并更新最长的字符串起始位置和长度
            if (right - left + 1 > resStr.length()) {
                resStr = s.substring(left, right + 1);
            }
        }
    }

    return resStr;
}
```

**动态规划**

中心拓展法做了很多重复计算。动态规划就是为了减少重复计算的问题。其实就是空间换时间，将计算结果暂存起来，避免重复计算。

用一个 `boolean dp[l][r]` 表示字符串从 i 到 j 这段是否为回文。如果 `dp[l][r]=true`，要判断 `dp[l - 1][r + 1]` 是否为回文，只需要判断字符串在 `(l - 1)` 和 `(r + 1)` 两个位置是否为相同的字符即可，就减少了很多重复计算。

- 初始状态，`l = r` 时，此时 `dp[l][r] = true`。
- 状态转移方程，`dp[l][r] = true` 并且 `(l - 1)` 和 `(r + 1)` 两个位置为相同的字符，此时 `dp[l - 1][r + 1] = true`。
- **滚动更新**最长的结果值即可。

```
public String longestPalindrome(String str) {
    if (str == null || str.length() < 2) return str;
    // 字符串长度
    int strLen = str.length();
    int maxStart = 0;  // 最长回文串的起点
    int maxEnd = 0;    // 最长回文串的终点
    int maxLen = 1;    // 最长回文串的长度
    // dp数组
    boolean[][] dp = new boolean[strLen][strLen];

    // 初始化dp数组, 以i为起点, i为终点的字符串肯定是回文串
    for (int i = 0; i < strLen; i++) {
        dp[i][i] = true;
    }

    for (int right = 1; right < strLen; right++) {
        for (int left = 0; left < right; left++) {
            // 状态转移方程
            if (str.charAt(left) == str.charAt(right) && (right - left <= 2 || dp[left + 1][right - 1])) {
                dp[left][right] = true;
                // 滚动更新结果
                if (right - left + 1 > maxLen) {
                    maxLen = right - left + 1;
                    maxStart = left;
                    maxEnd = right;
                }
            }
        }
    }

    // 根据索引找到最长回文字符串
    return str.substring(maxStart, maxEnd + 1);
}
```

### 回文子串数量(647-中等)

找出一个字符串中有多少个回文子串。具有不同开始位置或结束位置的子串，即使是由相同的字符组成，也会被计为是不同的子串。

```
输入: "abc"  输出: 3  解释: 三个回文子串: "a", "b", "c".
输入: "aaa"  输出: 6  解释: 6个回文子串: "a", "a", "a", "aa", "aa", "aaa".
```

[https://leetcode-cn.com/problems/palindromic-substrings](https://leetcode-cn.com/problems/palindromic-substrings)

**中心扩展法**

由中心往两边扩散。这里分为两种情况：

- 以**一个单独的元素作为中心**往两边扩散，能扩散则找到一个回文串，同时这个元素自己也是回文串。

![](/zimg/yuque-1777880830001-5.png)

- 以**两个相同的相邻元素作为中心**往两边扩散，能扩散则找到一个回文串，如果相邻的元素不相同，则直接忽略，否则这两个元素自己也构成回文串。

![](/zimg/yuque-1777880830017-6.png)

代码如下：

```
public int countSubstrings(String str) {
    if (str == null || str.isEmpty()) return 0;
    int strLen = str.length();

    // 记录回文串的数量
    int count = 0;

    // 以一个元素为中心进行扩散
    for (int center = 0; center < strLen; center++) {
        // 以当前center为中心
        int left = center;
        int right = center;
        // 以当前center为重复串的中心扩散, 能够扩散则说明找到一个回文串
        while (left >= 0 && right < strLen && str.charAt(right) == str.charAt(left)) {
            left--;
            right++;
            count++;
        }
    }

    // 以两个相邻元素为中心进行扩散
    for (int center = 0; center < strLen - 1; center++) {
        // 以两个相邻元素为中心
        int left = center;
        int right = center + 1;
        // 以两个相邻元素为中心往两边扩散 能扩散则找到一个回文串
        while (left >= 0 && right < strLen && str.charAt(right) == str.charAt(left)) {
            left--;
            right++;
            count++;
        }
    }

    return count;
}
```

**动态规划法**

状态：`dp[i][j]` 表示字符串 s 在 `[i, j]` 区间的子串是否是一个回文串。

状态转移方程：当 `s[i] == s[j] && (j - i < 2 || dp[i + 1][j - 1])` 时，`dp[i][j] = true`，否则为 false。

- 当只有一个字符时，比如 `a` 自然是一个回文串。
- 当有两个字符时，如果是相等的，比如 `aa`，也是一个回文串。
- 当有三个及以上字符时，比如 `ababa` 这个字符记作串 1，把两边的 `a` 去掉，也就是 `bab` 记作串 2，可以看出只要串 2 是一个回文串，那么左右各多了一个 `a` 的串 1 必定也是回文串。所以当 `s[i]==s[j]` 时，自然要看 `dp[i + 1][j - 1]` 是不是一个回文串。

```
public int countSubstrings(String str) {
    // dp[i][j]表示从i到j的子串是否是回文串
    boolean[][] dp = new boolean[str.length()][str.length()];
    int ans = 0;
    for (int j = 0; j < str.length(); j++) {
        for (int i = 0; i <= j; i++) {
            // 如果i和j位置的字符相等, 并且j-i<2或者dp[i+1][j-1]是回文串, 则dp[i][j]也是回文串
            if (str.charAt(i) == str.charAt(j) && (j - i < 2 || dp[i + 1][j - 1])) {
                dp[i][j] = true;
                ans++;
            }
        }
    }
    return ans;
}
```
