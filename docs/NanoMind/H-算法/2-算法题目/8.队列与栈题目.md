## 总结

单调栈实际上就是栈，只是利用了一些巧妙的逻辑，使得每次新元素入栈后，**栈内的元素都保持有序(单调递增或单调递减)**。

**使用单调栈或者单调队列时，一般存储的都是\*\***索引值\***\*，而不是具体的数字。因为通过索引值不仅能拿到对应元素的大小，还能进行更多的计算操作，比如计算两个元素的距离**。

### 用栈实现队列(232-简单) 💯

使用栈实现队列的下列操作：

- `push(x)`：将一个元素放入队列的尾部。
- `pop()`：从队列首部移除元素。
- `peek()`：返回队列首部的元素(不删除)。
- `empty()`：返回队列是否为空。

[https://leetcode-cn.com/problems/implement-queue-using-stacks](https://leetcode-cn.com/problems/implement-queue-using-stacks)

**双栈法**

![](/zimg/yuque-1777880833675-1.png)

栈的顺序为后进先出，而队列的顺序为先进先出。准备两个栈 inStack 和 outStack。

- **入队列直接往 inStack 中压入**。
- **出队列时，如果 outStack 中有元素，则直接弹出 outStack 栈顶元素。当 outStack 中没有元素时，再把 inStack 中的元素全部倒入到 outStack 中再进行弹栈，这样可以保证先进先出的顺序**。
- in 栈用来处理入栈(push)操作，out 栈用来处理出栈(pop)操作。一个元素进入 in 栈之后，出栈的顺序被反转。当元素要出栈时，需要先进入 out 栈，此时元素出栈顺序再一次被反转，因此出栈顺序就和最开始入栈顺序是相同的，先进入的元素先退出，这就是队列的顺序。

整体时间复杂度 `O(1)`，空间复杂度 `O(N)`。

```
private Stack<Integer> inStack;
private Stack<Integer> outStack;

private int head;

public MyQueue() {
    inStack = new Stack<>();
    outStack = new Stack<>();
}

public void push(int x) {
    // 直接压入inStack
    inStack.add(x);
}

public int pop() {
    // 边界情况, 都没有元素则抛异常
    if (outStack.isEmpty() && inStack.isEmpty()) {
        throw new IllegalStateException("No element.");
    }

    // 当outStack为空但inStack不为空时, 将inStack中的元素全部转移到outStack中
    if (outStack.isEmpty()) {
        while (!inStack.isEmpty()) {
            outStack.add(inStack.pop());
        }
    }

    // 从out栈弹出栈顶元素
    return outStack.pop();
}

public int peek() {
    // 边界情况, 都没有元素则抛异常
    if (outStack.isEmpty() && inStack.isEmpty()) {
        throw new IllegalStateException("No element.");
    }

    // 当outStack为空但inStack不为空时, 将inStack中的元素全部转移到outStack中
    if (outStack.isEmpty()) {
        while (!inStack.isEmpty()) {
            outStack.add(inStack.pop());
        }
    }

    return outStack.peek();
}

public boolean empty() {
    return outStack.isEmpty() && inStack.isEmpty();
}
```

### 用队列实现栈(225-简单) 💯

使用队列实现栈的下列操作：

- `push(x)`：元素 x 入栈。
- `pop()`：移除栈顶元素。
- `top()`：获取栈顶元素。
- `empty()`：返回栈是否为空。

[https://leetcode-cn.com/problems/implement-stack-using-queues](https://leetcode-cn.com/problems/implement-stack-using-queues)

**双队列法**

类似与上一个题，采用双队列的形式，压栈和弹栈操作都在一个队列进行。比较简单。

**单队列循环法**

![](/zimg/yuque-1777880833807-2.png)

在将一个元素 x 插入队列时，为维护后进先出顺序，需要让 x 插入队列首部。队列的默认插入顺序是队列尾部，因此在将 x 插入队列尾部之后，需要让除了 x 之外的所有元素出队列，然后依次再入队列，也就是换个方向插入到队列末尾。

过程：**每次插入元素的时候都入到队列尾部，然后把前面的元素全部搬到当前元素后面。搬运的元素个数即当前队列的大小**。

```
private Queue<Integer> queue = new LinkedList<>();

public void push(int num) {
    // 记录当前队列的大小
    int curSize = queue.size();
    // 新的元素加入到队列中
    queue.add(num);
    // 每次push的时候都把前面的元素全部搬运到刚插入的元素的后面
    // 之后新加入的元素就在队列最前面了, 也就是栈顶的位置了
    for (int i = 0; i < curSize; i++) {
        queue.add(queue.poll());
    }
}

public int pop() {
    return queue.remove();
}

public int top() {
    return queue.peek();
}

public boolean empty() {
    return queue.isEmpty();
}
```

### 有效的括号(20-简单) 💯

给定一个只包括 `'(', ')', '{', '}', '[', ']'` 的字符串，判断字符串是否有效。有效字符串需满足：左括号必须用相同类型的右括号闭合；左括号必须以正确的顺序闭合；注意空字符串可被认为是有效字符串。注意这个字符串只包含有这几个符号。

```

输入: "([)]"   输出: false
```

[https://leetcode-cn.com/problems/valid-parentheses](https://leetcode-cn.com/problems/valid-parentheses)

**辅助栈法**

利用辅助栈实现匹配。**遇到左括号就压栈，遇到右括号就弹栈，且需要看弹出的栈是否匹配，最后看栈是否为空即可**。剪枝：奇数长度的字符串不符合条件。

![](/zimg/yuque-1777880833832-3.gif)

代码如下：

```
public boolean isValid(String str) {
    Deque<Character> stack = new LinkedList<>();
    // 遍历这个字符串
    for (char c : str.toCharArray()) {
        // 遇到开符号就压栈
        if (c == '(' || c == '{' || c == '[') {
            stack.push(c);
            // 表示遇到闭括号
        } else {
            // 如果此时栈为空则直接返回false
            if (stack.isEmpty()) {
                return false;
            }

            // 弹出栈顶符号
            char temp = stack.pop();
            // 分别判断是否与当前的匹配
            if ((temp == '(' && c != ')')) return false;
            if ((temp == '[' && c != ']')) return false;
            if ((temp == '{' && c != '}')) return false;
        }
    }
    // 最后看栈是否为空
    return stack.isEmpty();
}
```

```
之后得到字符串 "aaca", 其中又只有 "aa" 可以执行重复项删除操作, 所以最后的字符串为 "ca".
```

[https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/](https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/)

**辅助栈法**

本题也是用栈来解决的经典题目。总体思路：**使用辅助栈，遍历字符串，如果当前的字符与栈顶的字符相同，则弹出栈顶元素。否则就将当前字符压栈**。

那么栈里应该放的是什么元素呢?

在删除相邻重复项的时候，其实就是要知道当前遍历的这个元素，在前一位是不是遍历过一样数值的元素，那么如何记录前面遍历过的元素呢? 所以就是用栈来存放，那么栈的目的，就是存放遍历过的元素，当遍历当前的这个元素的时候，去栈里看一下是不是遍历过相同数值的相邻元素。注意这里题目是说的一次性删除 2 个元素，而不是把所有相邻的元素都删了。

然后再去做对应的消除操作。如动画所示：

![](/zimg/yuque-1777880833859-4.gif)

从栈中弹出剩余元素，此时是字符串 `ac`，因为从栈里弹出的元素是倒序的，所以**再对字符串进行反转一下**，就得到了最终的结果。

代码如下：

```
public String removeDuplicates(String str) {
    // 使用辅助栈
    LinkedList<Character> stack = new LinkedList<>();
    for (char c : str.toCharArray()) {
        // 栈为空或者当前元素与栈顶元素不同则入栈
        if (!stack.isEmpty() && stack.peek() == c) {
            stack.pop();
        } else {
            // 当前元素与栈顶元素相同则弹出栈顶元素
            stack.push(c);
        }
    }

    StringBuilder res = new StringBuilder();
    // 剩余的元素即为不重复的元素
    while (!stack.isEmpty()) {
        // 栈里面的元素时倒序的, 所以需要逆序添加到结果中
        res.insert(0, stack.pop());
    }
    return res.toString();
}
```

### 逆波兰表达式求值(150-中等) 💯

根据逆波兰表示法，求表达式的值。有效的运算符包括 `+ - * /`。每个运算对象可以是整数，也可以是另一个逆波兰表达式。说明：整数除法只保留整数部分。给定逆波兰表达式总是有效的。换句话说，表达式总会得出有效数值且不存在除数为 0 的情况。

逆波兰表达式主要有以下两个优点：

- 去掉括号后表达式无歧义，上式即便写成 `1 2 + 3 4 + *` 也可以依据次序计算出正确结果。
- **适合用栈操作运算：遇到数字则入栈；遇到运算符则取出栈顶两个数字进行计算，并将结果压入栈中**。

**辅助栈法**

本题中每一个子表达式要得出一个结果，然后拿这个结果再进行运算，这其实就是一个相邻字符串消除的过程，和 [删除字符串中的所有相邻重复项(1047-简单)](https://www.yuque.com/daerwendecha/qwuv92/mwbupcddc46kti5o#OYFlP) 中的对对碰游戏是不是就非常像了。

如动画所示：

![](/zimg/yuque-1777880833927-5.gif)

这和 [删除字符串中的所有相邻重复项(1047-简单)](https://www.yuque.com/daerwendecha/qwuv92/mwbupcddc46kti5o#OYFlP) 是差不多的，只不过本题不要相邻元素做消除了，而是**做运算**！其实就是遍历字符串，**如果遇到数字就压栈，遇到运算符就弹出两个元素进行运算并将结果再次压栈**，需要注意的是，弹出两个元素时，第一个弹出的是第二个操作数，二个弹出的是第一个操作数。

```
public int evalRPN(String[] tokens) {
    // 辅助栈
    Deque<Integer> stack = new LinkedList<>();
    for (String token : tokens) {
        if (isNumber(token)) {
            stack.push(Integer.parseInt(token));
            continue;
        }

        // 注意: 这里先弹出来的是第二个操作数!!后弹出来的事第一个操作数
        int num2 = stack.pop();
        int num1 = stack.pop();
        // 根据不同的符号求值后将结果压栈
        if ("+".equals(token)) stack.push(num1 + num2);
        if ("-".equals(token)) stack.push(num1 - num2);
        if ("*".equals(token)) stack.push(num1 * num2);
        if ("/".equals(token)) stack.push(num1 / num2);
    }

    return stack.pop();
}

private boolean isNumber(String token) {
    return !"+".equals(token) && !"-".equals(token) && !"*".equals(token) && !"/".equals(token);
}
```

### 栈的压入弹出序列匹配(946-中等) 🔑

输入两个整数序列，第一个序列表示栈的压入顺序，请判断第二个序列是否可能为该栈的弹出顺序。假设压入栈的所有数字均不相等。例如序列 `1, 2, 3, 4, 5` 是某栈的压入顺序，序列 `4, 5, 3, 2, 1` 是该压栈序列对应的一个弹出序列，但 `4, 3, 5, 1, 2` 就不可能是该压栈序列的弹出序列。(注意：这两个序列的长度是相等的)。

```
输入: pushed = [1,2,3,4,5], popped = [4,5,3,2,1]   输出: true
输入: pushed = [1,2,3,4,5], popped = [4,3,5,1,2]   输出: false
```

[https://leetcode.cn/problems/validate-stack-sequences/description/](https://leetcode.cn/problems/validate-stack-sequences/description/)

**模拟栈法**

借助一个辅助栈进行模拟，遍历压栈序列放入到辅助栈中模拟压栈行为。先将第一个元素放在栈中，这里是 1，然后判断栈顶元素是不是出栈序列的第一个元素，这里 `1 ≠ 4`，所以继续压栈，直到压栈序列的元素与弹栈序列的元素相等之后开始出栈。每出栈一个元素，对于出栈序列向后移动一位，持续此操作直到两个序列不相等。这样循环操作直到压栈顺序遍历完成，如果辅助栈还不为空，说明弹出序列不是该栈的弹出顺序。

举例：入栈 `1, 2, 3, 4, 5`，出栈 `4, 5, 3, 2, 1`。首先 1 入辅助栈，此时栈顶 `1 ≠ 4`，继续入栈 2。此时栈顶 `2 ≠ 4`，继续入栈 3。此时栈顶 `3 ≠ 4`，继续入栈 4 。此时栈顶 `4 ＝ 4`，出栈 4，弹出序列向后一位，此时为 5，辅助栈里面是 1，2，3。此时栈顶 `3 ≠ 5`，继续入栈 5。此时栈顶 `5 = 5`，出栈 5，弹出序列向后一位，此时为 3，辅助栈里面是 `1, 2, 3`。

![](/zimg/yuque-1777880833974-6.gif)

代码如下：

```
// pushed = [1,2,3,4,5], popped = [4,5,3,2,1]
public boolean validateStackSequences(int[] push, int[] pop) {
    if (push == null || pop == null) {
        return false;
    }
    // 长度不同直接不可能匹配
    if (push.length != pop.length) {
        return false;
    }

    // 辅助栈
    LinkedList<Integer> stack = new LinkedList<>();

    // 弹出序列的索引
    int popIndex = 0;

    // 遍历压栈序列
    for (int k : push) {
        // 每个元素都压入到栈顶中
        stack.push(k);
        // 如果当前压入的元素与弹出序列不一致 则继续压栈
        if (k != pop[popIndex]) {
            continue;
            // 当前压入的元素与弹出序列一致, 则持续对比栈顶和弹栈序列, 如果相同则一直弹出元素并移动指针
        } else {
            while (!stack.isEmpty() && stack.peek() == pop[popIndex]) {
                stack.pop();
                popIndex++;
            }
        }
    }

    // 最后看辅助栈是否为空
    return stack.isEmpty();
}
```

### 字符串解码(394-中等) 💯

给定一个经过编码的字符串，返回它解码后的字符串。编码规则为：`k[encodedString]`，表示其中方括号内部的 `encodedString` 正好重复 k 次。注意 k 保证为正整数。

```
输入：s = "3[a]2[bc]"  输出："aaabcbc"
输入：s = "3[a2[c]]"   输出："accaccacc"
输入：s = "2[abc]3[cd]ef"  输出："abcabccdcdcdef"
```

[https://leetcode.cn/problems/decode-string/description/](https://leetcode.cn/problems/decode-string/description/)

**双栈模拟法**

整个过程需要**先记录片段重复的次数，然后找出片段，这里就可以使用栈来分别缓存这两部分数据**，而且使用栈可以处理括号重复嵌套的问题。

整体流程为：**遍历原始字符串，遇到左括号则认为重新开启一次片段的统计，遇到右括号则从栈中弹出数据，进行字符串片段的构造与拼接，遇到数字则计算需要重复的次数，遇到普通字符则直接拼接**。

```
public String decodeString(String s) {
    LinkedList<Integer> numStack = new LinkedList<>();
    LinkedList<StringBuilder> strStack = new LinkedList<>();

    int num = 0;
    StringBuilder cur = new StringBuilder();

    for (char c : s.toCharArray()) {
        // 对当前字符分类讨论
        // 如果是左括号, 则将当前状态存起来, 先处理括号内的信息
        if (c == '[') {
            numStack.push(num);
            strStack.push(cur);
            // 重置状态
            num = 0;
            cur = new StringBuilder();

            // 如果是右括号，则需要进行解码
        } else if (c == ']') {
            // 取出对应的次数和字符串, 并进行拼接
            int loopNum = numStack.pop();
            StringBuilder str = strStack.pop();
            while (loopNum-- > 0) {
                // 这里会把之前的字符串拼接上去
                str.append(cur.toString());
            }

            // 继续拼接
            cur = str;
        } else if (Character.isDigit(c)) {
            // 注意此处数字并非只有一位, 又因为数字都是相邻的, 因此要拼接数字
            num = num * 10 + c - '0';
            // 如果是普通字母则直接拼接
        } else {
            cur.append(c);
        }
    }

    return cur.toString();
}
```

### 最长有效括号(32-困难) 💯

给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

```
输入：s = "(()"      输出：2
解释：最长有效括号子串是 "()"
输入：s = ")()())"   输出：4
解释：最长有效括号子串是 "()()"
```

[https://leetcode.cn/problems/longest-valid-parentheses/description/](https://leetcode.cn/problems/longest-valid-parentheses/description/)

**栈匹配法**

对于这种括号匹配问题，一般都是使用栈。

思路：**先找到所有可以匹配的括号的索引值，对索引值排序后，找出最长连续数列即为最长有效的括号长度**！

例如：`s = )(()())`，用栈可以找到，

- 位置 2 和位置 3 匹配
- 位置 4 和位置 5 匹配
- 位置 1 和位置 6 匹配

这个数组为：`2, 3, 4, 5, 1, 6` 这是通过栈找到的，按递增排序：`1, 2, 3, 4, 5, 6`。**找出该数组的最长连续递增数列的长度就是最长有效括号长度**！

代码如下：

```
public int longestValidParentheses(String s) {
    if (s == null || s.isEmpty()) {
        return 0;
    }

    Deque<Integer> stack = new ArrayDeque<>();

    // 存储匹配的括号的索引
    List<Integer> matchList = new ArrayList<>();

    // 遍历栈找寻合适的左右括号
    for (int i = 0; i < s.length(); i++) {
        // 如果找到左括号则入栈(注意存的是索引)
        if (s.charAt(i) == '(') {
            stack.push(i);
        } else {
            // 这里遇到了右括号
            // 如果栈为空且当前字符是右括号则直接跳过, 因为这个右括号一定无法构成有效括号
            if (stack.isEmpty()) {
                continue;
            }

            // 如果栈不为空, 且栈顶是左括号, 则匹配成功, 弹出栈顶的左括号, 且记录一下匹配的两个索引
            if (s.charAt(stack.peek()) == '(') {
                matchList.add(stack.pop());
                matchList.add(i);
            } else {
                // 如果栈顶是右括号, 则直接入栈
                stack.push(i);
            }
        }
    }

    // 对匹配的括号索引进行排序
    matchList.sort(Integer::compareTo);

    // 求最长连续递增的子序列长度
    int res = 0, count = 1;
    for (int i = 1; i < matchList.size(); i++) {
        if (matchList.get(i) - matchList.get(i - 1) == 1) {
            count++;
        } else {
            // 重新计数
            count = 1;
        }
        // 滚动更新结果值
        res = Math.max(res, count);
    }

    return res;
}
```

## 单调栈与单调队列题目

使用单调栈或者单调队列时，一般存储的都是索引值，而不是具体的数字。因为通过索引值不仅能拿到对应元素的大小，还能进行更多的计算操作，比如计算两个元素的距离。

### 最小值栈(155-简单) 🔑💯

设计一个支持 push，pop，top 等操作，并能在常数时间内检索到最小元素的栈。

- `push(x)`：将元素 x 压入栈中。
- `pop()`：删除栈顶的元素。
- `top()`：获取栈顶元素。
- `getMin()`：检索栈中的最小元素。

[https://leetcode-cn.com/problems/min-stack](https://leetcode-cn.com/problems/min-stack)

**辅助栈法**

借用一个辅助栈 minStack，用于不断存储获取 stack 中最小值。

- `push()` 方法：每当 `push()` 新值进来时，如果 小于等于 minStack 栈顶值，则一起 `push()` 到 minStack，即更新了栈顶最小值。
- `pop()` 方法：判断将 `pop()` 出去的元素值是否是 minStack 栈顶元素值(即最小值)，如果是则将 minStack 栈顶元素一起 `pop()`，这样可以保证 minStack 栈顶元素始终是 stack 中的最小值。
- `getMin()` 方法：返回 minStack 栈顶即可。

![](/zimg/yuque-1777880833991-7.gif)

minStack 作用分析：**minStack 等价于遍历 stack 所有元素，把升序的数字都删除掉，留下一个从栈底到栈顶降序的栈**。相当于给 stack 中的降序元素做了标记，每当 `pop()` 这些降序元素，minStack 会将相应的栈顶元素 `pop()` 出去，保证其栈顶元素始终是 stack 中的最小元素。

```
static class MinStack {
    // 数据栈
    private Stack<Integer> dataStack = new Stack<>();
    // 最小值栈
    private Stack<Integer> minStack = new Stack<>();

    public void push(int data) {
        // 所有数据都压入普通数据栈
        dataStack.push(data);
        // 如果当前最小值栈为空或新值小于最小值栈的栈顶则同时压入最小值栈
        if(minStack.isEmpty() || data <= minStack.peek()) {
            minStack.push(data);
        }
    }

    public void pop() {
        // 如果弹出的元素等于最小值栈的栈顶, 则两个栈同时弹出元素
        if (dataStack.peek().equals(minStack.peek())) {
            dataStack.pop();
            minStack.pop();
        } else {
            // 否则只弹出普通栈数据
            dataStack.pop();
        }
    }

    public int top() {
        return dataStack.peek();
    }

    public int getMin() {
        // 获取最小值
        return minStack.peek();
    }
}
```

一个自己写的便于理解的写法，思路就是**让数据栈和最小值栈的元素数量是一致的，也就是每次入栈都得选一个当前最小的入最小值栈，出栈则两个栈都出**。

```
class MinStack {

    private LinkedList<Integer> dataStack;
    private LinkedList<Integer> minStack;

    public MinStack() {
        dataStack = new LinkedList<>();
        minStack = new LinkedList<>();
    }

    public void push(int val) {
        dataStack.push(val);
        // 注意: 这里的思路是每次压栈的时候也往最小值栈压一个数据, 保证两个栈的数据量是一样的
        if (minStack.isEmpty()) {
            minStack.push(val);
        } else {
            // 比较当前最小值栈的栈顶的加入的元素, 取较小者加入最小值栈
            minStack.push(Math.min(minStack.peek(), val));
        }
    }

    public void pop() {
        if (dataStack.isEmpty()) {
            return;
        }

        // 由于两个栈保持数量一致, 所以需要同时弹出
        dataStack.pop();
        minStack.pop();
    }

    public int top() {
        if (dataStack.isEmpty()) {
            return 0;
        }

        return dataStack.peek();
    }

    public int getMin() {
        if (minStack.isEmpty()) {
            return 0;
        }

        return minStack.peek();
    }
}
```

### 每日温度(739-中等) 💯

请根据每日气温列表，重新生成一个列表。对应位置的输出为：**要想观测到更高的气温，至少需要等待的天数**。如果气温在这之后都不会升高，请在该位置用 0 来代替。

例如，给定一个列表 `temperatures：[73, 74, 75, 71, 69, 72, 76, 73]`，输出 `[1, 1, 4, 2, 1, 1, 0, 0]`。

```
Input: [73, 74, 75, 71, 69, 72, 76, 73]  Output: [1, 1, 4, 2, 1, 1, 0, 0]
```

[https://leetcode-cn.com/problems/daily-temperatures](https://leetcode-cn.com/problems/daily-temperatures)

**单调栈法**

使用一个辅助的单调栈，这里的栈中存的是**温度的索引位置**而不是具体的温度值。**栈中索引对应的温度值是单调递减的**。遍历温度数组，如果当前温度小于栈顶温度，说明当前温度不是前面温度的下一个最高温度，此时就需要把这些温度缓存到栈中，直到遇到下一个较高的温度为止，比如 `75-71-69` 都会在栈顶缓存起来。当前温度大于栈顶温度，则说明目前栈里面可能有多个位置的下一个较大温度就是当前温度，这里用 while 找出所有这种温度并从栈中弹出，索引值相减就是结果值。此时即可更新对应位置的结果了。

视频讲解，看一次就明白了：

[https://leetcode.cn/problems/daily-temperatures/solutions/71433/leetcode-tu-jie-739mei-ri-wen-du-by-misterbooo/](https://leetcode.cn/problems/daily-temperatures/solutions/71433/leetcode-tu-jie-739mei-ri-wen-du-by-misterbooo/)

```
public int[] dailyTemperatures(int[] temps) {
    int[] result = new int[temps.length];

    // 单调减小栈(存储的是温度的索引, 索引对应的值单调递减)
    Stack<Integer> minStack = new Stack<>();

    // 遍历温度数组
    for (int i = 0; i < temps.length; i++) {
        // 当前温度比栈顶温度高时, 可以持续处理一批比当前温度低的温度的天数
        while (!minStack.isEmpty() && temps[i] > temps[minStack.peek()]) {
            int prevIndex = minStack.pop();
            result[prevIndex] = i - prevIndex;
        }
        // 将当前索引入栈
        minStack.push(i);
    }

    // 栈中剩余的元素都是后面没有更高温度的, 默认为0即可
    return result;
}
```

给定一个循环数组(最后一个元素的下一个元素是数组的第一个元素)，输出每个元素的下一个更大元素。数字 x 的下一个更大的元素是按数组遍历顺序，这个数字之后的第一个比它更大的数，这意味着应该循环地搜索它的下一个更大的数。如果不存在，则输出 -1。

```
输入: [1,2,1]   输出: [2,-1,2]
解释: 第一个1的下一个更大的数是2;数字2找不到下一个更大的数; 第二个1的下一个最大的数需要循环搜索, 结果也是2.
```

[https://leetcode-cn.com/problems/next-greater-element-ii](https://leetcode-cn.com/problems/next-greater-element-ii)

**单调栈法**

这个题就是 "下一个更大元素" 的升级版本。比较麻烦的就是**当前元素的下一个最大元素可能在前面**。一个朴素的思想是，可以把这个循环数组「拉直」，即复制该序列的前 `n − 1` 个元素拼接在原序列的后面。这样就可以将这个新序列当作普通序列，用 [每日温度(739-中等)](https://www.yuque.com/daerwendecha/qwuv92/mwbupcddc46kti5o#yET0E) 的方法来处理。

而在本题中，不需要显性地将该循环数组「拉直」，而只需要在处理时对下标取模即可。

![](/zimg/yuque-1777880834954-8.png)

通过取模的方式优化的代码如下。

```
public int[] nextGreaterElements(int[] nums) {
    int len = nums.length;
    int[] res = new int[len];
    // 不存在下一个较大的元素则返回-1, 这里初始化一下
    Arrays.fill(res, -1);

    // 辅助单调栈
    LinkedList<Integer> stack = new LinkedList<>();

    // 需要遍历两次, 因此长度为两个拼接在一起数组之和
    for (int i = 0; i < len * 2 - 1; i++) {
        // 注意: 取模获取当前实际的索引位置
        int curIndex = i % len;
        // 持续找出比当前元素更小的栈顶的位置元素 并更新结果
        while (!stack.isEmpty() && nums[stack.peek()] < nums[curIndex]) {
            int preIndex = stack.pop();
            res[preIndex] = nums[curIndex];
        }
        // 记录当前的索引
        stack.push(curIndex);
    }

    return res;
}
```

### 下一个更大元素(496-简单)

给定两个没有重复元素的数组 nums1 和 nums2，其中 nums1 是 nums2 的子集。**找到 nums1 中每个元素在 nums2 中的下一个比其大的值**。nums1 中数字 x 的下一个更大元素是指 x 在 nums2 中对应位置的右边的第一个比 x 大的元素。如果不存在时对应位置输出 -1。

```
示例 1: 输入: nums1 = [4,1,2], nums2 = [1,3,4,2].   输出: [-1,3,-1]
解释:
    对于num1中的数字4, 无法在第二个数组中找到下一个更大的数字, 因此输出 -1.
    对于num1中的数字1, 第二个数组中数字1右边的下一个较大数字是 3.
    对于num1中的数字2, 第二个数组中没有下一个更大的数字, 因此输出 -1.
示例 2: 输入: nums1 = [2,4], nums2 = [1,2,3,4].     输出: [3,-1]
```

[https://leetcode-cn.com/problems/next-greater-element-i](https://leetcode-cn.com/problems/next-greater-element-i)

**单调栈法**

这是 "每日温度" 的升级版本。可以忽略数组 nums1，**先对 nums2 中的每一个元素，先利用单调栈的解法求出每个元素的下一个更大的元素**。并将这些答案放入 HashMap 中(这就是强调元素不相同的原因)，最后再遍历 nums1 直接找出答案。

注意：这里思想与 "每日温度" 是一样的，**只不过由于元素是不同的，所以这里栈中这里直接存的是元素值而非索引值，进行比较时亦是如此**。

```
public int[] nextGreaterElement(int[] nums1, int[] nums2) {
    // 存放元素及其下一个较大元素的映射
    HashMap<Integer, Integer> num2NextBigger = new HashMap<>();
    // 辅助单调栈存储索引值
    LinkedList<Integer> stack = new LinkedList<>();

    for (int i = 0; i < nums2.length; i++) {
        while (!stack.isEmpty() && nums2[stack.peek()] < nums2[i]) {
            int j = stack.pop();
            num2NextBigger.put(nums2[j], nums2[i]);
        }
        // 索引位置入栈
        stack.push(i);
    }


    int[] res = new int[nums1.length];

    // 直接从映射中拿到结果
    for (int i = 0; i < nums1.length; i++) {
        res[i] = num2NextBigger.getOrDefault(nums1[i], -1);
    }
    return res;
}
```

给定一个数组 nums，有一个大小为 k 的滑动窗口从数组的最左侧移动到数组的最右侧。只可以看到在滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。返回滑动窗口中的最大值。

```
输入: nums = [1,3,-1,-3,5,3,6,7], k = 3   输出: [3,3,5,5,6,7]
解释:
滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

[https://leetcode.cn/problems/sliding-window-maximum/description/](https://leetcode.cn/problems/sliding-window-maximum/description/)

**单调队列法**

维护一个**双端队列，这个队列里面存储的是索引值而不是具体的元素，且索引值对应的元素是单调递减的**。

算法流程如下：

1. 初始化：创建结果数组和双端队列。
2. 依次遍历数组：**移除不在当前窗口范围内的索引(队列头部)；并移除队列尾部所有小于当前元素的索引(这一步把队列移除完也有可能)，这样就能保持队列的单调递减；再将当前索引入队。这里可能同时需要弹出队头和队尾，所以需要使用双端队列**。
3. 记录结果：当窗口形成后(`i >= k - 1`)，队列头部即为当前窗口最大值，每次记录一下结果值即可。

为了更直观的感受到单调队列的工作过程，以题目示例为例，输入：nums = `[1,3,-1,-3,5,3,6,7]`，和 `k = 3`，动画如下：

![](/zimg/yuque-1777880834965-9.gif)

代码如下：

```
public int[] maxSlidingWindow(int[] nums, int k) {
    if (nums == null || nums.length == 0 || k <= 0) {
        return new int[0];
    }

    int length = nums.length;
    int[] result = new int[length - k + 1];
    Deque<Integer> maxQueue = new LinkedList<>();

    for (int i = 0; i < length; i++) {
        // 移除不在当前窗口范围内的索引
        while (!maxQueue.isEmpty() && maxQueue.peekFirst() < i - k + 1) {
            maxQueue.pollFirst();
        }

        // 移除队列尾部所有小于当前元素的索引, 这样才能保证队列单调递减, 队头元素就是当前窗口的最大值
        while (!maxQueue.isEmpty() && nums[maxQueue.peekLast()] < nums[i]) {
            maxQueue.pollLast();
        }

        // 将当前索引入队
        maxQueue.offerLast(i);

        // 前k个元素不处理, 当形成长度为k的窗口后, 开始记录队头结点作为结果
        if (i >= k - 1) {
            result[i - k + 1] = nums[maxQueue.peekFirst()];
        }
    }

    return result;
}
```

**基于堆实现**

基于**堆实现**。这个代码会超出时间限制。

```
public int[] maxSlidingWindow(int[] nums, int k) {
    if(nums == null || nums.length == 0 || k < 1) {
        return new int[]{};
    }

    ArrayList<Integer> resList = new ArrayList<>();
    // 大顶堆
    PriorityQueue<Integer> heap = new PriorityQueue<>((o1, o2) -> o2 - o1);
    // 先将前k个元素加入堆中
    for (int i = 0; i < k; i++) {
        heap.add(nums[i]);
    }

    // 此时第一个元素就是当前最大值
    resList.add(heap.peek());

    // 维护一个大小为size的大顶堆
    for (int i = 0, j = i + k; j < nums.length; i++, j++) {
        // 不断取出窗口的左边界并加入右边界
        heap.remove(nums[i]);
        heap.add(nums[j]);
        // 每次滑动一个都得到一个值
        resList.add(heap.peek());
    }

    // 列表转为数组
    int[] res = new int[resList.size()];
    for (int i = 0; i < res.length; i++) {
        res[i] = resList.get(i);
    }

    return res;
}
```

### 柱状图中的最大矩形面积(84-困难) 💯

给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1。求在该柱状图中能够**勾勒出来的矩形的最大面积**。

![](/zimg/yuque-1777880835162-10.png)

```
输入：heights = [2,1,5,6,2,3]  输出：10
解释：最大的矩形为图中红色区域，面积为 10
```

[https://leetcode.cn/problems/largest-rectangle-in-histogram/description/](https://leetcode.cn/problems/largest-rectangle-in-histogram/description/)

**单调栈法**

首先要想找到第 i 位置最大面积是什么？

是以 i 为中心，向左找第一个小于 `heights[i]` 的位置 `left_i`；向右找第一个小于 `heights[i]` 的位置 `right_i`，即最大面积为 `heights[i] * (right_i - left_i - 1)`，如下图所示：

![](/zimg/yuque-1777880835181-11.png)

所以问题就变成如何找 `right_i` 和 `left_i`。最简单的思路就是，就是暴力法，直接分别在 i 左右移动。但这个会超时。

**可以使用单调栈，预先计算出每个位置 i 的左边和右边的第一个小于该值的位置信息，这样计算位置 i 的面积时，直接获取到左右的索引，即可计算出矩形的宽度，然后乘以当前位置的高度就是面积了**。

代码如下：

```
public int largestRectangleArea(int[] heights) {
    int len = heights.length;
    // 单调栈
    Deque<Integer> stack = new ArrayDeque<>();

    // 找到每个柱子左边第一个小于当前柱子的柱子的索引
    int[] left = new int[len];
    for (int i = 0; i < len; i++) {
        int height = heights[i];
        // 如果栈不为空且当前柱子的高度小于等于栈顶所指的柱子的高度, 则弹出栈顶
        while (!stack.isEmpty() && height <= heights[stack.peek()]) {
            stack.pop();
        }
        // 如果栈为空, 则说明当前柱子左边没有比它矮的柱子
        // 否则, left[i] 为栈顶元素的索引
        left[i] = stack.isEmpty() ? -1 : stack.peek();
        // 将当前柱子的索引压入栈中
        stack.push(i);
    }

    // 清空栈以便重新使用
    stack.clear();

    // 找到每个柱子右边第一个小于当前柱子的柱子的索引
    int[] right = new int[len];
    for (int i = len - 1; i >= 0; i--) {
        int height = heights[i];
        // 如果栈不为空且当前柱子的高度小于等于栈顶所指的柱子的高度，则弹出栈顶
        while (!stack.isEmpty() && height <= heights[stack.peek()]) {
            stack.pop();
        }
        right[i] = stack.isEmpty() ? len : stack.peek();
        stack.push(i);
    }

    // 计算最大的矩形面积
    int res = 0;
    // 遍历每个柱子, 计算以该柱子为高度的最大矩形面积
    for (int i = 0; i < len; i++) {
        // 其实就是找到左右两边第一个小于当前柱子的柱子的索引则可计算出矩形的宽度
        // 矩形的宽度为 right[i] - left[i] - 1
        int width = right[i] - left[i] - 1;
        // 计算以 heights[i] 为高度的矩形面积并更新最大面积
        res = Math.max(res, heights[i] * width);
    }

    return res;
}
```

### 移除 K 个数字使结果最小(402-中等)

给定一个以字符串表示的非负整数 num，移除这个数中的 k 位数字，使得剩下的数字最小。

```
输入: num = "1432219", k = 3    输出: "1219"
解释: 移除掉三个数字 4, 3, 和 2 形成一个新的最小的数字 1219.
输入：num = "10200", k = 1       输出："200"
解释：移掉首位的 1 剩下的数字为 200. 注意输出不能有任何前导零。
```

[https://leetcode.cn/problems/remove-k-digits/description/](https://leetcode.cn/problems/remove-k-digits/description/)

**单调栈法**

**维护一个单调栈(单调不减栈)，从左到右(先处理高位)遍历原来字符串中的每一个字符，如果当前字符小于栈顶的字符，则较大的字符出栈，出栈的数字就是需要移除的数字**。如果遍历完后，移除的数字个数未达到 k，则直接移除栈顶的元素。

最后取出栈中的索引，注意取出的时候需要判断字符串前面的是否是 0，如果是就需要去除 0。

```
public String removeKdigits(String num, int k) {
    char[] chars = num.toCharArray();
    if (k >= chars.length) {
        return "0";
    }

    // 栈中记录的是索引的指针
    Stack<Integer> stack = new Stack<>();
    int count = 0;

    for (int i = 0; i < chars.length; i++) {
        // 下一个元素如果小于栈顶元素就弹出栈顶元素(表示被移除)
        while (!stack.isEmpty() && chars[stack.peek()] > chars[i]) {
            // count记录弹出几次
            if (count < k) {
                count++;
                stack.pop();
            } else {
                break;
            }
        }
        // 将当前索引加入栈中
        stack.push(i);
    }

    // 如果 count==k 说明 stack 栈内元素正好是 num 去掉 k 个的数字
    // 如果 count<k 说明栈没有去掉 k 个数, 所以继续弹出栈顶元素
    for (int i = count; i < k; i++) {
        stack.pop();
    }

    // 栈中即为结果, 构造时去掉前面的0
    StringBuilder resBuilder = new StringBuilder();

    // 判断是否遇到非0元素
    boolean leadingZero = true;
    for (int digit : stack) {
        // 去掉前面的0
        if (leadingZero && chars[digit] == '0') {
            continue;
        }
        leadingZero = false;
        resBuilder.append(chars[digit]);
    }
    if (resBuilder.length() == 0) {
        return "0";
    }
    return resBuilder.toString();
}
```
