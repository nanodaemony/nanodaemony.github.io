## 快慢双指针题目

双指针法(快慢指针法)在数组和链表的操作中是非常常见的，很多考察数组，链表，字符串等操作的面试题。都使用双指针法。

### 移动数组中的 0 到末尾(283-简单) 💯

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

```
输入: nums = [0,1,0,3,12]   输出: [1,3,12,0,0]
```

[https://leetcode.cn/problems/move-zeroes/description](https://leetcode.cn/problems/move-zeroes/description)

**快慢双指针交互法**

**使用快慢双指针，每次快指针都会移动一步，如果快指针遇到非 0 的值，则与慢指针的位置交互，并移动慢指针**，这样快指针到末尾即处理完成。

```
// [0,1,0,3,12]
public void moveZeroes(int[] nums) {
    if (nums == null || nums.length == 0) {
        return;
    }

    // 快慢双指针
    int fast = 0;
    int slow = 0;
    while (fast < nums.length) {
        // 如果快指针不为0则与慢指针交互
        if (nums[fast] != 0) {
            swap(nums, slow, fast);
            slow++;
        }
        // 每次都会移动一下快指针
        fast++;
    }
}

private void swap(int[] nums, int i, int j) {
    int temp = nums[i];
    nums[i] = nums[j];
    nums[j] = temp;
}
```

也可以单指针遍历一次，**从前往后遍历**，遇到非 0 的就拷贝的数组前面。最后将后面的元素全部置 0。

```
public void moveZeroes(int[] nums) {
    int index = 0;
    for (int num : nums) {
        // 不为0就拷贝到前面
        if (num != 0) {
            nums[index++] = num;
        }
    }
    // 将后面的全部置0
    while (index < nums.length) {
        nums[index++] = 0;
    }
}
```

### 移除元素(27-简单) 💯

给你一个数组 nums 和一个值 val，原地移除所有数值等于 val 的元素，并返回移除后数组的新长度。必须仅使用 `O(1)` 额外空间并原地修改输入数组。元素的顺序可以改变。不需要考虑数组中超出新长度后面的元素。

```
示例 1: 给定 nums = [3,2,2,3], val = 3, 函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2. 不需要考虑数组中超出新长度后面的元素.
示例 2: 给定 nums = [0,1,2,2,3,0,4,2], val = 2, 函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4.
```

**快慢双指针法**

使用快慢双指针进行遍历，快指针遇到等于 val 的元素直接则直接跳过，否则将快指针的元素赋值给慢指针，并移动慢指针。这样并没有改变元素的相对位置！

整个过程如下：

![](/zimg/yuque-1777880825855-1.gif)

代码如下：

```
public int removeElement(int[] nums, int deleteVal) {
    if (nums == null) {
        return 0;
    }

    // 由于第一个元素可能就是删除元素, 因此快指针需要从0开始
    int slow = 0, fast = 0;
    while (fast < nums.length) {
        if (nums[fast] != deleteVal) {
            nums[slow] = nums[fast];
            slow++;
        }
        // 快指针一直向前走
        fast++;
    }
    return slow + 1;
}
```

### 删除排序数组中的重复项(26-简单) 💯

给定一个排序数组，需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。不要使用额外的数组空间，必须在原地修改输入数组并在使用 `O(1)` 额外空间的条件下完成。

```
示例 1:
给定数组 nums = [1,1,2], 返回新的长度 2, 并且原数组 nums 的前两个元素被修改为 1, 2.
不需要考虑数组中超出新长度后面的元素.
示例 2:
给定 nums = [0,0,1,1,1,2,2,3,3,4], 返回新的长度 5, 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4.
```

[https://leetcode-cn.com/problems/remove-duplicates-from-sorted-array](https://leetcode-cn.com/problems/remove-duplicates-from-sorted-array)

**快慢双指针法**

数组是有序的，那么重复的元素一定会相邻。要求删除重复元素，实际上就是将不重复的元素移到数组的左侧。

使用快慢双指针的流程如下：

比较 slow 和 fast 位置的元素是否相等。如果相等则 fast 后移 1 位，相当于跳过重复元素。如果不相等，将 fast 位置的元素复制到 `slow + 1` 位置上(先将 `slow + 1`)，slow 后移一位，fast 后移 1 位。重复上述过程，直到 fast 到达数组末尾。返回 `slow + 1` 即为新数组长度。

![](/zimg/yuque-1777880826157-2.png)

动图示例：

![](/zimg/yuque-1777880826179-3.gif)

代码如下：

```
public int removeDuplicates(int[] nums) {
    if (nums == null || nums.length == 0) return 0;

    // 快慢双指针
    int slow = 0, fast = 1;

    // 1 1 2 2 3 3 3 4 4 5 6 7
    while (fast < nums.length) {
        // 如果快慢指针指向的值相等, 快指针继续向前走
        if (nums[fast] == nums[slow]) {
            fast++;
            // 如果快慢指针指向的值不相等, 慢指针向前走一步, 并将快指针的值赋给慢指针
        } else {
            // 慢指针先向前移动一次, 不然会覆盖掉slow之前指向的位置的元素
            slow++;
            nums[slow] = nums[fast];
            fast++;
        }
    }
    // slow是索引, 需要+1才是长度
    return slow + 1;
}
```

### 删除有序数组中的重复项 II(80-中等) 💯

给你一个有序递增的数组 nums，原地删除重复出现的元素，使得出现次数超过两次的元素只出现两次，返回删除后数组的新长度。

```
示例 1:
输入: nums = [1,1,1,2,2,3]   输出: 5, nums = [1,1,2,2,3]
解释: 函数应返回新长度 length = 5, 并且原数组的前五个元素被修改为 1, 1, 2, 2, 3.
示例 2:
输入: nums = [0,0,1,1,1,1,2,3,3]  输出: 7, nums = [0,0,1,1,2,3,3]
解释: 函数应返回新长度 length = 7, 并且原数组的前七个元素被修改为 0, 0, 1, 1, 2, 3, 3.
```

[https://leetcode.cn/problems/remove-duplicates-from-sorted-array-ii/description/?envType=study-plan-v2&envId=top-interview-150](https://leetcode.cn/problems/remove-duplicates-from-sorted-array-ii/description/?envType=study-plan-v2&envId=top-interview-150)

**快慢双指针法**

本题思路与 [删除排序数组中的重复项(26-简单)](https://www.yuque.com/daerwendecha/qwuv92/yrwhwzgeukb75mru#SfDm1) 类似，也是使用双指针，只是需要每个元素保留两次。

前两个元素是一定不会动的，所以快慢指针起始指向第二、三个元素。之后不断移动快指针，如果快指针的元素值与慢指针的前两个元素都相同，则说明慢指针的元素已经出现了两次，此时直接移动快指针即可。如果不相同，则把快指针位置的元素复制到慢指针处，也就相当于把重复的元素抹掉了。

![](/zimg/yuque-1777880826199-4.png)

最后数组后面剩余的元素(比如最后以 `2333333333` 结尾, 这里只需要保留 "`233`")是不用管的，因为方法只需要返回慢指针的长度即可。

代码如下：

```
public int removeDuplicates(int[] nums) {
    if (nums == null) {
        return 0;
    }

    if (nums.length <= 2) {
        return nums.length;
    }

    // 第一个元素一定不会动, 快慢指针均指向第二, 三个元素
    int slow = 1;
    int fast = 2;

    while (fast < nums.length) {
        // 如果快指针的元素与慢指针及其之前的元素都相同, 此时说明出现次数超过2了, 则快指针继续向前移动
        if (nums[fast] == nums[slow] && nums[fast] == nums[slow - 1]) {
            fast++;
            // 否则将快指针处的元素复制到慢指针处
        } else {
            // 慢指针先向前移动一次, 不然会覆盖掉slow之前指向的位置的元素
            slow++;
            // 将快指针的值赋给慢指针
            nums[slow] = nums[fast];
            fast++;
        }
    }

    // slow是索引, 需要+1才是长度
    return slow + 1;
}
```

### 合并两个有序数组(88-简单) 💯

给你两个按非递减顺序排列的整数数组 nums1 和 nums2,，另有两个整数 m 和 n，分别表示 nums1 和 nums2 中的元素数目。合并 nums2 到 nums1 中，使合并后的数组同样按 非递减顺序 排列。

注意：最终合并后数组存储在数组 nums1 中。nums1 的初始长度为 m + n，其中前 m 个元素表示应合并的元素，后 n 个元素为 0, 应忽略。nums2 的长度为 n。

```
示例 1:
输入: nums1 = [1,2,3,0,0,0], m = 3, nums2 = [2,5,6], n = 3   输出: [1,2,2,3,5,6]
示例 2:
输入: nums1 = [0], m = 0, nums2 = [1], n = 1  输出: [1]
解释: 需要合并的数组是 [] 和 [1] .注意, 因为 m = 0 , 所以 nums1 中没有元素. nums1 中仅存的 0 仅仅是为了确保合并结果可以顺利存放到 nums1 中.
```

[https://leetcode.cn/problems/merge-sorted-array/description/](https://leetcode.cn/problems/merge-sorted-array/description/)

**逆向双指针法**

本题不能对两个数组合并后排序，这样时间复杂度比较高。

直接使用两个指针，分别指向两个数组有效位置的末尾，然后逐一比较元素大小，较大的元素填充到 nums1 的最后面即可，也就是从右往左进行逆向双指针遍历。整个过程直到其中一个数组合并完为止，最后如果其中一个还没合并完全，则再单独合并这个数组即可。

![](/zimg/yuque-1777880826223-5.gif)

代码如下：

```
public void merge(int[] nums1, int m, int[] nums2, int n) {
    if (nums1 == null || nums2 == null) {
        return;
    }

    // 🌟两个指针指向两个数组的末尾
    int idx1 = m - 1;
    int idx2 = n - 1;

    // 结果指针指向nums1的末尾, 从后往前填充结果
    int resIdx = nums1.length - 1;

    // 🌟从后往前合并, 退出循环的条件是其中一个数组已经合并完
    while (idx1 >= 0 && idx2 >= 0) {
        if (nums1[idx1] >= nums2[idx2]) {
            nums1[resIdx--] = nums1[idx1--];
        } else {
            nums1[resIdx--] = nums2[idx2--];
        }
        // 结果指针都需要移动
        resIdx--;
    }

    // 最后检查一下：如果两个数组还有剩余, 则直接填充到nums1中
    while (idx1 >= 0) {
        nums1[resIdx--] = nums1[idx1--];
    }
    while (idx2 >= 0) {
        nums1[resIdx--] = nums2[idx2--];
    }
}
```

### 三数之和(15-中等) 💯

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 `a, b, c`，使得 `a + b + c = 0`？请找出所有满足条件且不重复的三元组。注意：答案中**不可以包含重复的三元组**。

```
示例:
给定数组 nums = [-1, 0, 1, 2, -1, -4],  满足要求的三元组集合为:
[
 [-1, 0, 1],
 [-1, -1, 2]
]
```

[https://leetcode-cn.com/problems/3sum](https://leetcode-cn.com/problems/3sum)

**排序+双指针法**

这个题排序很关键！直接用系统的排序算法即可，`Arrays.sort()`。先将数组排序，然后遍历数组，作为第一个数，并在这个数的右半部分区间使用双指针进行查找，需要注意去重。

过程图解：

![](/zimg/yuque-1777880826831-6.gif)

具体流程如下：

- 首先对数组进行排序，排序后固定一个数 `nums[i]`，再使用左右指针指向 `nums[i]` 后面的两端，数字分别为 `nums[left]` 和 `nums[right]`，计算三个数的和 sum 判断是否满足为 0，满足则添加进结果集。
- 如果 `nums[i]` 大于 0，则三数之和必然无法等于 0，结束循环。
- 如果 `nums[i] == nums[i - 1]`，则说明该数字重复，会导致结果重复，所以应该跳过。
- 当 `sum == 0` 时，`nums[left] == nums[left + 1]` 则会导致结果重复，应该跳过，`left++`。
- 当 `sum == 0` 时，`nums[right] == nums[right − 1]` 则会导致结果重复，应该跳过，`right--`。

```
public static List<List<Integer>> threeSum(int[] nums) {
    if (Objects.isNull(nums) || nums.length < 3) {
        return Collections.emptyList();
    }

    List<List<Integer>> resList = new ArrayList<>();

    // 🌟对原始数组排序
    Arrays.sort(nums);

    // 遍历整个数组
    for (int i = 0; i < nums.length; i++) {
        // 如果当前数字大于0, 因为已经排序好, 所以后面不可能有三个数加和等于 0, 直接终止循环
        if (nums[i] > 0) {
            break;
        }

        // 🌟当前的值等于前一个元素的值, 那么得到的结果将会和前一次相同, 这里进行去重
        if (i > 0 && nums[i] == nums[i - 1]) {
            continue;
        }

        // 当前元素右边部分的左右指针
        int left = i + 1;
        int right = nums.length - 1;

        // 终止条件是左右指针相遇
        while (left < right) {
            // 三数求和
            int sum = nums[i] + nums[left] + nums[right];
            // 和为0则加入结果集
            if (sum == 0) {
                resList.add(Arrays.asList(nums[i], nums[left], nums[right]));
                // 同时移动两个指针, 因为已经找到了一个解
                left++;
                right--;

                // 🌟记录一次结果后持续移动两个指针去重剪枝!!! 注意是等于的时候剪枝
                while(left <= nums.length - 1 && nums[left] == nums[left - 1]) {
                    left++;
                }
                while(right >= 0 && nums[right] == nums[right + 1]) {
                    right--;
                }

                // 和小于0则左指针右移
            } else if (sum < 0) {
                left++;
                // 和大于0则右指针左移
            } else {
                right--;
            }
        }
    }

    return resList;
}
```

public List<List<Integer>> threeSum(int[] nums) {

            int sum = nums[i] + nums[left] + nums[right];

                right--;
            } else {
                left++;
            }
        }
    }

### 四数之和(18-中等)

给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 `a, b, c 和 d`，使得 `a + b + c + d` 的值与 target 相等？找出所有满足条件且不重复的四元组。答案中不可以包含重复的四元组。

```
示例: 给定数组 nums = [1, 0, -1, 0, -2, 2], 和 target = 0.
满足要求的四元组集合为:
[[-1,  0, 0, 1], [-2, -1, 1, 2], [-2,  0, 0, 2]]
```

[https://leetcode.cn/problems/4sum/description/](https://leetcode.cn/problems/4sum/description/)

**排序+双指针法**

四数之和与三数之和是一个思路，都是使用双指针法，基本解法就是在三数之和的基础上再套一层 for 循环，并且进行去重。

但有一些细节需要注意，例如不要判断 `nums[k] > target` 就返回了，三数之和可以通过 `nums[i] > 0` 就返回了，因为 0 已经是确定的数了，四数之和这道题目 target 是任意值。比如数组是 `[-4, -3, -2, -1]`，target 是 -10，不能因为 `-4 > -10` 而跳过。但是依旧可以去做剪枝，逻辑变成 `nums[i] > target && (nums[i] >= 0 || target >= 0)` 就可以了。

代码如下：

```
public List<List<Integer>> fourSum(int[] nums, int target) {
    List<List<Integer>> resList = new ArrayList<>();
    // 数组排序
    Arrays.sort(nums);

    // 第一个指针
    for (int i = 0; i < nums.length; i++) {
        // 剪枝操作: 当前元素 nums[i] > target 直接返回
        if (nums[i] > 0 && nums[i] > target) {
            return resList;
        }

        // 对nums[i]去重
        if (i > 0 && nums[i - 1] == nums[i]) {
            continue;
        }

        // 第二个指针
        for (int j = i + 1; j < nums.length; j++) {
            // 对 nums[j]去重
            if (j > i + 1 && nums[j - 1] == nums[j]) {
                continue;
            }

            // 使用双指针找剩下的两个元素
            int left = j + 1;
            int right = nums.length - 1;
            while (right > left) {
                // int 可能会溢出
                long sum = (long) nums[i] + nums[j] + nums[left] + nums[right];
                // 找到目标结果
                if (sum == target) {
                    resList.add(Arrays.asList(nums[i], nums[j], nums[left], nums[right]));
                    // 对 nums[left]和 nums[right]去重
                    while (right > left && nums[right] == nums[right - 1]) right--;
                    while (right > left && nums[left] == nums[left + 1]) left++;
                    // 同时移动两个指针
                    left++;
                    right--;
                    // 大于目标值, 右指针左移
                } else if (sum > target) {
                    right--;
                    // 小于目标值, 左指针右移
                } else {
                    left++;
                }
            }
        }
    }
    return resList;
}
```

### 最接近的三数之和(16-中等)

给定一个包括 n 个整数的数组 nums 和 一个目标值 target。找出 nums 中的三个整数，使得它们的和与 target 最接近。返回这三个数的和。假定每组输入只存在唯一答案。

```
给定数组 nums = [-1, 2, 1, -4], 和 target = 1. 与target最接近的三个数的和为 2. (-1 + 2 + 1 = 2).
```

[https://leetcode-cn.com/problems/3sum-closest](https://leetcode-cn.com/problems/3sum-closest)

**排序+双指针法**

三数之和题目的变形，解题思路基本一致。直接利用三数之和的排序+双指针方法进行求解。只是注意编码的问题。比如只需要找到目标值即可直接返回了，以及需要注意剪枝操作。

动图连接: [算法动图](https://leetcode-cn.com/problems/3sum-closest/solution/hua-jie-suan-fa-16-zui-jie-jin-de-san-shu-zhi-he-b/)

```
public int threeSumClosest(int[] nums, int target) {
    // 🌟对数组排序
    Arrays.sort(nums);
    int ans = Integer.MAX_VALUE;

    for (int i = 0; i < nums.length; i++) {
        // 剪枝操作
        if (i > 0 && nums[i] == nums[i - 1]) {
            continue;
        }

        // 🌟对当前索引位置后面的元素进行双指针遍历
        int left = i + 1;
        int right = nums.length - 1;

        // 不断移动当前下双指针
        while (left < right) {
            int sum = nums[i] + nums[left] + nums[right];
            // 如果刚好找到等于target的值, 则直接返回找到目标元素即可
            if (sum == target) {
                return sum;
            }

            // 🌟如果当前三数之和与target的距离值更小则更新ans值!!!
            // 注意这里比较的是两个数字到target的距离
            if (Math.abs(target - sum) < Math.abs(target - ans)) {
                ans = sum;
            }

            // 🌟然后根据情况移动左右指针
            if (sum > target) {
                right--;
            } else if (sum < target) {
                left++;
            }
        }
    }
    return ans;
}
```

### 判断子序列(392-中等) 💯

给定字符串 s 和 t，判断 s 是否为 t 的子序列。字符串的一个子序列是原始字符串删除一些(也可以不删除)字符而不改变剩余字符相对位置形成的新字符串。(例如 "`ace`" 是 "`abcde`" 的一个子序列，而 "`aec`" 不是)。两个字符串都只由小写字符组成。

```
输入: s = "abc", t = "ahbgdc"  输出: true
输入: s = "axc", t = "ahbgdc"  输出: false
```

[https://leetcode.cn/problems/is-subsequence/description/](https://leetcode.cn/problems/is-subsequence/description/)

**双指针法**

设置双指针 `i, j` 分别指向字符串 `s, t` 的首个字符，遍历字符串 t：

- 当 `s[i] == t[j]` 时，代表匹配成功，此时同时 `i++, j++`，进而若 i 已走过 s 尾部，代表 s 是 t 的子序列，此时应提前返回 true。
- 当 `s[i] != t[j]` 时，代表匹配失败，此时仅 `j++`，若遍历完字符串 t 后，字符串 s 仍未遍历完，代表 s 不是 t 的子序列，此时返回 false。

![](/zimg/yuque-1777880827670-7.gif)

代码如下：

```
public boolean isSubsequence(String s, String t) {
    if (s.isEmpty()) {
        return true;
    }
    int i = 0;  // 指向s的下标
    int j = 0;  // 指向t的下标

    while (j < t.length()) {
        // 如果s的当前字符和t的当前字符相等，则移动s的下标
        if (s.charAt(i) == t.charAt(j)) {
            i++;
            // 若已经遍历完s, 则提前返回true
            if (i == s.length()) {
                return true;
            }
        }
        // 无论是否相等，t的下标都要移动
        j++;
    }

    return false;
}
```

输入：s = "abpcplea", dictionary = ["a","b","c"]  输出："a"
```

[https://leetcode.cn/problems/longest-word-in-dictionary-through-deleting/](https://leetcode.cn/problems/longest-word-in-dictionary-through-deleting/)

**双指针法**

使用 res 来记录找到的最长的字符串，之后遍历字典中的所有字符串，如果当前字符串比最长字符串短或者字典序更大，则直接忽略。

最后判断当前字符串是否是一个合法的子串，是则更新结果。这里的判断方式就是通过删除字符串 s 中的一个字符能得到字符串 t，可以认为 t 是 s 的子序列，因此可以使用双指针来判断一个字符串是否为另一个字符串的子序列。`isSubsequence()` 就是上一题的解法。

```
public String findLongestWord(String s, List<String> dictionary) {
    String res = "";

    // 遍历找字典中的合法字符串
    for (String temp : dictionary) {
        // 寻找的是最长的字符串, 如果当前字符串比已经找到的字符串长度小则直接忽略
        if (temp.length() < res.length()) {
            continue;
        }
        // 当前字符串与已经找到的字符串长度相同, 则选择字典序小的
        if (temp.length() == res.length() && res.compareTo(temp) < 0) {
            continue;
        }
        // 判断一个字符串是否是给定字符串的子序列, 是则更新结果
        if (isSubsequence(temp, s)) {
            res = temp;
        }
    }
    return res;
}

public boolean isSubsequence(String s, String t) {
    if (s.isEmpty()) {
        return true;

    int i = 0;  // 指向s的下标
    int j = 0;  // 指向t的下标

    while (j < t.length()) {
        // 如果s的当前字符和t的当前字符相等，则移动s的下标

            i++;
            // 若已经遍历完s, 则提前返回true

                return true;
            }
        }
        // 无论是否相等，t的下标都要移动
        j++;
    }

    return false;
}

给定一个包含 `n + 1` 个整数的数组 nums，其数字都在 `[1, n]` 范围内（包括 1 和 n），可知至少存在一个重复的整数。假设 nums **只有 一个重复的整数**，返回这个重复的数。设计的解决方案必须 不修改 数组 nums 且只用常量级 `O(1)` 的额外空间。

```
输入：nums = [1,3,4,2,2]   输出：2
输入：nums = [3,1,3,4,2]   输出：3
输入：nums = [3,3,3,3,3]   输出：3
```

[https://leetcode.cn/problems/find-the-duplicate-number/description/](https://leetcode.cn/problems/find-the-duplicate-number/description/)

**双指针遍历法**

将这个题目给的特殊的数组**当作一个链表来看**，数组的下标就是指向元素的指针，把数组的元素也看作指针。如 0 是指针，指向 `nums[0]`，而 `nums[0]` 也是指针，指向 `nums[nums[0]]`。

假设有这样一个样例：`[1,2,3,4,5,6,7,8,9,5]`。如果按照上面的循环下去就会得到这样一个链表的路径：`1 2 3 4 5 [6 7 8 9] [6 7 8 9] [6 7 8 9]...` 这样就有了一个环，也就是 `6 7 8 9`。point 会一直在环中循环的前进。所以位置 5 就是进入环的元素，这样这个题目就**变成了求带环链表的第一个入口结点的问题了**！

![](/zimg/yuque-1777880828451-8.png)

解法就类似了，**设置两个一快(fast)一慢(slow)两个指针，一个每次走两步，一个每次走一步，直到两者相遇。然后慢指针重置，再同时走，后面相遇的地方就是入环的第一个结点，也就是重复的数字了**。

```
public int findDuplicate(int[] nums) {
    int slow = 0, fast = 0;
    // 快慢指针, 快指针走两步, 慢指针走一步
    do {
        slow = nums[slow];
        fast = nums[nums[fast]];
    } while (slow != fast);

    // 慢指针回到起点, 然后同时走, 相遇时即为环的入口
    slow = 0;

    while (slow != fast) {
        slow = nums[slow];
        fast = nums[fast];
    }

    return slow;
}
```

给你 n 个非负整数 `a1, a2, ..., an`，每个数代表坐标中的一个点 `(i, ai)`。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 `(i, ai) 和 (i, 0)`。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。说明：n 的值至少为 2。

![](/zimg/yuque-1777880828461-9.jpeg)

图中垂直线代表输入数组 `[1, 8, 6, 2, 5, 4, 8, 3, 7]`。在此情况下容器能够容纳水(表示为蓝色部分)的最大值为 49。

```
输入: [1,8,6,2,5,4,8,3,7] 输出: 49
```

[https://leetcode-cn.com/problems/container-with-most-water](https://leetcode-cn.com/problems/container-with-most-water)

**左右双指针移动短板法**

注意：**每次移动的是较短的那边的柱子**。

![](/zimg/yuque-1777880828474-10.gif)

在每个状态下，无论长板或短板向中间收窄一格，都会导致水槽底边宽度 −1 变短：

- **若向内 移动短板，水槽的短板** `**min(h[i], h[j])**` **可能变大，因此\*\***下个水槽的面积可能增大\*\*。
- **若向内 移动长板，水槽的短板** `**min(h[i], h[j])**` **不变或变小，因此下个水槽的面积 一定变小。**

因此初始化双指针分列水槽左右两端，循环每轮将短板向内移动一格，并**滚动更新面积最大值**，直到两指针相遇时跳出，即可获得最大面积。

```
public int maxArea(int[] nums) {
    if (nums == null || nums.length < 2) return 0;

    // 左右双指针
    int left = 0, right = nums.length - 1;
    // 暂存最大面积
    int maxArea = Integer.MIN_VALUE;

    // 左右指针未相遇, 这里不用等于是因为等于时面积为0
    while (left < right) {
        // 计算当前面积: 宽度为右指针减左指针, 高度为左右指针中的最小值
        maxArea = Math.max(maxArea, (right - left) * Math.min(nums[left], nums[right]));

        // 重要: 每次移动高度较小的指针
        if (nums[left] < nums[right]) {
            left++;
        } else {
            right--;
        }
    }
    return maxArea;
}
```

**左右双指针法**

数组已经有序的，不过负数平方之后可能成为最大数了。那么数组平方的最大值就在数组的两端，不是最左边就是最右边，不可能是中间。使用双指针分别指向数组的左右两边，哪个的绝对值更大就移动哪个指针，并从右到左更新结果数组。

动画示意如下：

![](/zimg/yuque-1777880828945-11.gif)

代码如下：

```
public int[] sortedSquares(int[] nums) {
    if (nums == null) {
        return null;
    }

    // 左右双指针指向左右两边
    int left = 0;
    int right = nums.length - 1;

    // 结果数组的大小与原始数组一致, 注意需要从后往前填充
    int[] res = new int[nums.length];
    int resIndex = res.length - 1;

    // 逐渐移动左右指针 退出条件是左指针大于右指针 因为左右指针相遇时也需要处理
    while (left <= right) {
        // 如果左边的绝对值大于右边的绝对值, 则左边的平方值更大
        if (Math.abs(nums[left]) >= Math.abs(nums[right])) {
            res[resIndex] = nums[left] * nums[left];
            left++;
        } else {
            res[resIndex] = nums[right] * nums[right];
            right--;
        }
        resIndex--;
    }
    return res;
}
```

### 先增后减数组中不重复数字个数

一个数组先递增再递减，找出数组不重复的个数，比如 `[1, 3, 9, 1]`，结果为 3，不能使用额外空间，复杂度 `o(n)`。

**左右双指针法**

使用左右双指针从两头向中间逼近，并使用 res 记录出现的数字的个数，如果遇到相同的数字，需要注意去重。

```
public int diffnum(int[] nums) {
    if (nums == null || nums.length == 0) return 0;
    // 左右双指针
    int left = 0;
    int right = nums.length - 1;

    int res = 0;

    // 逐渐移动左右指针 退出条件是左指针大于右指针 因为左右指针相遇时也需要处理
    while (left <= right) {
        if (nums[left] == nums[right]) {
            res++;
            int temp = nums[left];
            // 左右分别去重
            while (left <= right && nums[right] == temp) right--;
            while (left <= right && nums[left] == temp) left++;
        } else if (nums[left] < nums[right]) {
            res++;
            int temp = nums[left];
            // 去重
            while (left <= right && nums[left] == temp) left++;
        } else {
            res++;
            int temp = nums[right];
            // 去重
            while (left <= right && nums[right] == temp) right--;
        }
    }
    return res;
}
```

### 调整数组顺序使奇数位于偶数前面(简单) 🔑

调整数组顺序使奇数位于偶数前面。

![](/zimg/yuque-1777880829158-12.png)

**左右双指针法**

使用左右双指针，左边指针持续寻找偶数，右边指针持续寻找奇数，寻找到之后就交换两个元素，直到两个指针相遇。

![](/zimg/yuque-1777880829171-13.png)

代码如下：

```
public int[] exchange(int[] nums) {
    // 左右双指针
    int left = 0;
    int right = nums.length - 1;

    // 如果两个指针相遇, 则不需要处理 因此退出条件是左指针小于右指针
    while (left < right) {
        // 保证索引不越界的情况下从左找到一个偶数
        while (left < right && (nums[left] & 1) == 1) left++;
        // 保证索引不越界的情况下从右向左找到一个奇数
        while (left < right && (nums[right] & 1) == 0) right--;
        // 交换两个数
        swap(nums, left, right);
    }
    return nums;
}

private void swap(int[] nums, int left, int right) {
    int tmp = nums[left];
    nums[left] = nums[right];
    nums[right] = tmp;
}
```

```
public int[] twoSum(int[] numbers, int target) {
    if (numbers == null) {
        return null;
    }

    int left = 0, right = numbers.length - 1;

    while (left < right) {
        int sum = numbers[left] + numbers[right];
        // 找到目标值
        if (sum == target) {
            return new int[]{left + 1, right + 1};
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }

    return null;
}
```

[https://leetcode.cn/problems/reverse-string-ii/](https://leetcode.cn/problems/reverse-string-ii/)

**双指针法**

使用 for 循环寻找每次需要反转的区间的范围，然后使用双指针对这个区间内的元素进行反转。

在遍历字符串的过程中，让 `i += (2 * k)`，i 每次移动 `2 * k` 就可以了，然后判断是否需要有反转的区间。因为要找的也就是每 `2 * k` 区间的起点。所以**当需要固定规律一段一段去处理字符串的时候，可以在 for 循环的表达式上做做文章**。

public String reverseStr(String s, int k) {
    char[] chars = s.toCharArray();

    // 每次i增加2k, 并翻转2k这个区间的字符
    for (int i = 0; i < chars.length; i += 2 * k) {
        // 寻找需要反转的区间的左右范围
        // 左边界
        int left = i;
        // 右边界: 寻找前k个字符; 注意这里有可能最后剩下的字符串长度不足k 根据题意需要把剩下的字符串全部反转
        int right = Math.min(i + k - 1, chars.length - 1);

        // 反转这个区间内的字符
        while (left < right) {
            swap(chars, left++, right--);
        }
    }
    return new String(chars);
}

private void swap(char[] chars, int i, int j) {
    char tmp = chars[i];
    chars[i] = chars[j];
    chars[j] = tmp;
}
```

public String reverseOnlyLetters(String s) {
    char[] chars = s.toCharArray();
    // 左右双指针
    int left = 0, right = chars.length - 1;

    // 左右指针相遇时不需要处理, 因此指针相遇时结束循环
    while (left < right) {
        // 找到左边的字母
        while (left < right && !Character.isLetter(chars[left])) {
            left++;
        }
        // 找到右边的字母
        while (left < right && !Character.isLetter(chars[right])) {
            right--;
        }
        // 交换字母
        swap(chars, left, right);
        // 移动指针
        left++;
        right--;
    }
    return new String(chars);
}

private void swap(char[] chars, int left, int right) {
    char tmp = chars[left];
    chars[left] = chars[right];
    chars[right] = tmp;
}

```
public String reverseOnlyLetters(String str) {
    Stack<Character> letters = new Stack();
    // 将全部字母压栈
    for (char c: str.toCharArray()) {
        if (Character.isLetter(c)) {
            letters.push(c);
        }
    }
    // 构造结果
    StringBuilder ans = new StringBuilder();
    for (char c: str.toCharArray()) {

        if (Character.isLetter(c)) {
            ans.append(letters.pop());
        } else {
            // 其他字符直接加入到结果
            ans.append(c);
        }
    }
    return ans.toString();
}

使用左右双指针，每次对比指针的值，如果相同则持续移动指针直到两者相遇，否则由于只能删除一个字符，所以只需要验证删除左边或者右边字符之后剩余的部分是否还是回文串即可。

public boolean validPalindrome(String str) {
    if (str == null) {
        return false;
    }
    if (str.length() <= 1) {
        return true;
    }

    // 左右双指针
    int left = 0;
    int right = str.length() - 1;

    while (left <= right) {
        // 如果两个字符相同则逼近指针
        if (str.charAt(left) == str.charAt(right)) {
            left++;
            right--;
        } else {
            // 遇到不同的,由于只能删除一个字符,所以可以删左边或者右边
            // 此时分别看删左边还是右边,删除之后如果剩下的还是回文就 OK,否则就不行
            return isPalindrome(str, left + 1, right) || isPalindrome(str, left, right - 1);
        }
    }
    return true;
}

private boolean isPalindrome(String str, int i, int j) {
    while (i < j) {
        if (str.charAt(i++) != str.charAt(j--)) {
            return false;
        }
    }
    return true;
}

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，
可以接 6 个单位的雨水（蓝色部分表示雨水）。

**按行求值**

先找出最高的层数，然后每次求每一层可以装的水的数量。找的时候就是找出这一层的左右边界，然后再找空位。比较好理解，只是执行会超时。

![](/zimg/yuque-1777880829307-19.png)

```
// 每次计算一行的水量的方法
public int trap(int[] heights) {
    if (heights == null || heights.length == 0) {
        return 0;
    }
    // 获取最大高度
    int maxHeight = getMaxHeight(heights);

    int res = 0;
    for (int i = 1; i <= maxHeight; i++) {
        // 计算每一层可以接的雨水量
        res += getLevelCount(heights, i);
    }
    return res;
}

private int getLevelCount(int[] heights, int level) {
    int leftBorder = 0;
    int rightBorder = heights.length - 1;

    // 找到左右边界
    while (leftBorder <= heights.length && heights[leftBorder] < level) {
        leftBorder++;
    }

    while (rightBorder >= 0 && heights[rightBorder] < level) {
        rightBorder--;
    }

    if (leftBorder >= rightBorder) {
        return 0;
    }

    // 计算这一层的雨水量
    int count = 0;
    for (int i = leftBorder; i <= rightBorder; i++) {
        if (heights[i] < level) {
            count++;
        }
    }

    return count;
}

private int getMaxHeight(int[] heights) {
    int max = 0;
    for (int i = 0; i < heights.length; i++) {
        max = Math.max(max, heights[i]);
    }
    return max;
}
```

**⭐\*\***️ 双向单调栈法\*\*

面试用：解决这个问题的关键在于**找到每个位置左边和右边的最高柱子，然后计算当前位置能接住的雨水量**。具体步骤如下：

1. **找到每个位置左边的最高柱子：对于数组中的每个元素，找到它左边（包括自己）的最大值**。
2. **找到每个位置右边的最高柱子：对于数组中的每个元素，找到它右边（包括自己）的最大值。**
3. 计算每个位置能接住的雨水量：**对于每个位置，取左边最高柱子和右边最高柱子中的较小值，减去当前柱子的高度，就是该位置能接住的雨水量**。
4. 累加所有位置的雨水量：将所有位置的雨水量相加，得到最终的结果。

代码如下，这个解法比较好理解，其实就是**类似维护了左右两个单调栈，每次都能直接取到当前位置左右两边较大值的数据**。面试可用。

```
public int trap(int[] heights) {
    if (heights == null || heights.length == 0) {
        return 0;
    }

    // 从左到右, 记录每个元素左边出现的最大元素是多少
    int[] leftMaxes = new int[heights.length];
    leftMaxes[0] = heights[0];
    for (int i = 1; i < heights.length; i++) {
        leftMaxes[i] = Math.max(heights[i], leftMaxes[i - 1]);
    }

    // 从右到左, 记录每个元素右边出现的最大元素是多少
    int[] rightMaxes = new int[heights.length];
    rightMaxes[heights.length - 1] = heights[heights.length - 1];
    for (int i = heights.length - 2; i >= 0; i--) {
        rightMaxes[i] = Math.max(heights[i], rightMaxes[i + 1]);
    }

    int res = 0;

    for (int i = 0; i < heights.length; i++) {
        // 获取当前位置左边与右边出现的最大元素
        int leftMax = leftMaxes[i];
        int rightMax = rightMaxes[i];
        // 如果当前位置比左边或者右边任意一个大 说明兜不住雨水
        if (leftMax <= heights[i] || rightMax <= heights[i]) {
            continue;
        }

        // 去左右的较小者减去当前位置高度就是当前位置的雨水量
        res = res + Math.min(leftMax, rightMax) - heights[i];
    }

    return res;
}
```

## 滑动窗口题目

滑动窗口就是不断的调节子序列的起始位置和终止位置，从而得出要想的结果。

### 长度最小的子数组(209-中等) 💯

给定一个含有 n 个正整数的数组和一个正整数 s，找出该数组中满足其和大于等于 s 的长度最小的连续子数组，并返回其长度。如果不存在符合条件的子数组则返回 0。

```
示例: 输入: s = 7, nums = [2,3,1,2,4,3]  输出: 2
解释: 子数组 [4,3] 是该条件下的长度最小的子数组.
```

[https://leetcode.cn/problems/minimum-size-subarray-sum/description/](https://leetcode.cn/problems/minimum-size-subarray-sum/description/)

**滑动窗口**

暴力法是一个 for 循环滑动窗口的起始位置，一个 for 循环为滑动窗口的终止位置，用两个 for 循环实现不断搜索区间的过程。

那么滑动窗口如何用一个 for 循环来完成这个操作？首先要思考如果用一个 for 循环，那么应该表示滑动窗口的起始位置，还是终止位置。如果只用一个 for 循环来表示滑动窗口的起始位置，那么如何遍历剩下的终止位置？此时难免再次陷入暴力解法的怪圈。所以只用一个 for 循环，那么这个循环的索引，一定是表示滑动窗口的终止位置。

那么问题来了，滑动窗口的起始位置如何移动呢？举个例子 `s=7`, 数组是 `2, 3, 1, 2, 4, 3`，来看一下查找的过程：

![](/zimg/yuque-1777880829318-20.gif)

最后找到 `4, 3` 是最短距离。

可以发现滑动窗口也可以理解为双指针法的一种！只不过这种解法更像是一个窗口的移动，所以叫做滑动窗口更适合一些。在本题中实现滑动窗口，主要确定如下三点：

- 窗口内是什么？
- 如何移动窗口的起始位置？
- 如何移动窗口的结束位置？

窗口就是满足其和 `≥ s` 的长度最小的连续子数组。

窗口的起始位置如何移动：如果当前窗口的值大于等于 s 了，窗口就要向前移动了(也就是该缩小了)。

窗口的结束位置如何移动：窗口的结束位置就是遍历数组的指针，也就是 for 循环里的索引。

可以发现滑动窗口的精妙之处在于根据当前子序列和大小的情况，不断调节子序列的起始位置。从而将 `O(n^2)` 暴力解法降为 `O(n)`。

代码如下：

```
public int minSubArrayLen(int s, int[] nums) {
    // left和right表示窗口的左右边界
    int left = 0;
    int right = 0;
    // 记录窗口内的和
    int windowSum = 0;
    // 记录最小窗口大小
    int minWindow = Integer.MAX_VALUE;
    // 不断移动窗口右边界
    while (right < nums.length) {
        // 计算窗口内的和
        windowSum += nums[right];
        // 如果当前窗口内的元素和大于等于s则持续移动左边界, 减小窗口大小
        while (windowSum >= s) {
            // 不断更新最小值
            minWindow = Math.min(minWindow, right - left + 1);
            windowSum -= nums[left];
            left++;
        }
        right++;
    }
    return minWindow == Integer.MAX_VALUE ? 0 : minWindow;
}
```

public int[][] fileCombination(int target) {
    int left = 1;  // 滑动窗口的左边界(整数从1开始)
    int right = 1; // 滑动窗口的右边界
    int windowSum = 0; // 滑动窗口中数字的和
    List<int[]> resList = new ArrayList<>();

    while (right < target) {
        // 计算窗口内的和
        windowSum += right;
        // 如果当前窗口内的元素和大于等于target则持续移动左边界, 减小窗口大小
        while (windowSum >= target) {
            // 记录一次结果
            if (windowSum == target) {
                int[] res = new int[right - left + 1];
                for (int i = left; i <= right; i++) {
                    res[i - left] = i;
                }
                resList.add(res);
            }

            // 不断更新窗口左边界
            windowSum -= left;
            left++;
        }

        // 移动窗口右边界
        right++;
    }

    return resList.toArray(new int[resList.size()][]);
}

### 无重复字符的最长子串(3-中等) 💯🔑

找出一个字符串中不含有重复字符的最长子串的长度。

```
输入: s = "abcabcbb"  输出: 3  解释: 因为无重复字符的最长子串是 "abc", 所以其长度为 3.
输入: s = "bbbbb"     输出: 1
```

[https://leetcode.cn/problems/longest-substring-without-repeating-characters/description/](https://leetcode.cn/problems/longest-substring-without-repeating-characters/description/)

**滑动窗口法**

维护一个滑动窗口，窗口内的都是没有重复的字符，去尽可能的扩大窗口的大小，窗口不停的向右滑动。

- 如果当前遍历到的字符从未出现过，那么直接扩大右边界。
- 如果当前遍历到的字符出现过，则缩小窗口(左边索引向右移动)，然后继续观察当前遍历到的字符。

![](/zimg/yuque-1777880829570-22.gif)

代码如下：

```
public int lengthOfLongestSubstring(String s) {
    // 滑动窗口左右指针
    int left = 0, right = 0;
    // 记录窗口中字符出现的次数
    int[] windowCnt = new int[128];
    // 记录最长长度
    int maxUniqueLen = 0;

    while (right < s.length()) {
        // 右边界字符
        char rightChar = s.charAt(right);
        windowCnt[rightChar]++;
        // 移动窗口右边界
        right++;
        // 如果滑动窗口中出现重复字符, 则开始移动左边界缩小窗口
        while (windowCnt[rightChar] > 1) {
            char leftChar = s.charAt(left);
            windowCnt[leftChar]--;
            left++;
        }
        // 更新结果值
        maxUniqueLen = Math.max(maxUniqueLen, right - left);
    }
    return maxUniqueLen;
}
```

    int i = -1, res = 0;
    // 遍历字符串
    for (int j = 0; j < s.length(); j++) {
        // 如果map已经存在这个元素, 则更新左指针i

```
public int lengthOfLongestSubstring(String s) {
    if (s == null || s.length() <= 0) {
        return 0;
    }

    // 使用队列维护滑动窗口
    LinkedList<Character> queue = new LinkedList<>();
    int maxLen = 0;

    for (char c : s.toCharArray()) {
        // 队列中不包含当前字符, 直接加到队列中
        if (!queue.contains(c)) {
            queue.add(c);
            maxLen = Math.max(maxLen, queue.size());
            continue;
        }

        // 说明队列中包含当前字符, 则把队列里面之前出现过的当前字符全部弹出
        while (true) {
            if (queue.peek() == c) {
                queue.pop();
                break;
            }
            queue.pop();
        }
        // 再将当前字符加入队列中
        queue.add(c);
        maxLen = Math.max(maxLen, queue.size());
    }

    return maxLen;
}
```

### 找到字符串中的所有异位词(438-中等) 💯

给定两个字符串 s 和 p，找到 s 中所有 p 的异位词的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

```
输入: s = "cbaebabacd", p = "abc"  输出: [0,6]
解释: 起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
     起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。
```

[https://leetcode.cn/problems/find-all-anagrams-in-a-string/description](https://leetcode.cn/problems/find-all-anagrams-in-a-string/description)

**滑动窗口法**

因为字符串中的字符全是小写字母，**可以用长度为 26 的数组记录字母出现的次数**。

首先统计字符串 p 中每个字符出现的次数，然后统计 s 的前 `p.length()` 个子串中每个字符出现的次数。滑动窗口从 `p.length()` 的长度开始，**每次移动一个字符，移动后需要根据情况，减少左边界移除的字符个数，增加右边界新增的字符的个数**。每次移动后都进行个数的对比，匹配则找到一个异位词的索引。

注意寻找的是异位词，所以这个**滑动窗口是一个固定大小的窗口**，其大小为 `p.length()`。

```
// s = "cbaebabacd", p = "abc"
public List<Integer> findAnagrams(String s, String p) {
    int n = s.length(), m = p.length();
    List<Integer> res = new ArrayList<>();
    if (n < m) {
        return res;
    }

    // 记录p中每个字符出现的次数
    int[] pCnt = new int[26];
    // 记录s中每个字符出现的次数(注意这里只统计了前m个字符)
    int[] sCnt = new int[26];

    for (int i = 0; i < m; i++) {
        pCnt[p.charAt(i) - 'a']++;
        sCnt[s.charAt(i) - 'a']++;
    }

    // 如果初始时两个数组相等则加入结果中
    if (Arrays.equals(sCnt, pCnt)) {
        res.add(0);
    }

    // 滑动窗口, 从m开始遍历即可, 对比的是 [m - p.length(), m] 这个滑动区间的字符个数
    for (int i = m; i < n; i++) {
        // 由于是滑动窗口, 需要将窗口左边界的字符个数减少一个, 右边界的字符个数增加一个
        sCnt[s.charAt(i - m) - 'a']--;
        sCnt[s.charAt(i) - 'a']++;

        // 对比两个数组是否相等, 相等则说明是异位词
        if (Arrays.equals(sCnt, pCnt)) {
            res.add(i - m + 1);
        }
    }

    return res;
}
```

public List<Integer> findAnagrams(String s, String p) {
    List<Integer> resList = new ArrayList<>();
    if (s == null || p == null || p.length() > s.length()) {
        return resList;
    }

    // 初始化目标字符频率数组
    int[] target = new int[26];
    for (char c : p.toCharArray()) {
        target[c - 'a']++;
    }

    // 使用s的前p.length()个字符初始化滑动窗口
    int[] window = new int[26];
    for (int i = 0; i < p.length(); i++) {
        window[s.charAt(i) - 'a']++;
    }

    // 检查初始窗口是否满足
    if (isMatch(target, window)) {
        resList.add(0);
    }

    // 滑动窗口
    for (int i = p.length(); i < s.length(); i++) {
        // 移出左边界字符, 减少对应的频率
        window[s.charAt(i - p.length()) - 'a']--;
        // 移入右边界字符, 增加对应的频率
        window[s.charAt(i) - 'a']++;
        // 检查当前窗口是否满足
        if (isMatch(target, window)) {
            resList.add(i - p.length() + 1);
        }
    }

    return resList;
}

private boolean isMatch(int[] target, int[] window) {
    for (int i = 0; i < 26; i++) {
        if (window[i] != target[i]) {
            return false;
        }
    }
    return true;
}

### 最小覆盖子串(76-困难) 💯

给你一个字符串 s、一个字符串 t。返回 s 中涵盖 t 所有字符的最小子串。如果 s 中不存在涵盖 t 所有字符的子串，则返回空字符串 ""。

```
输入：s = "ADOBECODEBANC", t = "ABC"   输出："BANC"
解释：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
```

[https://leetcode.cn/problems/minimum-window-substring/description/](https://leetcode.cn/problems/minimum-window-substring/description/)

**滑动窗口法**

枚举 s 子串的右端点 right（子串最后一个字母的下标），如果子串涵盖 t，就不断移动左端点 left 直到不涵盖为止。在移动过程中更新最短子串的左右端点。

具体来说：

- 枚举 s 子串的右端点 right(子串最后一个字母的下标)，如果子串涵盖 t，就不断移动左端点 left 直到不涵盖为止。在移动过程中更新最短子串的左右端点。
- **使用长度为 128 的数组 target 统计字符串 t 中每个字符的出现次数**(覆盖所有 ASCII 字符)。
- 使用另一个长度为 128 的数组 window **动态记录当前滑动窗口中各字符的出现次数**。
- 双指针初始化：`left` 和 `right` 指针初始化为 0，表示窗口的左右边界。
- **窗口扩展**：`right` 指针向右移动，将 `chars[right]` 加入窗口(window 中对应字符计数加 1)。
- **窗口持续收**：每当窗口满足覆盖 t 所有字符的条件时，则持续尝试收缩 left 指针以寻找更小的窗口，并**动态更新最短子串** resStr(如果当前窗口更小)的结果值，同时将 `chars[left]` 移出窗口(`window` 中对应字符计数减 1)，再将 left 右移收缩窗口。

```
public String minWindow(String s, String t) {
    char[] chars = s.toCharArray();
    String resStr = s;

    // 统计中字母的出现次数
    int[] target = new int[128];
    for (char c : t.toCharArray()) {
        target[c]++;
    }

    // 记录滑动窗口内字母的出现次数
    int[] window = new int[128];

    // 记录是否覆盖过, 如果没有覆盖过则返回空字符串
    boolean isCover = false;

    int left = 0;
    int right = 0;

    while (right < s.length()) {
        // 右端点新的字母加入滑动窗口, 窗口内对应的字母出现次数加1
        window[chars[right]]++;
        // 校验涵盖, 如果覆盖看看是不是最短的情况, 是则记录起来
        while (checkIsCovered(window, target)) {
            isCover = true;
            // 找到更短的子串, 更新结果
            if (right - left < resStr.length()) {
                resStr = s.substring(left, right + 1);
            }
            // 左端点字母移出子串
            window[chars[left]]--;
            left++;
        }

        right++;
    }

    return isCover ? resStr : "";
}

private boolean checkIsCovered(int[] window, int[] target) {
    for (int i = 'A'; i <= 'Z'; i++) {
        if (window[i] < target[i]) {
            return false;
        }
    }

    for (int i = 'a'; i <= 'z'; i++) {
        if (window[i] < target[i]) {
            return false;
        }
    }

    return true;
}
```
