## 总结

### 技巧

- 递归解深度相关的题，**可以设置一个全局的最大深度**，用于记录已经访问到的最大深的深度。
- 掌握二叉树的**迭代层序遍历**，很多题可用！只需少量修改即可。一般层序遍历的递归解法的模板也是差不多的。
- 掌握二叉树的序列化与反序列化方法，可用用于子树问题的判断。
- 递归遍历的时候，**如果需要改变顺序，可以考虑先递归右树**。比如中序遍历二叉搜索树，先左后右是升序，先右后左就是降序。
- 求深度的题目可以用递归，也可以用层序遍历进行迭代求解。
- 暴力递归类似于后序遍历，也就是默认能得到左右子树的信息再构造自己的信息，树形 dp。
- 可以考虑使用 HashMap 存储位置关系。
- 通过数组递归构造二叉树是有套路模板的。
- **如果只是访问数据，可以直接先序递归二叉树；如果需要从叶子往上计算到根节点，也就是自底向上的递归，可以使用后序递归方式；如果是涉及裁剪操作，一般也是后续递归**。
- 自底向上进行递归时，可以考虑使用一个辅助方法，这个方法的返回值需要定义好。
- 递归方法中，如果传入的是一个链表，StringBuilder 等对象，在递归完后需要进行回溯！！

### 树的遍历

树的遍历方式总体分为两类：深度优先搜索(DFS)，广度优先搜索(BFS)。

- 常见的 DFS：先序遍历、中序遍历、后序遍历。
- 常见的 BFS：层序遍历(即按层遍历)。

### 二叉搜索树

搜索二叉树利用其特殊的结构，往往有独特的解法。二叉搜索树递归时如果涉及到比较，可以每次直接砍掉一半不用递归。

**二叉搜索树的中序遍历是递增的，如果依然是中序遍历的模板，但是先遍历右子树，再处理数据，再遍历左子树就会是降序**。

### 二叉树构造与变换类题目

可以考虑使用 HashMap 存储位置关系。

通过数组递归构造二叉树是有套路模板的。

### Trie 树

Trie，又称前缀树或字典树，用于判断字符串是否存在或者是否具有某种字符串前缀。

### ⭐️ 二叉树迭代前序遍历(144-中等)

给定二叉树的根节点 root，返回它节点值的 前序 遍历。

[https://leetcode-cn.com/problems/binary-tree-preorder-traversal/](https://leetcode-cn.com/problems/binary-tree-preorder-traversal/)

**辅助栈模拟递归法**

迭代法本质上是在模拟递归，因为递归过程使用了系统栈，所以在迭代解法中常用 Stack 来模拟系统栈。

**使用栈，先将根节点压栈，只要栈不为空则一直弹出。每次弹出一个结点进行处理，然后将弹出结点的右结点压栈，再压入左结点(由于是利用栈，所以压栈顺序相反是先右后左)**。

![](/zimg/yuque-1777880835467-4.gif)

代码如下：

```
public List<Integer> preorderTraversal(TreeNode root) {
    if(root == null) {
        return new ArrayList<>();
    }

    // 🌟使用辅助栈并把根节点压栈
    Stack<TreeNode> stack = new Stack<>();
    stack.push(root);

    List<Integer> resList = new ArrayList<>();

    // 🌟不断从栈中弹出元素弹出时处理数据, 直到栈为空
    while(!stack.isEmpty()) {
        // 🌟先序遍历则弹出时就处理数据
        TreeNode node = stack.pop();
        resList.add(node.val);

        // 🌟注意: 先压入右结点, 再压入左结点, 这样弹出就是先左后右
        if(node.right != null) stack.push(node.right);
        if(node.left != null) stack.push(node.left);
    }

    return resList;
}
```

### ⭐️ 二叉树迭代中序遍历(94-中等) 💯

给定二叉树的根节点 root，返回它节点值的 中序 遍历。

[https://leetcode-cn.com/problems/binary-tree-inorder-traversal/](https://leetcode-cn.com/problems/binary-tree-inorder-traversal/)

**辅助栈模拟递归法**

递归的调用过程是这样的：

```
dfs(root.left)
	dfs(root.left)
		dfs(root.left)
			为null返回
		打印节点
		dfs(root.right)
			dfs(root.left)
				dfs(root.left)
				...
```

递归的调用过程是不断往左边走，当左边走不下去了，就弹出并处理节点，然后转向右边结点继续这个过程。算法过程如下：

- **先将根节点入栈**。
- **将当前节点的所有左子结点入栈，直到左子树为空**。
- **弹出栈顶元素，如果栈顶元素存在右子结点，则变换到右子节点继续第 2 步**。
- **重复第 2，3 步，直到栈为空并且所有的节点都被访问**。

![](/zimg/yuque-1777880835508-5.gif)

**使用 Stack 存储结点，注意先把左结点全部压栈，然后弹出一个结点进行处理，然后把结点指向右结点继续处理**。代码如下：

```
public List<Integer> inorderTraversal(TreeNode root) {
    // 使用辅助栈
    Stack<TreeNode> stack = new Stack<>();

    List<Integer> resList = new ArrayList<>();

    // 退出条件是栈不为空且root不为null
    while (root != null || !stack.isEmpty()) {
        // 🌟将当前结点的全部左子结点压栈, 模拟递归调用
        if (root != null) {
            stack.push(root);
            root = root.left;
        } else {
            // 🌟当前节点为空, 说明左子树走到头, 从栈中弹出节点 此时开始处理结点
            TreeNode temp = stack.pop();
            resList.add(temp.val);
            // 🌟处理完当前结点后转向右边节点, 继续上面整个过程
            root = temp.right;
        }
    }

    return resList;
}
```

### 二叉树迭代后序遍历(145-困难)

输出二叉树后序遍历。

[https://leetcode-cn.com/problems/binary-tree-postorder-traversal/](https://leetcode-cn.com/problems/binary-tree-postorder-traversal/)

**双栈法**

使用两个栈 s1 和 s2。前序遍历的过程是中左右。将其转化成中右左。也就是压栈的过程中优先压入左子树，再压入右子树。然后将这个结果返回来，再利用栈的先进后出倒序打印，即是后序遍历。

1. 将根节点压入 s1 中。
2. 从 s1 中弹出元素，并依次将其左孩子和右孩子压入 s1 中。此过程中每从 s1 弹出一个元素都将其压入到栈 s2 中。直到 s1 为空为止。
3. 从 s2 中依次弹出元素处理即是后序遍历。

![](/zimg/yuque-1777880835731-6.gif)

代码如下：

```
public List<Integer> postorderTraversal(TreeNode root) {
    if (root == null) return new ArrayList<>();

    // 准备2个辅助栈
    Stack<TreeNode> stack1 = new Stack<>();
    Stack<TreeNode> stack2 = new Stack<>();
    // 根节点压入栈1
    stack1.push(root);

    // 退出条件为栈1为空
    while (!stack1.isEmpty()) {
        // 栈1每弹出一个元素就放入栈2
        root = stack1.pop();
        stack2.add(root);

        // 再将当前弹出元素的左右子节点分别入栈, 一定是先左再右
        if (root.left != null) stack1.push(root.left);
        if (root.right != null) stack1.push(root.right);
    }

    LinkedList<Integer> resList = new LinkedList<>();
    while (!stack2.isEmpty()) {
        resList.add(stack2.pop().val);
    }

    return resList;
}
```

这里**可以利用 LinkedList 来存储结果，每次让结点值插入当前结果头部即可实现倒序打印的效果。也就可以少用一个栈了。\*\***面试用此法\*\*。

```
public List<Integer> postorderTraversal(TreeNode root) {
    if (root == null) return new ArrayList<>();
    LinkedList<Integer> resList = new LinkedList<>();

    // 使用辅助栈
    Stack<TreeNode> stack = new Stack<>();
    // 根节点压入栈1
    stack.push(root);

    // 退出条件为栈1为空
    while (!stack.isEmpty()) {
        // 栈1每弹出一个元素就放入栈2
        root = stack.pop();
        // 结点值插入结果链表头部
        resList.addFirst(root.val);

        // 再将当前弹出元素的左右子节点分别入栈,一定是先左再右
        if (root.left != null) stack.push(root.left);
        if (root.right != null) stack.push(root.right);
    }
    return resList;
}
```

### N 叉树的后序遍历(590-简单)

使用后序遍历方式遍历一颗 N 叉树。后序遍历顺序为：`[5, 6, 3, 2, 4, 1]`。

![](/zimg/yuque-1777880835775-7.png)

多叉树结点类 Node 如下。

```
public class Node {
	public int val;
	public List<Node> children;
	public Node() {}
	public Node(int val) {
		this.val = val;
	}
	public Node(int val, List<Node> children) {
		this.val = val;
		this.children = children;
	}
}
```

[https://leetcode-cn.com/problems/n-ary-tree-postorder-traversal/](https://leetcode-cn.com/problems/n-ary-tree-postorder-traversal/)

**递归法 DFS**

直接使用递归的方式进行遍历。与普通递归后序遍历类似，不过这里是遍历各个子结点，最后再处理结果。

```
private List<Integer> resList = new LinkedList<>();

public List<Integer> postorder(Node root) {
    // 深度优先遍历
    dfs(root);
    return resList;
}

private void dfs(Node root) {
    // Base case
    if(root == null) return;

    // 整体后序遍历的架构, 先遍历全部子结点最后再处理数据
    for(Node child : root.children) {
        dfs(child);
    }

    resList.add(root.val);
}
```

**迭代法**

写法与 [二叉树迭代后序遍历(145-困难)](https://www.yuque.com/daerwendecha/qwuv92/gmi10ot8m7wa1ugt#xAxUO) 十分类似。

使用一个辅助栈来得到后序遍历。首先把根节点入栈。当每次从栈顶取出一个节点时，就把该结点的所有子节点顺序压入栈中。例如子节点从左到右为 `v1，v2，v3`，那么入栈顺序应为 `v1，v2，v3`，这样就保证了下一个遍历到的节点(即第一个子节点 `v3`)出现在栈顶的位置。在遍历结束之后，把遍历结果反转，就可以得到后序遍历。

```
public List<Integer> postorder2(Node root) {
    // Base case
    if (root == null) return new ArrayList<>();

    LinkedList<Integer> resList = new LinkedList<>();

    // 使用一个辅助栈并压入根结点
    Stack<Node> stack = new Stack<>();
    stack.add(root);

    // 结束条件是栈为空
    while (!stack.isEmpty()) {
        // 栈中弹出一个结点并处理
        Node node = stack.pop();
        // 注意这里是往头部添加实现逆序, 这样最后就无需单独进行逆序操作了! !
        resList.addFirst(node.val);

        // 顺序压入其他子节点
        for (Node item : node.children) {
            if (item != null) {
                stack.add(item);
            }
        }
    }
    return resList;
}
```

### ⭐️⭐️ 二叉树的层序遍历 I(102-中等) 💯🔑

层序遍历二叉树。

![](/zimg/yuque-1777880835786-8.png)

```
输出:
[
  [3],
  [9,20],
  [15,7]
]
```

[https://leetcode-cn.com/problems/binary-tree-level-order-traversal](https://leetcode-cn.com/problems/binary-tree-level-order-traversal)

**迭代法 BFS**

对二叉树的层序遍历就是使用 BFS 遍历，可以使用队列进行辅助。

![](/zimg/yuque-1777880835799-9.gif)

**下面的模板非常有用，记住可以解很多题**。

- **根节点入队**。
- **循环直到队列为空，每次出队一个结点，并将其左右子节点入队**。
- **为了控制按层进行打印，需要有个循环记录一下这一层的结点个数，所以使用一个 for 循环来遍历这一层**。
- **当一层的元素处理完的时候，下一层的结点又已经全部入队，持续循环即可**。

```
public List<List<Integer>> levelOrder(TreeNode root) {
    // Base case
    if(root == null) {
        return new ArrayList<>();
    }

    List<List<Integer>> resList = new ArrayList<>();

    // 使用辅助队列并将根节点入队列
    Queue<TreeNode> queue = new LinkedList<>() ;
    queue.add(root);

    // 队列为空时才退出循环
    while(!queue.isEmpty()) {
        // 获取当前队列中的元素个数, 也就是当前这一层有多少个结点, 这是为了控制仅访问当前层的结点
        int size = queue.size();

        List<Integer> levelList = new ArrayList<>();

        // 将当前层的元素依次出队列并处理数据, 每弹出一个结点并再次将其左右子节点入队列
        for(int i = 0; i < size; i++) {
            // 出队列并处理数据
            TreeNode temp = queue.poll();
            levelList.add(temp.val);
            // 将出队列结点的左右子节点(如果有)依次入队
            if(temp.left != null) queue.add(temp.left);
            if(temp.right != null) queue.add(temp.right);
        }

        // 这一层弹出的元素处理完毕
        resList.add(levelList);
        // 此时下一层的全部节点已经全部加入到队列, 再次循环即可
    }
    return resList;
}
```

**递归法 DFS**

下面是递归的层序遍历模板。使用了递归方法 `dfs(node, level)`，参数是**当前节点和节点的层次**。整个过程如下：

- 输出列表称为 resList，当前最高层数就是列表的长度 `size()`。比较访问节点所在的层次 level 和当前最高层次 size 的大小，如果两者相同就说明访问到了新的 level 则向 resList 初始化添加一个空列表用于存放这一层的所有元素。
- 将当前节点插入到对应层的列表 `resList[level]` 中。
- 递归所有孩子节点并把层数 + 1。

```
public List<List<Integer>> levelOrder(TreeNode root) {
    if (root == null) {
        return resList;
    }

    // 从根结点开始递归, 根结点是第0层
    dfs(root, 0);

    return resList;
}

List<List<Integer>> resList = new ArrayList<>();

// 辅助DFS方法:整体是先序遍历的模板
public void dfs(TreeNode node, int level) {
    // Base case
    if (node == null) {
        return;
    }

    // 说明遇到了新的一层则新建一个列表, 添加一个新的ArrayList用于存放当前层的元素
    if (resList.size() == level) {
        resList.add(new ArrayList<>());
    }

    // 根据层数获取对应的列表并将当前节点的值加入到list中(先序遍历)
    resList.get(level).add(node.val);

    // 分别递归左右子结点
    if (node.left != null) dfs(node.left, level + 1);
    if (node.right != null) dfs(node.right, level + 1);
}
```

### 二叉树的层序遍历 II(107-简单)

给定一个二叉树，返回其节点值自底向上的层次遍历。即按从叶子节点所在层到根节点所在的层，逐层从左向右遍历。例如：给定二叉树 `[3, 9, 20, null, null, 15, 7]`。

```
	3
   / \
  9  20
    /  \
   15   7
```

输出：

```
[
  [15,7],
  [9,20],
  [3]
]
```

[https://leetcode-cn.com/problems/binary-tree-level-order-traversal-ii](https://leetcode-cn.com/problems/binary-tree-level-order-traversal-ii)

**层序遍历法**

思路类似前面的普通层序遍历，但是是反过来的，其实只需要把二叉树的层序遍历**结果逆序**即可。参考上述的 BFS 解法，只需在**最后调用逆序方法即可**。

```
Collections.reverse(resList);
```

最后将结果逆序即可。

```
public List<List<Integer>> levelOrderBottom(TreeNode root) {
    // Base case
    if(root == null) {
        return new ArrayList<>();
    }

    // 下面是层序遍历模板
    List<List<Integer>> resList = new ArrayList<>();

    Queue<TreeNode> queue = new LinkedList<>() ;
    queue.add(root);

    while(!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> levelList = new ArrayList<>();
        for(int i = 0; i < size; i++) {
            TreeNode temp = queue.poll();
            levelList.add(temp.val);
            if(temp.left != null) queue.add(temp.left);
            if(temp.right != null) queue.add(temp.right);
        }
        resList.add(levelList);
    }

    // 最后逆转一下结果
    Collections.reverse(resList);
    return resList;
}
```

### N 叉树的层序遍历(429-中等)

给定一个 N 叉树，返回其节点值的层序遍历，即从左到右，逐层遍历。例如给定一个 3 叉树 ：

![](/zimg/yuque-1777880835812-10.png)

返回其层序遍历：

```
[
   [1],
   [3,2,4],
   [5,6]
]
```

[https://leetcode-cn.com/problems/n-ary-tree-level-order-traversal/](https://leetcode-cn.com/problems/n-ary-tree-level-order-traversal/)

**迭代法 BFS**

参考普通的层序遍历方法，也是使用一个队列，区别是每个结点出队列时就将其全部子元素放入队列中。

详细图示：

[https://leetcode-cn.com/problems/n-ary-tree-level-order-traversal/solution/ncha-shu-de-ceng-xu-bian-li-by-leetcode/](https://leetcode-cn.com/problems/n-ary-tree-level-order-traversal/solution/ncha-shu-de-ceng-xu-bian-li-by-leetcode/)

```
public List<List<Integer>> levelOrder(Node root) {
    if(root == null) {
        return new ArrayList<>();
    }

    // 下面是层序遍历模板
    List<List<Integer>> resList = new ArrayList<>();

    Queue<Node> queue = new LinkedList<>() ;
    queue.add(root);

    while(!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> levelList = new ArrayList<>();
        for(int i = 0; i < size; i++) {
            Node temp = queue.poll();
            levelList.add(temp.val);
            // 将当前结点的全部子结点加入队列
            for (Node child : temp.children) {
                queue.add(child);
            }
        }
        resList.add(levelList);
    }

    return resList;
}
```

**递归 DFS**

这里递归的方式也是类似普通二叉树的层序遍历方式。

通常不能使用递归进行广度优先搜索。这是因为广度优先搜索基于队列，而递归运行时使用堆栈，适合深度优先搜索。但可以以不同的顺序添加到最终列表中，只要知道节点在哪一层并确保在那一层的列表顺序正确就可以了。

```
private List<List<Integer>> resList = new ArrayList<>();

public List<List<Integer>> levelOrder(Node root) {
    // 从第0层开始
    if (root != null) {
        dfs(root, 0);
    }
    return resList;
}

// 递归方法 传入层数
private void dfs(Node node, int level) {
    // 说明当前层大于结果List的size, 需要新建一个list来存储当前层的元素
    if (resList.size() == level) {
        resList.add(new ArrayList<>());
    }

    // 获取当前level所对应的list, 并处理当前节点的数据加入到后面
    resList.get(level).add(node.val);

    // 依次递归全部子节点, level + 1
    for (Node child : node.children) {
        dfs(child, level + 1);
    }
}

**层序遍历法 BFS**

类似二叉树的层序遍历的广度优先算法，只不过每次处理的数据的时候**仅记录这一层的最后一个结点**即可。

```
public List<Integer> rightSideView(TreeNode root) {
    if(root == null) {
        return new ArrayList<>();
    }

    List<Integer> resList = new ArrayList<>();

    queue.add(root);

        int size = queue.size();
        for(int i = 0; i < size; i++) {

            if (i == size - 1) {
                resList.add(temp.val);

            if(temp.left != null) queue.add(temp.left);
            if(temp.right != null) queue.add(temp.right);
        }
    }

        return;

    if (resList.size() == level) {
        resList.add(root.val);

### 二叉树的层平均值(637-简单) 💯

给定一个非空二叉树，返回一个由每层节点平均值组成的数组。

```
    3
   / \
  9  20
    /  \
   15   7
```

输出：`[3, 14.5, 11]`。解释：第 0 层的平均值是 3，第 1 层是 14.5，第 2 层是 11。

[https://leetcode-cn.com/problems/average-of-levels-in-binary-tree](https://leetcode-cn.com/problems/average-of-levels-in-binary-tree)

**迭代法 BFS**

参考层序遍历二叉树的代码，**求每一层的平均值即可**。在层序遍历模板代码上小改即可。

```
public List<Double> averageOfLevels(TreeNode root) {
    if(root == null) {
        return new ArrayList<>();
    }

    List<Double> resList = new ArrayList<>();

    // 根节点入队列
    Queue<TreeNode> queue = new LinkedList<>();
    queue.add(root);

    // 队列为空时退出循环
    while(queue.size() != 0) {
        int size = queue.size();
        // 保存这一层的节点数值的和
        double sum = 0;
        for(int i = 0; i < size; i++) {
            // 出队列并处理数据
            TreeNode temp = queue.poll();
            // 转换成double类型
            sum = sum + temp.val;
            // 将出队结点的左右子节点入队
            if(temp.left != null) queue.add(temp.left);
            if(temp.right != null) queue.add(temp.right);
        }
        // 一层处理完成, 计算层平均值
        resList.add(sum / size);
    }
    return resList;
}
```

**递归 DFS**

可以使用深度优先搜索模板遍历整颗二叉树，得到整棵树的全部结点。然后遍历每层的结点值计算平均值即可。

        double sum = 0;
        for (Integer num : levelList) {
            sum = sum + num;
        }
        resList.add(sum / levelList.size());
    }

    return resList;
}

    }

```
public List<Integer> largestValues(TreeNode root) {
    if(root == null) {
        return Collections.emptyList();
    }

    // 层序遍历模板
    List<Integer> resList = new ArrayList<>();
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while(!queue.isEmpty()) {
        int size = queue.size();

            if(temp.left != null) queue.offer(temp.left);
            if(temp.right != null) queue.offer(temp.right);

    return resList;
}

给定一个完美二叉树，其所有叶子节点都在同一层，每个父节点都有两个子节点。二叉树定义如下：

![](/zimg/yuque-1777880835855-14.png)

[https://leetcode.cn/problems/populating-next-right-pointers-in-each-node/description/](https://leetcode.cn/problems/populating-next-right-pointers-in-each-node/description/)

**层序遍历**

**遍历每一层的时候，找到这一层的第一个结点，然后逐一遍历这层的结点，进行指针的关联**。注意关联的时候需要取出每层第一个结点然后再进行类似链表的链接关联。代码如下：

```
public Node connect(Node root) {
    if (root == null) {
        return null;
    }

    Queue<Node> queue = new LinkedList<>();
    queue.add(root);

    // 队列为空时才退出循环
    while (!queue.isEmpty()) {
        int size = queue.size();
        // 找到一层的第一个结点
        Node first = queue.peek();
        Node cur = first;
        // 将当前层的元素依次出队列并建立指针连接
        for (int i = 0; i < size; i++) {
            // 出队列并建立指针
            Node temp = queue.poll();
            // 注意这里是i != 0, 因为第一个结点不需要指向自己
            if (i != 0) {
                cur.next = temp;
                cur = cur.next;
            }

            // 将出队列结点的左右子节点(如果有)依次入队
            if(temp.left != null) queue.add(temp.left);
            if(temp.right != null) queue.add(temp.right);
        }
    }

    return root;
}
```

**递归法**

使用前序递归模板，每一层的结点天然在链表内就是顺序排的，每次处理某一层时，直接拿到这一层的最后的结点，将其 next 指向当前结点即可。

```

        return root;
    }

    // 从根结点开始递归, 根结点是第0层
    dfs(root, 0);

    return root;
}

    List<Node> levelList = resList.get(level);
    // 如果当前层结点不为空, 则取出本层最后一个结点将其next指向当前结点即可

    // 分别递归左右子结点

### 二叉树的下一个结点 🔑

给定一个二叉树和其中的一个结点，请找出**中序遍历顺序的下一个结点并且返回**。注意这里树中的结点不仅包含左右子结点，同时包含指向父结点的指针。

```
private static class Node {
    int val;
    Node left = null;
    Node right = null;
    Node parent = null;

    Node(int val) {
        this.val = val;
    }
}
```

**解题思路**

可以直接用中序遍历找一波，但是这样估计就白给了。由于有**指向父节点的指针，所以是可以利用**的。

① 如果一个节点的右子树不为空，那么该节点的下一个节点是右子树的最左节点。

![](/zimg/yuque-1777880835880-16.gif)

② **否则，向上找第一个左链接指向的树包含该节点的祖先节点**。

![](/zimg/yuque-1777880835889-17.gif)

```
public Node getNext(Node pNode) {
    // 如果右子树不为空, 那么中序遍历的下一个结点是右子树的最左结点
    if (pNode.right != null) {
        // 找右子树最左的节点并返回
        return getLeftNode(pNode.right);

        // 否则向上找第一个左链接指向的树包含该节点的祖先节点
    } else {
        // 这里是排除当前节点是最后一个节点的情况
        while (pNode.parent != null) {
            // 得到父节点
            Node parent = pNode.parent;
            // 如果当前节点是其父节点的左节点就算找到了
            if (parent.left == pNode) {
                return parent;
            }
            // 否则继续向上层的父节点查找
            pNode = pNode.parent;
        }
    }
    // 没找到说明是最后一个节点, 返回null
    return null;
}

private Node getLeftNode(Node node) {
    if (node == null) {
        return null;
    }

    while (node.left != null) {
        node = node.left;
    }

    return node;
}

### 判断是否是完全二叉树(958-中等)

给定一个二叉树，确定它是否是一个完全二叉树。

```
       1
      / \
     2   3
    / \   \
   4   5   6
  / \
 7   8
```

[https://leetcode-cn.com/problems/check-completeness-of-a-binary-tree/](https://leetcode-cn.com/problems/check-completeness-of-a-binary-tree/)

**层序遍历法**

层序遍历，**只要出现过非满节点，后面结点只能是叶子节点**。

- 按层遍历，每层都从左到右进行遍历(所以整体是层序 BFS 遍历的框架，只是加了一些逻辑)。
- 如果当前节点有右子节点但没有左子节点，直接返回 false。
- **维护一个标志位，标志遇到第一个不是左右孩子都有的结点时打开**，即这个结点之后的都应该是叶子结点。如果当前节点并不是左右孩子都有，那么之后的节点必须全都为叶子结点，否则返回 false。

注意这里是连贯操作，所以无需用普通层序遍历中的 for 循环去遍历每层了。

```
public boolean isCompleteTree(TreeNode root) {
    if (root == null) {
        return true;
    }

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    // 是否遇到了非全结点标志
    boolean isMeetNonFullLeaf = false;

    // 队列不为空则继续处理
    while (!queue.isEmpty()) {
        // 弹出一个结点
        TreeNode node = queue.poll();

        // 普通结点左空, 右不空则不合法
        if (node.left == null && node.right != null) {
            return false;
        }

        // 遇到非nonFull结点后, 当前结点的左右子结点只能为空才合法
        if (isMeetNonFullLeaf && (node.left != null || node.right != null)) {
            return false;
        }

        // 遇到第一个不是全有左右子节点的结点 则后面的全部节点必须为叶子结点
        if (node.left == null || node.right == null) {
            isMeetNonFullLeaf = true;
        }

        // 左右结点加入队列
        if (node.left != null) queue.offer(node.left);
        if (node.right != null) queue.offer(node.right);
    }

    return true;
}
```

再来一种层序遍历的解法：**利用完全二叉树层序遍历时节点连续出现的性质，遍历到 null 之后再遍历到非 null 则说明不连续**。按照 BFS 层序遍历的方式遍历完全二叉树，队列最后留下的应该都是空指针。因此，需要将叶子结点的左右结点(即 null)也入队列。这样即可实现判断。

![](/zimg/yuque-1777880835898-18.png)

因此如果层序遍历过程中遇到了 null 结点则做个记录，遇到 null 结点后两种情况：

- **后面全是 null 结点，是完全二叉树**。
- **再次遇到非 null 结点，不是完全二叉树**。

```
public boolean isCompleteTree(TreeNode root) {
    if(root == null) {
        return true;
    }

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    // 是否遇到null结点
    boolean isMeetNull = false;

    while(!queue.isEmpty()) {
        // 弹出一个结点
        TreeNode node = queue.poll();

        // 首次遇到为null则做个记录
        if (node == null) {
            isMeetNull = true;
        } else {
            // 到这里说明null之后再次遇到非空结点
            if (isMeetNull) {
                return false;
            }
            // ⭐️ 注意这里node是非null结点, 左右子结点可能为空 此时加入的是null结点
            queue.offer(node.left);
            queue.offer(node.right);
        }
    }
    return true;
}
```

给定一个二叉树，判断它是否是高度平衡的二叉树。本题中一棵高度平衡二叉树定义为：一个二叉树每个节点的左右两个子树的高度差的绝对值不超过 1。

返回 true 。

 4   4

public boolean isBalanced(TreeNode root) {
    if (root == null) {
        return true;
    }

    depth(root);
    return balance;
}

// 是否平衡的标志位
boolean balance = true;

private int depth(TreeNode root) {
    // 这里剪枝 如果已经非平衡了直接返回不计算了
    if (root == null || !balance) {
        return 0;
    }

    // 计算左右子树高度
    int leftDepth = depth(root.left);
    int rightDepth = depth(root.right);

    // 动态更新：发现不平衡更新标志位
    if (Math.abs(leftDepth - rightDepth) > 1) {
        balance = false;

### 二叉树的最大深度(104-简单) 💯🔑

求一棵二叉树的深度。从根节点到叶节点的最长路径长度为树的深度。

![](/zimg/yuque-1777880836190-20.png)

[https://leetcode-cn.com/problems/maximum-depth-of-binary-tree/](https://leetcode-cn.com/problems/maximum-depth-of-binary-tree/)

**递归法 DFS(后序遍历)**

关键点：**树的深度等于左子树的深度 与右子树的深度中的最大值 + 1**。此处方法的返回值就只有深度值本身，直接用这个方法就能实现递归而无需定义辅助方法了。

```
public int maxDepth(TreeNode root) {
    if (root == null) {
        return 0;
    }

    // 计算左右子树高度
    int leftDepth = maxDepth(root.left);
    int rightDepth = maxDepth(root.right);

    // 返回自身高度最大高度: 即左右中较大者+1
    return Math.max(leftDepth, rightDepth) + 1;
}
```

简化一点：

```
public int maxDepth(TreeNode root) {
    if(root == null) return 0;
    return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
}
```

一句话：

```
public int maxDepth(TreeNode root) {

**层序遍历(BFS)**

二叉树的深度也就是层数。使用层序遍历模板，每遍历一层，则计数器 +1，直到遍历完成，则可得到树的深度。

```
public int maxDepth(TreeNode root) {
    if (root == null) {
        return 0;
    }

    // 深度
    int depth = 0;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while(!queue.isEmpty()) {
        // 到了新的一层, 层数+1
        depth++;
        int size = queue.size();
        for(int i = 0; i < size; i++) {
            TreeNode temp = queue.poll();
            if(temp.left != null) queue.offer(temp.left);
            if(temp.right != null) queue.offer(temp.right);
        }
    }
    return depth;
}
```

**自顶向下递归法**

整体就是先序遍历的递归模板，递归后优先处理结点，记录最深的层数。

```
public int maxDepth(Node root) {
    if (root == null) return 0;
    dfs(root, 1);
    return maxDepth;

int maxDepth = 0;

private void dfs(Node root, int level) {
    if (root == null) {
        return;
    }

    if (root.children == null || root.children.isEmpty()) {
        maxDepth = Math.max(maxDepth, level);
    }

    // 递归所有子树
    for (Node node : root.children) {
        dfs(node, level + 1);
    }
}

public int maxDepth(Node root) {

    if(root == null) {
        return 0;

    if(root.children == null || root.children.isEmpty()) {
        return 1;

    int max = Integer.MIN_VALUE;

    for(Node head : root.children) {
        max = Math.max(max, maxDepth(head));
    }

    // 返回各个子树最大高度+1
    return max + 1;
}

### 二叉树的最小深度(111-简单)

找出一颗二叉树的最小深度。最小深度是从根节点到最近叶子节点的最短路径上的节点数量。示例给定二叉树：

```
    3
   / \
  9  20
    /  \
   15   7
```

返回它的最小深度 2。

[https://leetcode-cn.com/problems/minimum-depth-of-binary-tree](https://leetcode-cn.com/problems/minimum-depth-of-binary-tree)

**层序遍历 BFS**

较优的解法。深度优先搜索的递归方法的缺陷是所有节点都必须访问。因此复杂度是 `O(N)`。利用广度优先搜索，按照树的层序遍历，**第一个访问到的叶子结点就是最小深度的节点**，这样就不用遍历所有的节点了。

```
public int minDepth(TreeNode root) {
    if(root == null) {
        return 0;
    }

    // 最小深度
    int minDepth = 0;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while(!queue.isEmpty()) {
        // 新的层深度+1
        minDepth++;

        int size = queue.size();
        for(int i = 0; i < size; i++) {
            TreeNode temp = queue.poll();

            // 当前结点是叶子结点 则直接返回
            if (temp.left == null && temp.right == null) {
                return minDepth;
            }

            if(temp.left != null) queue.offer(temp.left);
            if(temp.right != null) queue.offer(temp.right);
        }
    }
    return minDepth;
}
```

**递归法 DFS**

维护一个最低层数的变量，递归时每次遇到叶子节点的时候判断一下是否需要更新最小深度。

```
public int minDepth(TreeNode root) {
    if(root == null) return 0;
    // 深度从1开始
    dfs(root, 1);
    return minDepth;
}

// 记录最小层数
int minDepth = Integer.MAX_VALUE;

private void dfs(TreeNode root, int level) {
    if(root == null) return;

    // 如果当前是一个叶子节点就判断是否需要更新最小深度
    if(root.left == null && root.right == null) {
        minDepth = Math.min(minDepth, level);
    }

    dfs(root.left, level + 1);
    dfs(root.right, level + 1);
}
```

**自底向上递归法**

直接使用**自底向上的递归方式**，也就是类似后序遍历的方式，先处理左右再处理自己。

```
public int minDepth(TreeNode root) {
    // Base case
    if(root == null) {
        return 0;
    }

    // 直接获取左边的信息和右边信息
    int minLeft = minDepth(root.left);
    int minRight = minDepth(root.right);

    // 构造自己的返回信息
    if(minLeft != 0 && minRight == 0) {
        return minLeft + 1;
    }
    if(minLeft == 0 && minRight != 0) {
        return minRight + 1;
    }

    // 左右高度较小的+1
    return Math.min(minLeft, minRight) + 1;
}
```

### 相同的树(100-简单) 💯

![](/zimg/yuque-1777880836226-22.png)

[https://leetcode-cn.com/problems/same-tree](https://leetcode-cn.com/problems/same-tree)

**递归法**

这个题是典型的操作两棵树的题目，直接使用递归的方式进行判断即可。整体是先序遍历。注意这里需要分别**递归比较两棵树的左左和右右两个部分**。

```
public boolean isSameTree(TreeNode root1, TreeNode root2) {
    // Base case 如果两个都为空返回true
    if(root1 == null && root2 == null) return true;
    // 如果其中一个结点为空或者两个的值不同, 返回false
    if((root1 == null && root2 != null) || (root1 != null && root2 == null) || (root1.val != root2.val)) {
        return false;
    }
    // 递归获取左右子树信息
    boolean left = isSameTree(root1.left, root2.left);
    boolean right = isSameTree(root1.right, root2.right);
    // 必须左右子树都是true才为true
    return left && right;
}
```

### 对称二叉树(101-简单) 💯🔑

判断一颗二叉树是否是镜像对称的。例如下面二叉树是对称的。

![](/zimg/yuque-1777880836239-23.png)

[https://leetcode-cn.com/problems/symmetric-tree](https://leetcode-cn.com/problems/symmetric-tree)

**递归法 DFS**

递归思路如下：

![](/zimg/yuque-1777880836252-24.gif)

参考解答：

[https://leetcode.cn/problems/symmetric-tree/solution/dong-hua-yan-shi-101-dui-cheng-er-cha-shu-by-user7/](https://leetcode.cn/problems/symmetric-tree/solution/dong-hua-yan-shi-101-dui-cheng-er-cha-shu-by-user7/)

递归的辅助方法需要传入左右两个结点进行对比，只有都为空或者都不为空且值相同才合法。递归下一层时，需要**分别对比(左左，右右)，(左右，右左)两个组合**的结果。

```
public boolean isSymmetric(TreeNode root) {
    if (root == null) return true;
    // 调用递归函数, 比较左节点, 右节点
    return dfs(root.left, root.right);
}

boolean dfs(TreeNode left, TreeNode right) {
    // 递归的终止条件是两个节点都为空
    // 或两个节点中有一个为空
    // 或两个节点的值不相等
    if (left == null && right == null) return true;
    if (left == null || right == null) return false;
    if (left.val != right.val) return false;

    // 再递归比较左节点的左孩子和右节点的右孩子
    // 以及比较左节点的右孩子和右节点的左孩子
    return dfs(left.left, right.right) && dfs(left.right, right.left);
}
```

简单写法 2：

维护一个**单独的 symmetric 变量**用于记录是否对称。

- 如果递归中左右子结点有一个为空一个不为空或者左右子结点值不相同，则不对称，此时直接返回即可。
- **分别递归对比 (左左，右右) (左右，右左)**。
- 剪枝操作：如果已经发现不对称则直接返回。

```
public boolean isSymmetric(TreeNode root) {
    if (root == null) {
        return true;
    }

    // 这里对比的是左右子树, 所以dfs方法传入左右两个结点
    dfs(root.left, root.right);
    return symmetric;
}

boolean symmetric = true;

private void dfs(TreeNode left, TreeNode right) {
    // 左右子结点均为空
    // 剪枝: 如果已经发现不对称则直接返回
    if (left == null && right == null || !symmetric) {
        return;
    }

    // 左右子结点有一个为空一个不为空或者左右子结点值不相同则认为不对称, 标记一下
    if ((left == null && right != null) || (left != null && right == null) || left.val != right.val) {
        symmetric = false;
        return;
    }

    // 分别递归对比两次 (左左, 右右) (左右, 右左)
    dfs(left.left, right.right);
    dfs(left.right, right.left);
}
```

**双结点迭代法**

图示如下：

![](/zimg/yuque-1777880836338-25.gif)

**使用普通层序遍历的模板，注意这里的写法，由于是需要对比两个结点，所以需要将一个结点的左右子结点都同时加入队列中，弹出时也弹出两个结点进行比较**。

```
public boolean isSymmetric(TreeNode root) {
    if (root == null || (root.left == null && root.right == null)) {
        return true;
    }

    LinkedList<TreeNode> queue = new LinkedList<>();
    // 将根节点的左右孩子放到队列中, 注意这里一次加入两个节点
    queue.add(root.left);
    queue.add(root.right);

    while (!queue.isEmpty()) {
        // 从队列中取出左右两个节点, 再比较这两个节点
        TreeNode left = queue.removeFirst();
        TreeNode right = queue.removeFirst();

        // 如果两个节点都为空就继续循环
        if (left == null && right == null) {
            continue;
        }
        // 两者有一个为空或者值不同就直接返回false
        if (left == null || right == null) {
            return false;
        }
        if (left.val != right.val) {
            return false;
        }

        // 将左节点的左孩子, 右节点的右孩子放入队列
        queue.add(left.left);
        queue.add(right.right);
        // 将左节点的右孩子, 右节点的左孩子放入队列
        queue.add(left.right);
        queue.add(right.left);
    }

    return true;
}
```

### 叶子相似的树(872-简单)

一颗二叉树上所有叶子的值按从左到右的顺序排列形成一个叶子值序列。

![](/zimg/yuque-1777880836425-26.png)

举个例子如上图所示，给定一颗叶值序列为 (`6, 7, 4, 9, 8`) 的树。如果有两颗二叉树的叶值序列是相同，那么就认为它们是叶相似的。判断两棵树是否是叶相似的。

[https://leetcode-cn.com/problems/leaf-similar-trees](https://leetcode-cn.com/problems/leaf-similar-trees)

**DFS 遍历叶子节点法**

**首先分别找出给定的两个树的叶值序列**。之后可以比较它们看看是否相等。要找出树的叶值序列，可以使用 DFS。如果结点是叶子，那么 dfs 方法会写入结点的值，然后递归地探索每个子结点。这可以保证按从左到右的顺序访问每片叶子，因为在右孩子结点之前完全探索了左孩子结点。

```
public boolean leafSimilar(TreeNode root1, TreeNode root2) {
    List<Integer> leaves1 = new ArrayList<>();
    List<Integer> leaves2 = new ArrayList<>();

    dfs(root1, leaves1);
    dfs(root2, leaves2);

    return leaves1.equals(leaves2);
}

// 深度优先遍历树
public void dfs(TreeNode node, List<Integer> values) {
    if (node == null) {
        return;
    }
    // 先序遍历方式逐一保存叶子节点值
    if (node.left == null && node.right == null) {
        values.add(node.val);
    }
    // 递归
    dfs(node.left, values);
    dfs(node.right, values);
}
```

### 单值二叉树(965-简单)

如果**二叉树每个节点都具有相同的值**，那么该二叉树就是单值二叉树。判断一颗二叉树是否是单值二叉树。

[https://leetcode-cn.com/problems/univalued-binary-tree/](https://leetcode-cn.com/problems/univalued-binary-tree/)

**迭代法**

BFS 遍历写法。这里直接整体遍历一遍，**遇到与根结点不同的值就返回 false**。这样可以避免访问所有的结点。

```
public boolean isUnivalTree3(TreeNode root) {
    if (root == null)
        return false;

    // 记录根结点值
    int rootValue = root.val;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while(!queue.isEmpty()) {
        TreeNode temp = queue.poll();
        // 遇到不同值直接退出
        if (temp.val != rootValue) {
            return false;
        }
        if(temp.left != null) queue.offer(temp.left);
        if(temp.right != null) queue.offer(temp.right);
    }
    return true;
}
```

**递归法**

一棵树是单值的，说明后面的每一个结点值都与根节点值一样。首先保存根节点值，然后依次判断是不是相同的即可。整体是先序遍历。

```
int value;
public boolean isUnivalTree(TreeNode root) {
    if(root == null) return false;
    // 保存根节点的值
    value = root.val;
    return process(root);
}

private boolean process(TreeNode root) {
    if(root == null) {
        return true;
    }

    // 遇到差异值
    if(root.val != value) {
        return false;
    }

    // 获取左右子树的信息
    boolean left = process(root.left);
    boolean right = process(root.right);

    // 必须左右子树都为true才是true
    return left && right;
}
```

再来一种自顶向下的递归写法。

```
// 独立变量
boolean res = true;

int rootVal = 0;

public boolean isUnivalTree(TreeNode root) {
    if(root == null) return false;
    rootVal = root.val;
    // 首先保存根节点的值
    dfs(root);
    return res;
}

private void dfs(TreeNode root) {
    // Base case
    // 剪枝: 如果已经非单值直接返回
    if(root == null || !res) return;
    // 遇到不同值
    if(root.val != rootVal) res = false;
    // 获取左右子树的信息
    dfs(root.left);
    dfs(root.right);
}
```

### 二叉树左子节点的和(404-简单)

计算给定二叉树的所有左叶子之和。示例：

![](/zimg/yuque-1777880836472-27.png)

在这个二叉树中，有两个左叶子，分别是 9 和 15，所以返回 24。

[https://leetcode-cn.com/problems/sum-of-left-leaves](https://leetcode-cn.com/problems/sum-of-left-leaves)

**迭代法**

使用层序遍历的模板，不过每次弹出结点时，判断其左子树是否是叶子结点，是的话就累加一下该值。

```
public int sumOfLeftLeaves(TreeNode root) {
    if (root == null) {
        return 0;
    }

    Queue<TreeNode> queue = new LinkedList<>();
    queue.add(root);

    int sum = 0;

    while (!queue.isEmpty()) {
        TreeNode node = queue.poll();

        // 左子结点不为空且左子结点为一个叶子结点则累加一下
        if (node.left != null && (node.left.left == null && node.left.right == null)) {
            sum = sum + node.left.val;
        }

        // 左右子结点不为空则入队列
        if (node.left != null) queue.add(node.left);
        if (node.right != null) queue.add(node.right);
    }

    return sum;
}
```

**递归法**

整体先序遍历模板。这里辅助方法维护一个当前结点是左还是右结点的变量，如果是左结点且是叶子结点，则累加上。

```
public int sumOfLeftLeaves(TreeNode root) {
    dfs(root, false);
    return sum;
}

private int sum;

// 递归整体先序遍历的架构
private void dfs(TreeNode root, boolean isLeft) {
    if (root == null) {
        return;
    }

    // 当前节点是叶子节点且是左结点则累加
    if (root.left == null && root.right == null && isLeft) {
        sum = sum + root.val;
    }

    // 递归左子树时传入true, 右子树传入false
    dfs(root.left, true);
    dfs(root.right, false);
}
```

### 完全二叉树的节点个数(222-简单) 💯

求出一个完全二叉树的节点个数。

![](/zimg/yuque-1777880836492-28.png)

输出：6。

[https://leetcode.cn/problems/count-complete-tree-nodes/description/](https://leetcode.cn/problems/count-complete-tree-nodes/description/)

**完全二叉树特征法**

对于普通二叉树求结点数，可以用递归方法，也就是分别求结点的左子树结点数 + 右子树结点数 + 1。这样会全遍历一次，这里利用完全二叉树的特征可以更方便的结题。

在完全二叉树中，如果递归向左遍历的深度等于递归向右遍历的深度，那说明就是满二叉树。如图：

![](/zimg/yuque-1777880836509-29.png)

如果递归向左遍历的深度不等于递归向右遍历的深度，则说明不是满二叉树，如图：

![](/zimg/yuque-1777880836530-30.png)

因此，递归的方法流程如下：

- 首先分别判断结点的左右遍历的深度，根据结果分为两种情况来递归。
- (1) 如果左右遍历深度相同，则以当前结点为根结点的二叉树就是满二叉树，此时用公式计算此棵树的结点数即可。
- (2) 如果左右遍历深度相同，则以当前结点为根结点的二叉树不是满二叉树，此时就只能使用普通的计算二叉树结点数的方法，老老实实计算左子树结点数 + 右子树结点数 + 1。

参考下图：

![](/zimg/yuque-1777880836551-31.png)

代码如下：

```
public int countNodes(TreeNode root) {
    if (root == null) {
        return 0;
    }
    TreeNode left = root.left;
    TreeNode right = root.right;

    // 分别计算当前结点最左边子树和最右边子树的深度
    int leftDepth = 1, rightDepth = 1;

    while (left != null) {
        left = left.left;
        leftDepth++;
    }
    while (right != null) {
        right = right.right;
        rightDepth++;
    }

    // 下面根据左右深度的情况走两个逻辑
    // (1) 左右深度相同, 说明是满二叉树, 直接根据公式计算结点个数 就不去递归了
    if (leftDepth == rightDepth) {
        // 利用完全二叉树的特征, (2^leftDepth - 1)即为以当前结点为根结点的满二叉树的结点个数
        return (int) Math.pow(2, leftDepth) - 1;
    }

    // (2) 左右深度不相同, 说明不是满二叉树, 则按照普通递归方式计算左右子树的结点个数并相加
    int leftCnt = countNodes(root.left);
    int rightCnt = countNodes(root.right);
    return leftCnt + rightCnt + 1;
}
```

### 另一棵树的子树(简单-572) 🔑

输入两棵二叉树 A，B，判断 B 是不是 A 的子结构。注意：空树不是任意一个树的子结构。

![](/zimg/yuque-1777880836570-32.png)

[https://leetcode.cn/problems/subtree-of-another-tree/description/](https://leetcode.cn/problems/subtree-of-another-tree/description/)

**递归对比子树法**

递归 A 的每个结点，然后分别以**这个结点为根结点构成的树与 B 树是否完全一致**，如果任意一个结点匹配成功则任务存在子结构。

因此这里原方法对 A 树进行每个子结点的递归，并使用辅助递归方法判断两棵树是否完全相同。

```
public boolean isSubtree(TreeNode root, TreeNode subRoot) {
    if (subRoot == null) {
        return false;
    }

    // 判断以当前节点root为根的子树是否和子树subRoot相同
    boolean isSame = isSameTree(root, subRoot);
    if (isSame) {
        return true;
    }

    // 如果不同, 且root不为空, 则递归处理root的左子节点和右子节点, 去判断更深层的子树是否与subRoot相同
    if (root == null) {
        return false;
    }
    return isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot);
}

// 递归判断两棵树是否完全相同
private boolean isSameTree(TreeNode root1, TreeNode root2) {
    // 两个节点都为空, 结构相同则相同
    if (root1 == null && root2 == null) {
        return true;
    }
    // 一个节点为空一个不为空, 或者两个节点不为空(结构相同)但值不同, 都是不同的
    if (root1 == null || root2 == null || root1.val != root2.val) {
        return false;
    }
    // 否则当前位置的节点相同, 分别比较它们的左右子节点是否相同
    return isSameTree(root1.left, root2.left) && isSameTree(root1.right, root2.right);
}
```

### 二叉树的最近公共祖先(236-中等) 💯

给定一个二叉树，找到该树中两个指定节点的最近公共祖先。最近公共祖先的定义为：对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大(一个节点也可以是它自己的祖先)。

例如，给定如下二叉树：

![](/zimg/yuque-1777880836583-33.png)

```
输入: root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 1   输出: 3
输入: root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 4   输出: 5
```

说明：所有节点的值都是唯一的。p，q 为不同节点且均存在于给定的二叉树中。

[https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/description/](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/description/)

**自底向上递归法**

一般递归是自顶向下递归的，这里要是能自底向上查找就好了，这样就可以找到公共祖先了。那么二叉树如何可以自底向上查找呢？回溯！二叉树回溯的过程就是从低到上。后序遍历(左右中)就是天然的回溯过程，也就是自底向上的递归过程。这样可以根据左右子树的返回值，来处理中节点的逻辑。

接下来就看如何判断一个节点是节点 q 和节点 p 的公共祖先。

首先最容易想到的一个情况：如果找到一个节点，发现左子树出现结点 p，右子树出现节点 q，或者左子树出现结点 q，右子树出现节点 p，那么该节点就是节点 p 和 q 的最近公共祖先。即情况一：

![](/zimg/yuque-1777880836600-34.png)

判断逻辑是：如果递归遍历遇到 q，就将 q 返回，遇到 p 就将 p 返回，那么如果左右子树的返回值都不为空，说明此时的中节点，一定是 q 和 p 的最近祖先。

那会不会左子树遇到 q 返回，右子树也遇到 q 返回，这样并没有找到 q 和 p 的最近祖先。这里就要审题了，题目强调：二叉树节点数值是不重复的，而且一定存在 q 和 p。

有一个容易忽略的情况，就是节点本身 p(q)，它拥有一个子孙节点 q(p)。情况二：

![](/zimg/yuque-1777880836620-35.png)

其实情况一 和 情况二 代码实现过程都是一样的，实现情况一的逻辑，也顺便包含了情况二。因为遇到 q 或者 p 就返回，这样也包含了 q 或者 p 本身就是公共祖先的情况。

递归三部曲：

**确定递归函数返回值以及参数**

需要递归函数返回值，来告诉我们是否找到节点 q 或者 p，那么返回值为 bool 类型就可以了。但还要返回最近公共节点，可以利用题目中返回值是 TreeNode，那么如果遇到 p 或者 q，就把 q 或者 p 返回，返回值不为空，就说明找到了 q 或者 p。

代码如下：

```
TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q)
```

**确定终止条件**

遇到空的话，因为树都是空了，所以返回空。

那么来说一说，如果 `root == q`，或者 `root == p`，说明找到 q p，则将其返回，这个返回值，后面在中节点的处理过程中会用到，那么中节点的处理逻辑，下面讲解。

代码如下：

```
if (root == q || root == p || root == null) return root;
```

**确定单层递归逻辑**

值得注意的是本题函数有返回值，是因为回溯的过程需要递归函数的返回值做判断，但本题依然要遍历树的所有节点。

递归函数有返回值就是要遍历某一条边，但有返回值也要看如何处理返回值！如果递归函数有返回值，如何区分要搜索一条边，还是搜索整个树呢？

搜索一条边的写法：

```
if (递归函数(root.left)) return ;
if (递归函数(root.right)) return ;
```

搜索整个树写法：

```
left = 递归函数(root.left);   // 左
right = 递归函数(root.right); // 右
left 与 right 的逻辑处理;      // 中
```

在递归函数有返回值的情况下：如果要搜索一条边，递归函数返回值不为空的时候，立刻返回，如果搜索整个树，直接用一个变量 left，right 接住返回值，这个 left，right 后序还有逻辑处理的需要，也就是后序遍历中处理中间节点的逻辑(也是回溯)。

那为什么要遍历整棵树呢？直观上来看，找到最近公共祖先，直接一路返回就可以了。

如图：

![](/zimg/yuque-1777880836635-36.png)

就像图中一样直接返回 7。

但事实上还要遍历根节点右子树(即使此时已经找到了目标节点了)，也就是图中的节点 `4, 15, 20`。

因为在如下代码的后序遍历中，如果想利用 left 和 right 做逻辑处理，不能立刻返回，而是要等 left 与 right 逻辑处理完之后才能返回。

```
left = 递归函数(root.left);   // 左
right = 递归函数(root.right); // 右
left 与 right 的逻辑处理;      // 中
```

所以需要知道这里要遍历整棵树。知道这一点，对本题就有一定深度的理解了。

那么先用 left 和 right 接住左子树和右子树的返回值，代码如下：

```
TreeNode left = lowestCommonAncestor(root.left, p, q);
TreeNode right = lowestCommonAncestor(root.right, p, q);
```

如果 left 和 right 都不为空，说明此时 root 就是最近公共节点。这个比较好理解。

如果 left 为空，right 不为空，就返回 right，说明目标节点是通过 right 返回的，反之也一样。

那为什么 left 为空，right 不为空，目标节点通过 right 返回呢？如下图：

![](/zimg/yuque-1777880836653-37.png)

图中节点 10 的左子树返回 null，右子树返回目标值 7，那么此时节点 10 的处理逻辑就是把右子树的返回值(最近公共祖先 7)返回上去！那么如果 left 和 right 都为空，则返回 left 或者 right 都是可以的，也就是返回空。

那么寻找最小公共祖先，完整流程图如下：

![](/zimg/yuque-1777880836676-38.png)

从图中可以看到是如何回溯遍历整棵二叉树，将结果返回给头结点的！

整体代码如下：

```
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    // 递归结束条件, root为空或者找到了p或者q
    if (root == null || root == p || root == q) {
        return root;
    }

    // 整体后序遍历
    // left和right分别接住了左右子树找到的公共祖先结点
    TreeNode left = lowestCommonAncestor(root.left, p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);

    // 若未找到节点 p 或 q
    if (left == null && right == null) {
        return null;
        // 如果 left 为空, right 不为空, 就返回 right
        // 说明目标节点是通过 right 返回的, 反之也一样
    } else if (left == null && right != null) {
        return right;
    } else if (left != null && right == null) {
        return left;
        // 如果 left 和 right 都不为空, 说明此时 root 就是最近公共节点
        // 因为是自底向上递归的!!!!
    } else {
        return root;
    }
}
```

下面总结如下三点：

1. 求最小公共祖先，需要从底向上遍历，那么二叉树，只能通过后序遍历(即：回溯)实现从底向上的遍历方式。
2. 在回溯的过程中，必然要遍历整棵二叉树，即使已经找到结果了，依然要把其他节点遍历完，因为要使用递归方法的返回值(也就是代码中的 left 和 right)做逻辑判断。
3. 要理解如果返回值 left 为空，right 不为空为什么要返回 right，为什么可以用返回 right 传给上一层结果。

可以说这里每一步，都是有难度的，都需要对二叉树，递归和回溯有一定的理解。

## 二叉搜索树题目

### 判断是否是二叉搜索树(98-中等) 💯

给定一个二叉树，判断其是否是一个有效的二叉搜索树。

[https://leetcode-cn.com/problems/validate-binary-search-tree](https://leetcode-cn.com/problems/validate-binary-search-tree)

**递归中序遍历法**

二叉搜索树中序遍历是递增的，可以利用这个特点，对比中序遍历的值是否满足这个条件。直接使用中序遍历模板。使用一个变量记录中序遍历中前一个结点的值，并与当前结点值进行对比即可。

```
// 记录前一个值
private long preValue = Long.MIN_VALUE;

public boolean isValidBST(TreeNode root) {
    // Base case
    if (root == null) {
        return true;
    }

    // 递归左子树
    boolean isLeftValid = isValidBST(root.left);
    if (!isLeftValid) {
        return false;
    }

    // 访问当前节点: 如果当前节点小于等于中序遍历的前一个节点, 说明不满足BST, 返回 false;否则继续遍历
    if (root.val <= preValue) {
        return false;
    }

    preValue = root.val;

    // 访问右子树
    return isValidBST(root.right);
}
```

写法 2：

```
private boolean isBst = true;

// 记录前一个结点的值, 初始值为最小值
private long pre = Long.MIN_VALUE;

public boolean isValidBST(TreeNode root) {
    if (root == null) {
        return false;
    }

    helper(root);

    return isBst;
}

private void helper(TreeNode root) {
    // 递归终止条件与剪枝
    if (root == null || !isBst) {
        return;
    }

    helper(root.left);

    // 不是二叉搜索树记录一下
    if (pre >= root.val) {
        isBst = false;
    }
    // 更新前置结点的值
    pre = root.val;

    helper(root.right);
}
```

**迭代中序遍历法**

利用迭代方式进行中序遍历，也可以进行校验。

```
public boolean isValidBST(TreeNode root) {
    if (root == null) {
        return true;
    }

    // 用栈来实现非递归中序遍历
    Stack<TreeNode> stack = new Stack<>();

    // 注意: 维护一个前一个节点的值
    long preValue = Long.MIN_VALUE;

    while (!stack.empty() || root != null) {
        if (root != null) {
            // 持续将左节点压入
            stack.push(root);
            root = root.left;
        } else {
            root = stack.pop();
            // 处理数据: 判断是否是升序, 不是直接返回
            if (preValue >= root.val) {
                return false;
            }
            // 更新当前值
            preValue = root.val;
            // 将节点切换到右子结点
            root = root.right;
        }
    }
    return true;
}
```

### 二叉搜索树的第 K 小结点(230-中等) 💯

给定一颗二叉搜索树，找到其中第 K 小的结点。

[https://leetcode.cn/problems/kth-smallest-element-in-a-bst/description/](https://leetcode.cn/problems/kth-smallest-element-in-a-bst/description/)

**解题思路**

### 二叉搜索树结点最小距离问题(783-简单) 💯

给定一个二叉搜索树的根节点 root，返回树中任意两节点的差的最小值。

![](/zimg/yuque-1777880836696-39.png)

```
输入: root = [4, 2, 6, 1, 3, null, null]  输出: 1
最小的差值是 1, 它是节点 1 和节点 2 的差值, 也是节点 3 和节点 2 的差值.
```

还有一个题是求任意两结点的绝对值之差的最小值，其实是一样的。连接：

[https://leetcode.cn/problems/minimum-absolute-difference-in-bst/description/](https://leetcode.cn/problems/minimum-absolute-difference-in-bst/description/)

[https://leetcode-cn.com/problems/minimum-distance-between-bst-nodes](https://leetcode-cn.com/problems/minimum-distance-between-bst-nodes)

**递归中序遍历法**

考查**二叉搜索树的中序遍历是递增的性质，最小的距离肯定产生与两个相邻的结点之间**。中序遍历可以通过递归或者迭代进行。

只需要记录前一个结点，然后对比前一个和当前节点的值即可。整体是中序遍历架构，中序遍历可用递归或者迭代实现。

```
public int getMinimumDifference(TreeNode root) {
    dfs(root);
    return minGap;
}

int minGap = Integer.MAX_VALUE;
// 记录上一个结点
TreeNode preNode = null;

// 中序遍历二叉搜索树
private void dfs(TreeNode root) {
    if (root == null) {
        return;
    }
    dfs(root.left);

    // 如果上一个结点不为空则计算差值判断是否更新最小值
    if (preNode != null) {
        minGap = Math.min(minGap, root.val - preNode.val);
    }
    // 更新前一个节点
    preNode = root;

    dfs(root.right);
}
```

**迭代中序遍历法**

修改成迭代方式进行，整体就是迭代中序遍历的方式。

```
public int getMinimumDifference(TreeNode root) {
    if (root == null) {
        return 0;
    }

    TreeNode preNode = null;
    int res = Integer.MAX_VALUE;
    Stack<TreeNode> stack = new Stack<>();

    while (!stack.isEmpty() || root != null) {
        // 持续将当前结点的左子节点压栈
        if (root != null) {
            stack.push(root);
            root = root.left;
        } else {
            root = stack.pop();

            // 注意: 处理数据
            if(preNode != null) {
                res = Math.min(res, root.val - preNode.val);
            }
            preNode = root;

            // 切换到右子树迭代
            root = root.right;
        }
    }
    return res;
}
```

### 二叉搜索树中查找结点(700-简单)

在一颗二叉搜索树中找到给定的目标结点，找不到返回 NULL。

[https://leetcode-cn.com/problems/search-in-a-binary-search-tree/](https://leetcode-cn.com/problems/search-in-a-binary-search-tree/)

**递归法**

利用二叉搜索树的性质进行递归搜索：

- 如果根节点为空 `root == null`(无法找到目标值) 或者根节点的值等于搜索值(找到目标值)，则返回根节点。
- 如果 `val < root.val`，递归进入根节点的左子树查找 `searchBST(root.left, val)`。
- 如果 `val > root.val`，递归进入根节点的右子树查找 `searchBST(root.right, val)`。

```
public TreeNode searchBST(TreeNode root, int target) {
    // 为null或者找到就返回
    if (root == null || target == root.val) {
        return root;
    }

    // 整体前序遍历:根据值决定往左还是右查找
    if(target < root.val) {
        return searchBST(root.left, target);
    } else {
        return searchBST(root.right, target);
    }
}
```

**迭代法**

可以将递归转换为迭代，类似于二分法：

- 如果根节点不空 `root != null` 且根节点不是目的节点 `val != rootVal`：
- 如果 `val < root.val`，进入根节点的左子树查找 `root = root.left`。
- 如果 `val > root.val`，进入根节点的右子树查找 `root = root.right`。

```
public TreeNode searchBST(TreeNode root, int target) {
    // 持续找直到root为空
    while (root != null) {
        // 找到目标值
        if(root.val == target) {
            return root;
            // 分别往左右寻找
        } else if (root.val < target) {
            root = root.right;
        } else {
            root = root.left;
        }
    }
    return null;
}
```

### 二叉搜索树插入结点(701-中等)

给定二叉搜索树(BST)的根节点和要插入树中的值，将值插入二叉搜索树。返回插入后二叉搜索树的根节点。保证原始二叉搜索树中不存在新值。注意可能存在多种有效的插入方式，只要树在插入后仍保持为二叉搜索树即可。可以返回任意有效的结果。

例如给定二叉搜索树和插入的值 5。

![](/zimg/yuque-1777880836709-40.png)

[https://leetcode-cn.com/problems/insert-into-a-binary-search-tree](https://leetcode-cn.com/problems/insert-into-a-binary-search-tree)

**递归方式**

二叉搜索树的优势就是：在平均情况下，能够在 `O(logN)` 的时间内完成搜索和插入元素。二叉搜索树的插入方法非常简单，将插入的节点作为叶子节点的子节点插入。插入到哪个叶节点可以遵循以下原则：

- 若 `val > node.val`，插入到右子树。
- 若 `val < node.val`，插入到左子树。

本题就是往叶子结点插入，也无需考虑树的平衡性。

递归插入算法：

- 若 `val > root.val`，插入到右子树。
- 若 `val < root.val`，插入到左子树。
- 直到 `root == null`，则可以插入元素，并返回 `TreeNode(val)`。
- 返回 root。

```
public TreeNode insertIntoBST(TreeNode root, int val) {
    // 树为空直接返回当前结点
    if (root == null) {
        return new TreeNode(val);
    }

    // 根结点值大于给定值 说明给定值需要插入到左子树
    if (root.val > val) {
        root.left = insertIntoBST(root.left, val);
    } else if (root.val < val) {
        root.right = insertIntoBST(root.right, val);
    }

    // 返回插入后的结点
    return root;
}
```

**迭代法**

上面的递归可以转换成迭代的写法。也就是不断根据结点值往左边或者右边找插入点。

```
public TreeNode insertIntoBST(TreeNode root, int val) {
    if (root == null) {
        return new TreeNode(val);
    }

    TreeNode cur = root;
    // 遍历找到插入位置
    while (true) {
        // 小于往左边插入
        if (val < cur.val) {
            if (cur.left != null) {
                cur = cur.left;
            } else {
                cur.left = new TreeNode(val);
                break;
            }
            // 大于往右边插入
        } else if (val > cur.val) {
            if (cur.right != null) {
                cur = cur.right;
            } else {
                cur.right = new TreeNode(val);
                break;
            }
        }
    }

    return root;
}
```

### 二叉搜索树的范围和(938-简单)

给定二叉搜索树的根结点 root，返回 L 和 R(含)之间的所有结点的值的和。二叉搜索树保证具有唯一的值。

```
输入: root = [10,5,15,3,7,null,18], L = 7, R = 15  输出: 32
```

[https://leetcode-cn.com/problems/range-sum-of-bst/](https://leetcode-cn.com/problems/range-sum-of-bst/)

**递归法 DFS**

利用二叉搜索树的特性，不用每个节点都去遍历。在代码中用递归和迭代的方法分别实现了深度优先搜索。

对树进行深度优先搜索，对于当前节点 node：

- 如果 `node.val` 小于等于 L，那么只需要继续搜索它的右子树。
- 如果 `node.val` 大于等于 R，那么只需要继续搜索它的左子树。
- 如果 `node.val` 在区间 `(L, R)` 中，则需要搜索它的所有子树。

这个方法可以。

```
int sum;
public int rangeSumBST(TreeNode root, int low, int high) {
    sum = 0;
    dfs(root, low, high);
    return sum;
}

public void dfs(TreeNode node, int left, int right) {

    if (node == null) {
        return;
    }

    // 这个值OK, 加上
    if (left <= node.val && node.val <= right) {
        sum = sum + node.val;
    }

    // 递归左子树(剪枝)
    if (left < node.val) {
        dfs(node.left, left, right);
    }

    // 递归右子树(剪枝)
    if (node.val < right) {
        dfs(node.right, left, right);
    }
}
```

写法 2：

```
public int rangeSumBST(TreeNode root, int low, int high) {
    if (root == null) {
        return 0;
    }
    // 结点值较大 最终的和是左子树的范围和
    if (root.val > high) {
        return rangeSumBST(root.left, low, high);
    }
    // 结点值较小 最终的和是右子树的范围和
    if (root.val < low) {
        return rangeSumBST(root.right, low, high);
    }
    // 当前结点值在给定的范围内, 递归两边的值+自己
    return root.val + rangeSumBST(root.left, low, high) + rangeSumBST(root.right, low, high);
}
```

**迭代 DFS**

可以使用中序遍历的迭代方式实现上述 DFS。无非就是进行了一次遍历。不过时间空间都不太好。

```
public int rangeSumBST(TreeNode root, int low, int high) {
    if (root == null) {
        return 0;
    }
    int sum = 0;

    // 用栈来实现非递归中序遍历
    Stack<TreeNode> stack = new Stack<>();

    while (!stack.empty() || root != null) {
        if (root != null) {
            // 持续将左节点压入
            stack.push(root);
            root = root.left;
        } else {
            root = stack.pop();
            // 处理数据
            if (root.val >= low && root.val <= high) {
                sum += root.val;
            }
            // 将节点切换到右子结点
            root = root.right;
        }
    }
    return sum;
}
```

### 两棵二叉搜索树值的合并(1305-中等)

给你 root1 和 root2 这两棵二叉搜索树。返回一个列表，其中包含两棵树中的所有整数并按升序排序。

![](/zimg/yuque-1777880836742-41.png)

```
输入: root1 = [2,1,4], root2 = [1,0,3] 输出: [0,1,1,2,3,4]
```

[https://leetcode-cn.com/problems/all-elements-in-two-binary-search-trees/](https://leetcode-cn.com/problems/all-elements-in-two-binary-search-trees/)

**中序遍历+有序链表合并法**

对二叉搜索树进行中序遍历，就可以直接得到树中所有元素升序排序后的结果。因此可以对两棵树分别进行中序遍历，得到数组 v1 和 v2，它们分别存放了两棵树中的所有元素，且均已有序。在这之后得到的两个结果链表就是有序的，最后对两个有序链表进行排序，就可以得到最终的结果。注意这里为了方便进行链表合并，使用的是 LinkedList，这样可以方便的对元素进行大小比较和弹出。

关键点是合并两个有序的链表，写法参考：合并两个有序链表(21)。

```
public List<Integer> getAllElements(TreeNode root1, TreeNode root2) {
    LinkedList<Integer> list1 = new LinkedList<>();
    LinkedList<Integer> list2 = new LinkedList<>();

    // 中序遍历方式将值放入列表
    dfs(root1, list1);
    dfs(root2, list2);

    // 归并排序合并两个有序链表
    return mergeLists(list1, list2);
}

// 中序遍历
private void dfs(TreeNode root, LinkedList<Integer> nums) {
    if (root == null) return;
    dfs(root.left, nums);
    nums.add(root.val);
    dfs(root.right, nums);
}

private List<Integer> mergeLists(LinkedList<Integer> list1, LinkedList<Integer> list2) {
    if (list1.isEmpty() && list2.isEmpty()) {
        return new ArrayList<>();
    }

    // 一个链表为空 直接返回另一个
    if (list1.isEmpty() || list2.isEmpty()) {
        return list1.isEmpty() ? list2 : list1;
    }

    List<Integer> resList = new ArrayList<>();

    // 两个链表都不为空则持续合并
    while (!list1.isEmpty() && !list2.isEmpty()) {
        if (list1.peek() < list2.peek()) {
            resList.add(list1.pop());
        } else {
            resList.add(list2.pop());
        }
    }

    // 剩下的还有元素的列表直接全部加到结果集中
    while (!list1.isEmpty()) {
        resList.add(list1.pop());
    }
    while (!list2.isEmpty()) {
        resList.add(list2.pop());
    }

    return resList;
}
```

时间复杂度：`O(n)`。空间复杂度：`O(n)`。

**优先级队列**

在树遍历的时候用一个优先队列(默认小根堆)来添加元素；然后将优先队列的元素逐个取出到 list 中即可。

```
// 使用优先队列
private PriorityQueue<Integer> priorityQueue;

public List<Integer> getAllElements(TreeNode root1, TreeNode root2) {
    priorityQueue = new PriorityQueue<>();
    // 两个树的值分别放入优先队列
    dfs(root1);
    dfs(root2);
    List<Integer> ansList = new ArrayList<>();
    // 取值放入结果列表
    while (!priorityQueue.isEmpty()) {
        ansList.add(priorityQueue.poll());
    }
    return ansList;
}

private void dfs(TreeNode root) {
    if (root == null) return;
    priorityQueue.offer(root.val);
    dfs(root.left);
    dfs(root.right);
}
```

### 二叉搜索树中的众数(501-中等)

给定一个有相同值的二叉搜索树(BST)，找出 BST 中的所有众数(出现频率最高的元素)。例如给点如下二叉搜索树，返回 `[2]`。

```
   1
    \
     2
    /
   2
```

[https://leetcode-cn.com/problems/find-mode-in-binary-search-tree/](https://leetcode-cn.com/problems/find-mode-in-binary-search-tree/)

**递归法(中序遍历)**

二叉搜索树的左右子结点和父结点之间有大小关系的限制，且二叉树的中序遍历是升序的。由此，问题可以转变成在**一组升序排列的数中查找众数**。

既然数字是升序的，就可以遍历一次完成统计。从小到大遍历数据找到次数最大的即可。**注意本题可能存在多个众数**。

```
// 最大的值个数
int maxCounter = 0;

// 当前值计数器
int tempCounter = 0;

// 存放众数的列表
List<Integer> resList = new LinkedList<>();

// 当前值
int curValue;

public int[] findMode(TreeNode root) {
    if (root == null) {
        return new int[0];
    }
    // 初始化当前值
    curValue = Integer.MIN_VALUE;
    tempCounter = 0;

    // 递归中序遍历
    dfs(root);

    int[] result = new int[resList.size()];

    for (int i = 0; i < resList.size(); i++) {
        result[i] = resList.get(i);
    }

    return result;
}

// 中序遍历
public void dfs(TreeNode root) {
    if (root == null) {
        return;
    }

    dfs(root.left);

    // 当前值与结点值相同则当前值计数器+1, 否则重置为1
    if (curValue == root.val) {
        tempCounter++;
    } else {
        tempCounter = 1;
    }

    // 更新当前值
    curValue = root.val;

    // 如果当前的计数器值等于最大值将这个数记录到列表中
    if (tempCounter == maxCounter) {
        resList.add(curValue);
    }

    // 说明产生了新的众数
    if (tempCounter > maxCounter) {
        // 更新最大值个数
        maxCounter = tempCounter;
        // 清空之前缓存的众数记录
        resList.clear();
        resList.add(curValue);
    }

    dfs(root.right);
}
```

### 搜索二叉树的最近公共祖先(235-中等)

给定一个二叉搜索树，找到该树中两个指定节点的最近公共祖先。例如如下二叉搜索树：

![](/zimg/yuque-1777880836766-42.png)

```
输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 8
输出: 6  解释: 节点2和节点8的最近公共祖先是6
```

[https://leetcode-cn.com/problems/lowest-common-ancestor-of-a-binary-search-tree](https://leetcode-cn.com/problems/lowest-common-ancestor-of-a-binary-search-tree)

对于搜索二叉树来说，最近公共结点是小于等于两个结点的较大值，大于等于两个结点的较小值。

**迭代法**

利用二叉搜索树的性质进行迭代。

- 从根节点遍历。
- 如果当前节点的值大于 p 和 q 的值，说明 p 和 q 应该在当前节点的左子树，因此将当前节点移动到它的左子节点。
- 如果当前节点的值小于 p 和 q 的值，说明 p 和 q 应该在当前节点的右子树，因此将当前节点移动到它的右子节点。
- 如果当前节点的值介于 p 和 q 之间，那么说明当前节点就是「分岔点」。此时，p 和 q 要么在当前节点的不同的子树中，要么其中一个就是当前节点。

```
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    while (true) {
        // 当前节点的值大于 p 和 q 的值, 说明 p 和 q 应该在当前节点的左子树
        if (root.val > p.val && root.val > q.val) {
            root = root.left;
            // 当前节点的值小于 p 和 q 的值, 说明 p 和 q 应该在当前节点的右子树
        } else if (root.val < p.val && root.val < q.val) {
            root = root.right;
        } else {
            // 当前结点值介于p和q之间 说明当前结点就是目标结点
            return root;
        }
    }
}
```

**层序遍历法**

利用上述思想，层序遍历找到第一个复合条件的结点即可。

```
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode node1, TreeNode node2) {
    if(root == null) {
        return null;
    }
    // 得到两个目标结点的较大值和较小值
    int min = Math.min(node1.val, node2.val);
    int max = Math.max(node1.val, node2.val);

    // 使用普通层序遍历的方式进行
    Queue<TreeNode> queue = new LinkedList();
    queue.add(root);

    while(!queue.isEmpty()) {
        TreeNode node = queue.poll();
        // 判断第一个符合条件的结点就直接返回
        if(node.val >= min && node.val <= max) {
            return node;
        }

        if(node.left != null) queue.offer(node.left);
        if(node.right != null) queue.offer(node.right);
    }

    // 没找到返回null
    return null;
}
```

**递归法**

依然利用搜索二叉树的特性，如果 p 和 q 在 root 的两边，那么 root 就是 p 和 q 的最低的公共祖先，这种情况包含三种具体的情况：

- p 和 q 在 root 的两边。
- p 就是 root，q 在 root 的哪边无所谓。
- q 就是 root，p 在 root 的哪边无所谓。

如果不是上面的情况，p 和 q 要么全在 root 的左边，要么全在 root 的右边，这时只需递归的去 root 的左子树或右子树中找 p 和 q 的最低公共祖先。

由于是找最近的公共祖先，所以整体是后序遍历的模板。

```
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    // Base case
    if (p == null || q == null || root == null) {
        return null;
    }

    // 最近公共祖先在左子树, 往左子树递归
    if (root.val > p.val && root.val > q.val) {
        return lowestCommonAncestor(root.left, p, q);
    }
    // 最近公共祖先在右子树, 往右子树递归
    if (root.val < p.val && root.val < q.val) {
        return lowestCommonAncestor(root.right, p, q);
    }
    // 找到结果
    return root;
}
```

还有一个找两个普通二叉树结点的最近公共祖先问题，见后面。

### 修剪二叉搜索树(669-中等)

给定一个二叉搜索树，同时给定最小边界 L 和最大边界 R。通过修剪二叉搜索树，使得所有节点的值在 `[L, R]` 中 (`R >= L`)。可能需要改变树的根节点，所以结果应当返回修剪好的二叉搜索树的新的根节点。

```
输入:
    1
   / \
  0   2
  L = 1  R = 2
输出:
    1
      \
       2
```

[https://leetcode-cn.com/problems/trim-a-binary-search-tree](https://leetcode-cn.com/problems/trim-a-binary-search-tree)

**自底向上递归修剪法**

如果根结点的值小于给定的左边界 L，那么当前结点及其左子树就会被修剪掉，修剪后的树应该是其右子树，但是右子树不一定是符合范围的树，所以也要对其右子树进行递归修剪，然后返回修剪后的右子树。同理，根结点的值大于给定的右边界 R，修剪后的树应该是其左子树且要对左子树修剪。涉及到改变树的结构，就需要更新链接，如果当前结点值在范围内，那么修建其左右子树，并且更新左右链接。最后将当前修剪好的子树返回。这个过程是自底向上的修剪所以整体是后序遍历架构。

时间复杂度为 `O(n)`，空间复杂度与树高成正比。

```
public TreeNode trimBST(TreeNode root, int low, int high) {
    // base case
    if (root == null) {
        return null;
    }

    // 当前结点小于最小值 说明当前结点的左子树直接裁剪了 直接返回递归右子树的结果
    if (root.val < low) {
        return trimBST(root.right, low, high);
    }

    // 当前结点大于最大值 说明则当前结点的右子树直接裁剪了 直接返回递归左子树的结果
    if (root.val > high) {
        return trimBST(root.left, low, high);
    }

    // 说明当前root结点是满足需求的, 需要重新构造其左右子树
    // 这里前面先递归 最后再处理数据 整体是自底向上的后序遍历模板
    root.left = trimBST(root.left, low, high);
    root.right = trimBST(root.right, low, high);

    return root;
}
```

### 把二叉搜索树转换为累加树(538-中等)

给定一个二叉搜索树，把它转换成为累加树，使得每个节点的值是原来的节点值加上所有大于它的节点值之和。例如输入原始二叉搜索树：

```
	  5
    /   \
   2     13
```

输出转换为累加树。

```
      18
    /   \
  20     13
```

[https://leetcode-cn.com/problems/convert-bst-to-greater-tree](https://leetcode-cn.com/problems/convert-bst-to-greater-tree)

**反序中序遍历法**

二叉搜索树的**中序遍历就是按照升序排列的**。那如果先遍历右子树则就是降序排列的！这个题的思路是先处理最大的数据，最大的就是其本身，所以不变，后续的结点都是当前的结点 + 后面的比它大的所有结点之和，所以维护一个 sum 变量，用于记录已经遍历过的结点的和。

注意：**整个代码是属于中序遍历的框架，但是是先递归右子树，再递归左子树，这样处理顺序就是从大到小了**。

```
// 记录目前为止的从大到小遍历结点的累加值
int sum = 0;

// 整体中序递归
public TreeNode convertBST(TreeNode root) {
    if (root == null) {
        return null;
    }

    // 先递归右子树!!!
    convertBST(root.right);

    // 更新sum
    sum = sum + root.val;
    // 更新当前结点值
    root.val = sum;

    // 再递归左子树!!!
    convertBST(root.left);

    return root;
}
```

### 二叉搜索树的后序遍历序列(LCR-152-中等) 🔑

输入一个整数数组，**判断该数组是不是某二叉搜索树的后序遍历的结果**。假设输入的数组的任意两个数字都互不相同。例如下图是后序遍历序列 `1, 3, 2` 所对应的二叉搜索树。

![](/zimg/yuque-1777880836775-43.png)

[https://leetcode.cn/problems/er-cha-sou-suo-shu-de-hou-xu-bian-li-xu-lie-lcof/description/](https://leetcode.cn/problems/er-cha-sou-suo-shu-de-hou-xu-bian-li-xu-lie-lcof/description/)

**递归法**

根据二叉搜索树的定义，可以通过递归，判断所有子树的正确性(即其后序遍历是否满足二叉搜索树的定义)，若所有子树都正确，则此序列为二叉搜索树的后序遍历。

- 划分左右子树：遍历后序遍历的 `[i, j]` 区间元素，寻找第一个大于根节点的节点，索引记为 `m`。此时，可划分出左子树区间 `[i, m − 1]`，右子树区间 `[m, j − 1]`，根节点索引 `j`。
- 递归判断左右区间是否合法：左子树区间 `[i, m − 1]` 内的所有节点都应小于 `postorder[j]` 。而前面划分左右子树时已经保证左子树区间的正确性，因此只需要判断右子树区间即可。`[m, j − 1]` 内的所有节点都应大于 `postorder[j]`。

代码如下：

```
private boolean isValid = true;

public boolean verifyTreeOrder(int[] postOrder) {
    if (postOrder == null || postOrder.length == 0) {
        return true;
    }

    // 递归验证一个范围内是不是二叉搜索树
    verify(postOrder, 0, postOrder.length - 1);

    return isValid;
}

private void verify(int[] nums, int left, int right) {
    // 如果只有一或两个元素直接返回, 同时也剪枝一下(如果遇到了不合法则肯定不合法)
    if (right - left <= 1 || !isValid) {
        return;
    }

    // 后序遍历序列的根结点值在元素的最后一个
    int rootVal = nums[right];

    // 遍历数组找到第一个大于等于根结点的元素作为裁切点, 后续进行拆分左右子树的区间
    int cutIndex = left;
    while (cutIndex < right && nums[cutIndex] <= rootVal) {
        cutIndex++;
    }

    // 切分点左边的数属于左子树, 其值应该都小于根结点值, 但前面其实找切分点的过程已经保证了这个, 所以不需要再校验左边的区间了
    // 切分点右边的部分属于右子树, 其值应该都大于根结点值, 这里进行比较, 一旦不合法则整棵树不合法
    for (int i = cutIndex; i < right; i++) {
        if (nums[i] < rootVal) {
            isValid = false;
        }
    }
    // 递归校验左右区间的数据
    verify(nums, left, cutIndex - 1);
    verify(nums, cutIndex, right - 1);
}
```

### 二叉树的所有路径(257-简单)

给定一个二叉树，返回所有从根节点到叶子节点的路径。示例输入：

```
   1
 /   \
2     3
 \
  5
```

输出：`["1->2->5", "1->3"]`。

[https://leetcode-cn.com/problems/binary-tree-paths](https://leetcode-cn.com/problems/binary-tree-paths)

**递归先序遍历法**

整体前序递归遍历的模板，当遇到子结点的时候进行数据处理。这里不要用 StringBuilder，它会记录状态导致不正确。注意这里需要递归的时候加一个箭头。

```
private List<String> resList = new ArrayList<>();
private String path = "";

public List<String> binaryTreePaths(TreeNode root) {
    if (root == null) {
        return Collections.emptyList();
    }

    dfs(root, "");

    return resList;
}

// 整体前序遍历
private void dfs(TreeNode root, String path) {
    if (root == null) {
        return;
    }

    // 拼接path
    path = path + root.val;

    // 遇到叶子结点则记录一次结果
    if (root.left == null && root.right == null) {
        resList.add(path);
    }

    // 如果后面还有结点则加上一个箭头
    path = path + "->";

    // 递归左右子树
    dfs(root.left, path);
    dfs(root.right, path);
}
```

### 根结点到叶子节点路径总和(112-简单) 💯

给定一个二叉树和一个目标和，判断该树中是否存在根节点到叶子节点的路径，这条路径上所有节点值相加等于目标和。示例：给定如下二叉树，以及目标和 `sum = 22`。

```
          5
         / \
        4   8
       /   / \
      11  13  4
     /  \      \
    7    2      1
```

返回 true，因为存在目标和为 22 的根节点到叶子节点的路径 `5 -> 4 -> 11 -> 2`。

[https://leetcode-cn.com/problems/path-sum](https://leetcode-cn.com/problems/path-sum)

**递归先序遍历法**

**直接先序递归，每经过一个节点就将目标值减去节点值，并与之后的节点值对比，如果刚好相等且为叶子节点，则认为找到**。

```
private boolean hasPath = false;

public boolean hasPathSum(TreeNode root, int targetSum) {
    if (root == null) {
        return false;
    }

    dfs(root, targetSum);

    return hasPath;
}

private void dfs(TreeNode root, int targetSum) {
    // Base case与剪枝
    if (root == null || hasPath) {
        return;
    }

    // 剩下的targetSum恰好是叶子结点的值
    if (targetSum == root.val && root.right == null && root.left == null) {
        hasPath = true;
    }

    // 处理完后target值减少
    targetSum = targetSum - root.val;

    dfs(root.left, targetSum);
    dfs(root.right, targetSum);
}
```

**迭代法**

利用栈迭代前序遍历。这里使用了两个栈，一个记录结点，另一个记录对应的结点的历史和，sum 栈的大小和结点栈的大小是一致的。

```
public boolean hasPathSum(TreeNode root, int sum) {
    if (root == null) {
        return false;
    }

    // 记录结点的栈
    LinkedList<TreeNode> nodeStack = new LinkedList<>();
    // 记录当前和的栈, 注意这个栈的元素个数与结点栈的个数一样, 只是同步记录一下对应的结点的和
    LinkedList<Integer> sumStack = new LinkedList<>();

    nodeStack.add(root);
    sumStack.add(root.val);

    while (!nodeStack.isEmpty()) {
        TreeNode node = nodeStack.pollLast();
        int curSum = sumStack.pollLast();
        // 找到目标结点
        if ((node.right == null) && (node.left == null) && (curSum == sum)) {
            return true;
        }

        // 同时记录结点和计算到当前结点的和
        if (node.right != null) {
            nodeStack.add(node.right);
            sumStack.add(curSum + node.right.val);
        }

        if (node.left != null) {
            nodeStack.add(node.left);
            sumStack.add(curSum + node.left.val);
        }
    }
    return false;
}
```

给定一个二叉树，它的每个结点都存放一个 `0-9` 的数字，每条从根到叶子节点的路径都代表一个数字。例如从根到叶子节点路径 `1->2->3` 代表数字 `123`。计算从根到叶子节点生成的所有数字之和。

  2   3
输出: 25
解释: 从根到叶子节点路径 1->2 代表数字 12. 从根到叶子节点路径 1->3 代表数字 13.
因此, 数字总和 = 12 + 13 = 25.
```

[https://leetcode-cn.com/problems/sum-root-to-leaf-numbers](https://leetcode-cn.com/problems/sum-root-to-leaf-numbers)

**递归先序遍历法**

直接先序遍历即可。注意这里使用的是**字符串**，这样递归之后不需要进行回溯。

```
private int sum = 0;

public int sumNumbers(TreeNode root) {

    if (root == null) {
        return;

    // 构造路径数值
    pathNum = pathNum + root.val;

    // 到了子结点就进行处理, 即结果累加
    if (root.left == null && root.right == null) {

        return;

    dfs(root.left, pathNum);
    dfs(root.right, pathNum);

大佬的简短代码。直接传数据进去即可，其实就是左子树 + 右子树的和。

```
public int sumNumbers(TreeNode root) {
    return dfs(root, 0);
}

public int dfs(TreeNode root, int num){
    if (root == null) {
        return 0;
    }

    int temp = num * 10 + root.val;

写法 2：使用 LinkedList 缓存各个节点的数据，需要注意这里递归完成后需要去掉之前加入的结点，进行回溯。

private int sum = 0;
private LinkedList<Integer> path = new LinkedList<>();

public int sumNumbers(TreeNode root) {

    return sum;
}

private void dfs(TreeNode root) {
    if (root == null) {
        return;

    path.add(root.val);

        for (Integer integer : path) {
            num.append(integer);
        }
        sum = sum + Integer.parseInt(num.toString());

    dfs(root.right);

    path.pollLast();
}
```

### 二叉树的坡度(563-简单)

给定一个二叉树，计算整个树的坡度。一个树的节点的坡度定义为，**该节点左子树的结点之和和右子树结点之和的差的绝对值**。空结点的的坡度是 0。整个树的坡度就是其所有节点的坡度之和。

```
输入:
         1
       /   \
      2     3
输出: 1
解释:
结点的坡度 2 : 0
结点的坡度 3 : 0
结点的坡度 1 : |2-3| = 1
树的坡度 : 0 + 0 + 1 = 1
```

[https://leetcode-cn.com/problems/binary-tree-tilt](https://leetcode-cn.com/problems/binary-tree-tilt)

**自底向上递归法**

根节点的坡度依赖于叶子结点的坡度计算，**整个过程自底向上计算，所以整个是后序遍历递归**。这里使用辅助方法，该方法返回值是以当前节点为树的整棵树的所有节点值之和。然后递归的过程中计算每个节点的坡度并更新 sum 即可。

```
private int sum = 0;

public int findTilt(TreeNode root) {
    getTreeSum(root);
    return sum;
}

// 辅助递归方法返回以自己为根节点的整棵树的所有节点值之和
private int getTreeSum(TreeNode root) {
    if (root == null) {
        return 0;
    }
    // 从左右子树获取其所有节点值之和的信息
    int left = getTreeSum(root.left);
    int right = getTreeSum(root.right);

    // 当时计算每个节点的坡度值并更新总的sum
    int poDu = Math.abs(left - right);
    sum = sum + poDu;

    // 返回以自己为根节点的整棵树的所有节点值之和
    return left + right + root.val;
}
```

### 二叉树的直径(543-简单) 💯

一棵二叉树的直径长度是任意两个结点路径长度中的最大值。这条路径可能穿过也可能不穿过根结点。求出一颗二叉树的直径。示例：

![](/zimg/yuque-1777880836787-44.png)

返回 3，它的长度是路径 `[4, 2, 1, 3]` 或者 `[5, 2, 1, 3]`。注意：两结点之间的路径长度是以它们之间边的数目表示。

[https://leetcode-cn.com/problems/diameter-of-binary-tree](https://leetcode-cn.com/problems/diameter-of-binary-tree)

**自底向上递归法**

最长路径是一个结点的左右子树的最大路径之和。这里使用一个辅助的自底向上递归方法，该方法获取以 root 为根节点的树的最大深度(注意这里是深度而不是直径)，通过分别获取左右子树的深度，即可算出 root 结点的直径，如果出现最大直径则更新一下即可；最后辅助方法返回的是左右子树深度的较大值 + 1。

```
private int maxDiameter = 0;

public int diameterOfBinaryTree(TreeNode root) {
    if (root == null) {
        return 0;
    }
    getDepth(root);
    return maxDiameter;
}

// 自底向上递归
// 辅助方法获取以root为根节点的树的最大深度
public int getDepth(TreeNode root) {
    if (root == null) {
        return 0;
    }

    // 分别获取左右子树的深度
    int leftDepth = getDepth(root.left);
    int rightDepth = getDepth(root.right);

    // 如果当前结点的左右子树的深度之和大于当前结点, 则记录一下最大直径值
    maxDiameter = Math.max(maxDiameter, leftDepth + rightDepth);

    // 当前结点的深度就是左右结点深度的较大值+1
    return Math.max(leftDepth, rightDepth) + 1;
}
```

### 节点与其祖先之间的最大差值(1026-中等)

给定二叉树的根节点 root，找出存在于不同节点 A 和 B 之间的最大值 V，其中 `V = |A.val - B.val|`，且 A 是 B 的祖先。示例：

![](/zimg/yuque-1777880836797-45.png)

```
输入: [8,3,10,1,6,null,14,null,null,4,7,13]  输出: 7
解释: 这里有大量的节点与其祖先的差值, 其中一些如下:

```

[https://leetcode-cn.com/problems/maximum-difference-between-node-and-ancestor](https://leetcode-cn.com/problems/maximum-difference-between-node-and-ancestor)

**自底向上递归法**

使用一个**辅助方法**，采用自底向上递归的方式求解。辅助方法返回的是以 root 为根节点的 **[最小值，最大值]** 的组合，处理数据时，判断是否可以更新最大差值即可。

```
private int maxDiff = 0;

public int maxAncestorDiff(TreeNode root) {
    getTreeInfo(root);
    return maxDiff;
}

// 自底向上递归
// 辅助方法返回的是以root为根节点的[最小值, 最大值]的组合
public int[] getTreeInfo(TreeNode root) {
    if (root == null) {
        // 需要保证空节点不影响后面最小最大值的判断
        return new int[]{Integer.MAX_VALUE, Integer.MIN_VALUE};
    }

    // 获取左右子树的最小最大值
    int[] leftInfo = getTreeInfo(root.left);
    int[] rightInfo = getTreeInfo(root.right);

    // 计算以当前root为根节点的[最小值, 最大值]
    int min = Math.min(root.val, Math.min(leftInfo[0], rightInfo[0]));
    int max = Math.max(root.val, Math.max(leftInfo[1], rightInfo[1]));

    // 看看是否需要更新结果
    maxDiff = Math.max(Math.max(Math.abs(max - root.val), Math.abs(min - root.val)), maxDiff);

    return new int[] {min, max};
}
```

**自顶向下递归法**

从根节点开始往下递归，**递归时往下传入的是目前遇到的最大值和最小值**，处理数据时不断更新差值即可。

```
private int maxDiff = 0;

public int maxAncestorDiff(TreeNode root) {
    dfs(root, root.val, root.val);
    return maxDiff;
}

public void dfs(TreeNode root, int maxValue, int minValue) {
    if (root == null) {
        return;
    }

    // 更新结果
    maxDiff = Math.max(maxDiff, Math.max(Math.abs(maxValue - root.val), Math.abs(minValue - root.val)));

    // 更新当前最大值
    maxValue = Math.max(maxValue, root.val);
    // 更新当前最小值
    minValue = Math.min(minValue, root.val);

    // 递归
    dfs(root.left, maxValue, minValue);
    dfs(root.right, maxValue, minValue);
}
```

### 最长同值路径(687-中等)

给定一个二叉树，找到最长的路径，这个路径中的每个节点具有相同值。这条路径可以经过也可以不经过根节点。注意：两个节点之间的路径长度由它们之间的边数表示。输入：

![](/zimg/yuque-1777880836811-46.png)

输出：2。

[https://leetcode-cn.com/problems/longest-univalue-path](https://leetcode-cn.com/problems/longest-univalue-path)

**自底向上递归法**

最长同值路径分为 2 种情况：

1. 以 root 为起点，经过左子树或右子树，如图中 `(2)`。
2. 不是以 root 为起点，root 是中间点，如图中 `(1)`。

![](/zimg/yuque-1777880836824-47.png)

根据题意可以先找当前节点左子树的路径 left，再找当前节点右子树的路径 right。

- 若 `root.val == root.left.val`，则当前节点的路径长为 `leftPath = left + 1`。
- 若 `root.val == root.right.val`，则当前节点的路径长为 `rightPath = right + 1`。

然后得到当前节点的左右路径后相加即为已当前节点为根节点的最长路径。

使用一个自底向上的辅助递归方法，该方法返回的是经过了 root 的最长路径的长度。因此会先分别获取经过左右子树的最长路径值，再将 root 的值与左右子树对比，如果相同，则经过 root 的路径长度就会增加，否则就置 0 了；同时处理数据的过程中滚动记录出现过的最大的路径值即可。

```
private int maxLen = 0;  // 全局最长路径

public int longestUnivaluePath(TreeNode root) {
    if (root == null) return 0;
    dfs(root);
    return maxLen;
}

// 辅助方法: 返回经过当前根节点的最大同值路径长度
private int dfs(TreeNode node) {
    if (node == null) {
        return 0;
    }

    // 递归计算左右子树的同值路径长度
    int left = dfs(node.left);
    int right = dfs(node.right);

    // 如果子节点和当前节点值相同, 则路径可以延伸. 这里分别往左右延伸一下
    int leftLen = (node.left != null && node.left.val == node.val) ? left + 1 : 0;
    int rightLen = (node.right != null && node.right.val == node.val) ? right + 1 : 0;

    // 更新全局最大值(当前节点的总路径长度)
    maxLen = Math.max(maxLen, leftLen + rightLen);

    // 注意: 返回给父节点的是单侧最长路径(因为路径不能分叉, 所以只能取左或者右的较大值)
    return Math.max(leftLen, rightLen);
}
```

### 二叉树中的最大路径和(124-困难) 💯

二叉树中的路径被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中至多出现一次。该路径至少包含一个节点，且不一定经过根节点。路径和是路径中各节点值的总和。找出一颗二叉树的最大路径和。

![](/zimg/yuque-1777880836833-48.png)

```
输入: root = [-10,9,20,null,null,15,7]   输出: 42
解释: 最优路径是 15 -> 20 -> 7 , 路径和为 15 + 20 + 7 = 42
```

[https://leetcode.cn/problems/binary-tree-maximum-path-sum/description/](https://leetcode.cn/problems/binary-tree-maximum-path-sum/description/)

**自底向上递归法**

使用一个辅助方法：**自底向上的后序递归，返回下面的结点到当前结点的最大链路和的值**。一定要搞清楚方法的返回值的含义，这里返回值不是以 root 为根节点的路径最大值，而是到 root 的某一侧的链的最大和。有了左右子树的最大链路和，就能两者加起来，再加上当前的值，计算出一个可能得最大值，滚动记录出现过的最大值即可。如果一侧的链路和小于 0 则返回 0，也就是会带来负收益的直接舍弃。

比如图中 `root = -10`，左子树到 root 最大链路和为 9，右子树到 root 的最大链路和为 `15+20=35`。

dfs 方法的**返回值是以当前结点为终点的最大侧边和**，自底向上递归过程中**滚动更新结果**！

```
private int maxSum = Integer.MIN_VALUE;

public int maxPathSum(TreeNode root) {
    dfs(root);
    return maxSum;
}

// ⭐辅助方法: 自底向上的后序递归, 返回下面的结点到当前结点的某一侧链的和的最大值
private int dfs(TreeNode node) {
    if (node == null) {
        return 0;
    }

    // ⭐分别获取左右子树的最大链路和
    int leftSum = dfs(node.left);
    int rightSum = dfs(node.right);

    // ⭐左右两个链再加当前结点值就是最大路径值, 滚动更新结果即可
    maxSum = Math.max(maxSum, leftSum + rightSum + node.val);

    // ⭐返回给上层的时当前子树最大链和, 也就是取左边的链还是右边的链的最大值
    // 不能分叉 所以只能选大的一边
    // 如果最大值是负数则直接返回0也就是后面不要这棵子树的结果
    return Math.max(Math.max(leftSum, rightSum) + node.val, 0);
}
```

### 从叶结点开始的最小字符串(988-中等)

给定一颗根结点为 root 的二叉树，树中的每一个结点都有一个从 0 到 25 的值，分别代表字母 'a' 到 'z'：值 0 代表 'a'，值 1 代表 'b'，依此类推。找出按字典序最小的字符串，该字符串从这棵树的一个叶结点开始，到根结点结束。(字符串中任何较短的前缀在字典序上都是较小的：例如，在字典序上 "ab" 比 "aba" 要小)

示例 1：

![](/zimg/yuque-1777880836844-49.png)

```
输入: [25,1,3,1,3,0,2]  输出: "adz"
```

[https://leetcode-cn.com/problems/smallest-string-starting-from-leaf](https://leetcode-cn.com/problems/smallest-string-starting-from-leaf)

**递归法**

直接用前序遍历的方式遍历整棵树，同时记录遍历到的路径，如果遍历到当前节点是叶子节点，则进行字典序的对比并滚动更新最小的字符串。

```
private String resStr = "";

public String smallestFromLeaf(TreeNode root) {
    if (root == null) {
        return "";
    }
    dfs(root, "");
    return resStr;
}

private void dfs(TreeNode root, String path) {
    if (root == null) {
        return;
    }

    // 构造字符串, 注意这里叶子结点的值是往字符串头部添加的
    path = (char) ('a' + root.val) + path;

    // 当前的结点为叶子结点时处理数据, 更新最小的字符串结果
    if (root.left == null && root.right == null) {
        // 若所生成字符串比当前最小字符串小, 则更新保留
        if ("".equals(resStr) || path.compareTo(resStr) < 0) {
            resStr = path;
        }
        return;
    }

    // 递归
    dfs(root.left, path);
    dfs(root.right, path);
}
```

写法 2：这里递归方法传入的是一个 StringBuilder，因此需要在递归完成后进行回溯。

```
// 这里把res初始化为一个大的字符串值
private String res = String.valueOf((char) 255);

public String smallestFromLeaf(TreeNode root) {
    if (root == null) {
        return "";
    }
    dfs(root, new StringBuilder());
    return res;
}

public void dfs(TreeNode node, StringBuilder builder) {
    if (node == null) {
        return;
    }
    builder.append((char) ('a' + node.val));

    // 遇到叶子结点则处理数据
    if (node.left == null && node.right == null) {
        // 注意这里记录的是从根节点到叶子结点的顺序, 需要逆序后进行对比
        builder.reverse();
        String path = builder.toString();
        builder.reverse();

        if (path.compareTo(res) < 0) {
            res = path;
        }
    }

    dfs(node.left, builder);
    dfs(node.right, builder);

    // 回溯当前结点值
    builder.deleteCharAt(builder.length() - 1);
}
```

### 二叉树中所有距离为 K 的结点(863-中等)

给定一个二叉树，一个目标结点 target，和一个整数值 K。返回到目标结点 target 距离为 K 的所有结点的值的列表。答案可以以任何顺序返回。注意结点中所有值都不相同。

![](/zimg/yuque-1777880836854-50.png)

示例二叉树的输出：`[7, 4, 1]`。解释：所求结点为与目标结点(值为 5)距离为 2 的结点，值分别为 7，4，以及 1。

[https://leetcode-cn.com/problems/all-nodes-distance-k-in-binary-tree](https://leetcode-cn.com/problems/all-nodes-distance-k-in-binary-tree)

**深度与广度优先遍历结合法**

与目标结点距离为 K，转换一下也就是以目标结点为 root 进行广度优先遍历，遍历到第 K 层的结点就是结果了。因此这里**使用深度优先遍历找出每个结点的父结点，并存储到 map 中方便查询**。然后以目标结点为根节点，进行广度优先遍历(整体类似与二叉树的层序遍历)，只不过这里每次需要处理三个结点：当前结点的左右子结点和当前结点的父节点。遍历到第 K 层是记录所有的结点值即可。注意：这里需要维护一个已经访问过的节点的 Set，如果访问过就不再加入到广度优先遍历的队列中了。

```
// 记录结点到其父节点的map
private Map<TreeNode, TreeNode> node2Parent = new HashMap<>();

public List<Integer> distanceK(TreeNode root, TreeNode targetNode, int distance) {
    if (root == null) {
        return new ArrayList<>();
    }

    // 辅助方法: 构造所有结点的父节点的map方便进行寻找
    dfs(root);

    List<Integer> resList = new ArrayList<>();
    // 记录已经访问过的节点
    HashSet<TreeNode> visitSet = new HashSet<>();
    // 层序遍历的queue
    Queue<TreeNode> queue = new LinkedList<>();
    // 使用目标结点作为第一层遍历的结点入队
    queue.add(targetNode);

    // 整体层序遍历(广度优先遍历)的模板
    while (!queue.isEmpty()) {
        int size = queue.size();

        for (int i = 0; i < size; i++) {
            // 弹出一个结点
            TreeNode node = queue.poll();
            // 一个结点的下一层就是其左右子结点和父节点, 因此这里处理了三次
            // 每个结点都需要判断是否存在且是否已经访问过, 如果没有访问过则加入队列中
            if (node.left != null && !visitSet.contains(node.left)) queue.add(node.left);
            if (node.right != null && !visitSet.contains(node.right)) queue.add(node.right);
            if (node2Parent.containsKey(node) && !visitSet.contains(node2Parent.get(node))) {
                queue.add(node2Parent.get(node));
            }
            // 标记为已访问
            visitSet.add(node);

            // 如果已经到了目标层则记录结果
            if (distance == 0) {
                resList.add(node.val);
            }
        }
        // 遍历完一层, 距离减一
        distance--;
    }
    return resList;
}


private void dfs(TreeNode root) {
    if (root == null) {
        return;
    }
    if (root.left != null) node2Parent.put(root.left, root);
    if (root.right != null) node2Parent.put(root.right, root);
    dfs(root.left);
    dfs(root.right);
}
```

### 路径总和 III(437-中等) 💯

给定一个二叉树的根节点 root，和一个整数 targetSum，求该二叉树里节点值之和等于 targetSum 的路径的数目。路径不需要从根节点开始，也不需要在叶子节点结束，但是路径方向必须是向下的(只能从父节点到子节点)。

![](/zimg/yuque-1777880836866-51.png)

```
输入: root = [10,5,-3,3,2,null,11,3,-2,null,1], targetSum = 8  输出: 3
```

[https://leetcode.cn/problems/path-sum-iii/description/](https://leetcode.cn/problems/path-sum-iii/description/)

**暴力递归**

使用辅助方法，即**返回以 root 为起点的满足条件的路径数量**。原始方法返回以根结点及其左右子结点的路径的和即可。

```
public int pathSum(TreeNode root, long targetSum) {
    if (root == null) {
        return 0;
    }

    // 以当前节点为根节点的路径和的个数+左子树的路径和的个数+右子树的路径和的个数
    int res = rootSum(root, targetSum);
    res += pathSum(root.left, targetSum);
    res += pathSum(root.right, targetSum);

    return res;
}

// 辅助方法: 计算以当前节点为根节点的路径和的个数
public int rootSum(TreeNode root, long targetSum) {
    if (root == null) {
        return 0;
    }

    int res = 0;
    int val = root.val;
    if (val == targetSum) {
        res++;
    }

    // 递归计算左右子树
    res += rootSum(root.left, targetSum - val);
    res += rootSum(root.right, targetSum - val);
    return res;
}
```

**前缀树**

前面的流程里面，其实有很多是重复计算的，所以可以使用**前缀树**解决此问题。

```
private Map<Long, Integer> map = new HashMap<>();
private int res, target;

public int pathSum(TreeNode root, int target) {
    if (root == null) {
        return 0;
    }
    this.target = target;
    map.put(0L, 1);

    dfs(root, root.val);

    return res;
}

private void dfs(TreeNode root, long val) {
    if (map.containsKey(val - target)) {
        res += map.get(val - target);
    }

    map.put(val, map.getOrDefault(val, 0) + 1);

    if (root.left != null) {
        dfs(root.left, val + root.left.val);
    }

    if (root.right != null) {
        dfs(root.right, val + root.right.val);
    }

    map.put(val, map.getOrDefault(val, 0) - 1);
}
```

### 有序数组构造成二叉搜索树(108-简单) 💯

将一个按照升序排列的有序数组，**转换为一棵高度平衡二叉搜索树**。一个高度平衡二叉树是指一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1。

示例：给定有序数组：`[-10, -3, 0, 5, 9]`。一个可能的答案是：`[0, -3, 9, -10, null, 5]`，它可以表示下面这个高度平衡二叉搜索树。

![](/zimg/yuque-1777880836885-52.png)

[https://leetcode-cn.com/problems/convert-sorted-array-to-binary-search-tree](https://leetcode-cn.com/problems/convert-sorted-array-to-binary-search-tree)

这个题可以当做通过数组递归构造二叉树的模板。

**二分切分递归构造法**

由于是高度平衡，所以考虑**使用二分方式进行数组的切分**。**每次取数组的中间元素作为根节点，根节点左边的作为左子树，右边的作为右子树**。如果只有一个元素，那么直接返回这个结点。注意这里辅助方法是采用了**闭区间**。

```
public TreeNode sortedArrayToBST(int[] nums) {
    if (nums == null || nums.length == 0) {
        return null;
    }
    return helper(nums, 0, nums.length - 1);
}

// 辅助方法: 对[left, right]闭区间内的元素进行构造
private TreeNode helper(int[] nums, int left, int right) {
    // 越界了直径返回null
    if (left > right) {
        return null;
    }
    // 只剩一个元素了则返回这个结点
    if (left == right) {
        return new TreeNode(nums[left]);
    }

    // 构造根节点
    int mid = (left + right) / 2;
    TreeNode root = new TreeNode(nums[mid]);

    // 递归构造左右子树
    root.left = helper(nums, left, mid - 1);
    root.right = helper(nums, mid + 1, right);

    // 返回根节点
    return root;
}
```

### 构造最大二叉树(654-中等)

给定一个不含重复元素的整数数组。一个以此数组构建的最大二叉树定义如下：

- 二叉树的根是数组中的最大元素。
- 左子树是通过数组中最大值左边部分构造出的最大二叉树。
- 右子树是通过数组中最大值右边部分构造出的最大二叉树。
- 通过给定的数组构建最大二叉树，并且输出这个树的根节点。

示例：输入：`[3, 2, 1, 6, 0, 5]`。输出：返回下面这棵树的根节点。

![](/zimg/yuque-1777880836896-53.png)

[https://leetcode-cn.com/problems/maximum-binary-tree](https://leetcode-cn.com/problems/maximum-binary-tree)

**递归法**

这个题就是套用了上题的模板，只是构造条件有所变化。**直接寻找当前范围有效数组中的最大值作为根结点，然后以此处的索引为分界线，分别构造其左右子树**。本题也是辅助方法使用了**闭区间**。

```
public TreeNode constructMaximumBinaryTree(int[] nums) {
    if (nums == null || nums.length == 0) {
        return null;
    }
    // 第一次构造的范围是0到length
    return helper(nums, 0, nums.length - 1);
}

// 辅助方法: 对[left, right]闭区间内的元素进行构造
public TreeNode helper(int[] nums, int start, int end) {
    // 越界之后直径返回null
    if (start > end) {
        return null;
    }

    // 获取当前闭区间内的最大值的索引
    int maxIndex = getMaxIndex(nums, start, end);
    // 构造根节点
    TreeNode root = new TreeNode(nums[maxIndex]);

    // 缩小区间并递归构造左右子树
    root.left = helper(nums, start, maxIndex - 1);
    root.right = helper(nums, maxIndex + 1, end);

    // 返回根节点
    return root;
}

private int getMaxIndex(int[] nums, int start, int end) {
    int maxIndex = start;
    int maxNum = nums[start];
    for (int i = start; i <= end; i++) {
        if (nums[i] > maxNum) {
            maxIndex = i;
            maxNum = nums[i];
        }
    }
    return maxIndex;
}
```

### 使用先序遍历构造搜索二叉树(1008-中等) 💯

使用先序遍历队列构造搜索二叉树。

```
输入: [8,5,1,7,10,12]  输出: [8,5,10,1,7,null,12]
```

![](/zimg/yuque-1777880836908-54.png)

[https://leetcode-cn.com/problems/construct-binary-search-tree-from-preorder-traversal/](https://leetcode-cn.com/problems/construct-binary-search-tree-from-preorder-traversal/)

**递归构造法**

本题依然可以套用前面的模板，只不过需要改变一下数组切分的规则。

**搜索二叉树的先序遍历数组的第一个节点一定是搜索二叉树的根节点。根结点后面的值被分为左子树部分(小于根)和右子树(大于根)部分(一定是严格区分的)。因此，数组中比根小的值为根的左子树，找到比根大的值为右子树。找到两者的分界点，然后进行递归构造即可**。

![](/zimg/yuque-1777880836918-55.png)

代码如下：

```
public TreeNode bstFromPreorder(int[] preorder) {
   // 使用辅助方法递归构造树
   return helper(preorder, 0, preorder.length - 1);
}

// 辅助方法: 对[left, right]闭区间内的元素进行构造
private TreeNode helper(int[] preorder, int left, int right) {
   if (left > right) {
       return null;
   }

   if (left == right) {
       return new TreeNode(preorder[left]);
   }

   // 前序遍历时, 根节点就在当前区间的第一个位置
   TreeNode root = new TreeNode(preorder[left]);

   // 找到[left, right]闭区间内左子树的结尾的索引
   int leftIndex = getLeftBorderIndex(preorder, left, right);
   // 递归构造左右子树 左子树范围是闭区间[left + 1, leftIndex] 右子树范围是[leftIndex + 1, right]
   root.left = helper(preorder, left + 1, leftIndex);
   root.right = helper(preorder, leftIndex + 1, right);

   return root;
}

private int getLeftBorderIndex(int[] preorder, int left, int right) {
   // 最左边的就是当前区间的根节点的值
   int rootVal = preorder[left];
   // 这里是在[left, right]闭区间内查找, 从右往左查找, 第一个小于根节点的位置就是左子树的区间末尾
   for (int i = right; i >= left; i--) {
       if (preorder[i] < rootVal) {
           return i;
       }
   }
   return left;
}
```

### 从先序与中序遍历序列重建二叉树(105-困难) 💯🔑

根据二叉树的前序遍历和中序遍历的结果，重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。

![](/zimg/yuque-1777880836937-56.png)

[https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/description/](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/description/)

**哈希表+递归构造法**

整体还是递归构造的思路，只不过需要寻找到切分数组进行递归的方法。

**前序遍历数组的第 1 个数(索引为 0)的数一定是二叉树的根结点，于是可以在中序遍历中找这个根结点的索引，然后把 "前序遍历数组" 和 "中序遍历数组" 分为两个相邻的部分，就分别对应二叉树的左子树和右子树，而且这两个字子树所包含的结点数量是相同的，这样就能根据这个数量对前序遍历序列进行切分。之后分别递归完成就可以了**。

这里可以**使用一个 Map 缓存中序遍历数组每个值对应的索引**，这样找根节点的索引就比较方便。

![](/zimg/yuque-1777880836949-57.png)

代码如下：

```
// 存放中序遍历数值到索引的映射 便于寻找根节点在中序序列中的位置
private Map<Integer, Integer> inOrderNum2Index = new HashMap<>();

private int[] preorder;

public TreeNode buildTree(int[] preorder, int[] inorder) {
    // 将中序遍历的结果放入Map并记录其索引位置
    for(int i = 0; i < inorder.length; i++) {
        inOrderNum2Index.put(inorder[i], i);
    }

    this.preorder = preorder;
    return buildTree(0, preorder.length - 1, 0, inorder.length - 1);
}


/**
 * 使用数组preorder在索引闭区间[preLeft, preRight]中的所有元素
 * 和数组inorder在索引闭区间[inLeft, inRight]中的所有元素构造二叉树
 *
 * @param preLeft  前序遍历结果的左边界
 * @param preRight 前序遍历结果的右边界
 * @param inLeft   后序遍历结果的左边界
 * @param inRight  后序遍历结果的右边界
 * @return 二叉树的根结点
 */
private TreeNode buildTree(int preLeft, int preRight, int inLeft, int inRight) {
    // 终止条件
    if (preLeft > preRight || inLeft > inRight) {
        return null;
    }
    // 先序遍历的第一个元素就是根节点
    int rootVal = preorder[preLeft];
    int rootIndex = inOrderNum2Index.get(rootVal);

    TreeNode root = new TreeNode(rootVal);

    // 这里的左右边界不要记忆, 画图推导即可
    root.left = buildTree(preLeft + 1, rootIndex - inLeft + preLeft, inLeft, rootIndex - 1);
    root.right = buildTree(rootIndex - inLeft + preLeft + 1, preRight, rootIndex + 1, inRight);
    return root;
}
```

### 从中序与后序遍历序列构造二叉树(106-困难) 💯

根据一棵树的中序遍历与后序遍历构造二叉树。注意树中没有重复的元素。例如给出：

```
中序遍历 inorder = [9,3,15,20,7]   后序遍历 postorder = [9,15,7,20,3]
```

返回如下的二叉树：

![](/zimg/yuque-1777880836974-58.png)

[https://leetcode-cn.com/problems/construct-binary-tree-from-inorder-and-postorder-traversal](https://leetcode-cn.com/problems/construct-binary-tree-from-inorder-and-postorder-traversal)

**哈希表+递归构造法**

其实也是使用前面的模板。与 [从先序与中序遍历序列重建二叉树(105-困难)](https://www.yuque.com/daerwendecha/qwuv92/gmi10ot8m7wa1ugt#JbHPn) 基本一样，区别在于**根节点是在数组的最后一个位置**。

根据中序遍历和后续遍历数组的特性对树的还原过程分析：

- 首先在后序遍历序列中找到根节点(数组最后一个元素)。
- 根据根节点在中序遍历序列中找到根节点的位置。
- 根据根节点的位置将中序遍历序列分为左子树和右子树。进行切分数组。
- **根据根节点的位置确定左子树和右子树在中序数组和后续数组中的左右边界位置。(重要！)确定边界之后将左右边界的数组传入递归函数**。
- 递归构造左子树和右子树。
- 返回根节点。

这里的区间范围也别死记硬背，画图进行推导即可。

```
// 使用辅助的HashMap存储元素, 键为中序数组元素值, 值为中序数组索引
HashMap<Integer, Integer> postMap = new HashMap<>();

private int[] post;

public TreeNode buildTree(int[] inOrder, int[] postOrder) {
    // 将中序数组的值及其对应的索引全部放入到Map中
    for (int i = 0; i < inOrder.length; i++) {
        postMap.put(inOrder[i], i);
    }
    post = postOrder;
    // 使用辅助方法
    return buildTree(0, inOrder.length - 1, 0, post.length - 1);
}

public TreeNode buildTree(int inStart, int inEnd, int postStart, int postEnd) {
    if (inEnd < inStart || postEnd < postStart) {
        return null;
    }
    // 根节点等于后序数组的最后一个元素值
    int rootVal = post[postEnd];

    // 从Map中找到等于这个值的元素的中序数组索引
    int rootIndex = postMap.get(rootVal);

    // 构造根结点
    TreeNode root = new TreeNode(rootVal);

    // 递归构造左右子树 左右子树的边界确定是最重要的 自己画个图看看!
    root.left = buildTree(inStart, rootIndex - 1, postStart, postStart + rootIndex - inStart - 1);
    root.right = buildTree(rootIndex + 1, inEnd, postStart + rootIndex - inStart, postEnd - 1);

    // 返回构造的根节点
    return root;
}
```

### 递增顺序查找树(897-简单)

给你一个二叉搜索树的 root，按中序遍历重新排列树，使树中最左边的结点现在是树的根，并且每个结点没有左子结点，只有一个右子结点。退化成一个链表。

![](/zimg/yuque-1777880836985-59.png)

[https://leetcode-cn.com/problems/increasing-order-search-tree](https://leetcode-cn.com/problems/increasing-order-search-tree)

**递归构造法**

直接使用递归方式修改指针。引入一个 dummy 结点和 cur 结点，使用中序递归遍历二叉树，处理一个结点时，就把该结点的左子树置为 null，并将其设置为 cur 的右子树，并移动 cur。持续操作即可。

```
// cur结点沿着链表移动进行设置
private TreeNode cur;

public TreeNode increasingBST(TreeNode root) {
    // 引入一个dummy节点
    TreeNode dummy = new TreeNode(0);
    cur = dummy;

    dfs(root);

    // 返回根节点
    return dummy.right;
}

// 整体采用中序递归进行遍历构造
public void dfs(TreeNode node) {
    if (node == null) {
        return;
    }

    dfs(node.left);

    // cur的左结点都为null
    node.left = null;
    // 把cur节点的右变成当前节点node
    cur.right = node;
    // 不断移动cur节点到链表尾部
    cur = node;

    dfs(node.right);
}
```

### 将二叉搜索树转换为双向链表(LCR-155-中等) 🔑

输入一棵二叉搜索树，将该二叉搜索树转换成一个排序的循环双向链表。要求不能创建任何新的结点，只能调整树中结点指针的指向。

输入：`root = [4, 2, 5, 1, 3]`。输出如下：

![](/zimg/yuque-1777880837002-60.png)

[https://leetcode.cn/problems/er-cha-sou-suo-shu-yu-shuang-xiang-lian-biao-lcof/description/](https://leetcode.cn/problems/er-cha-sou-suo-shu-yu-shuang-xiang-lian-biao-lcof/description/)

**中序递归构造法**

构造出循环双向链表，有三个要点：

- **排序链表**：节点应从小到大排序，因此应使用中序遍历从小到大访问树的节点。
- **双向链表**：在构建相邻节点的引用关系时，需要同时构造双向指针。
- **循环链表**：最后需要将双向链表的收尾相连。

这里引入一个 dummy 结点，指向最开始树的 root 结点，后续使用中序遍历来构造双向链表，构造之后 cur 指向链表尾部，再将 cur 与最开始的 root 结点建立双向连接，最后返回最开始的 root 结点即可。

```
public Node treeToDoublyList(Node root) {
    if (root == null) {
        return root;
    }
    // 将dummy指向最开始的头结点
    dummy.right = root;

    // 中序递归构造双向链表
    dfs(root);

    // 此时cur已经指向链表末尾, 由于需要构造循环链表, 所以需要将cur与初始的root进行双向连接
    cur.right = dummy.right;
    dummy.right.left = cur;

    // 返回dummy的下一个结点
    return dummy.right;
}

// 引入一个dummy头结点
private Node dummy = new Node(0);
private Node cur = dummy;

// 中序遍历
private void dfs(Node root) {
    if (root == null) {
        return;
    }

    dfs(root.left);

    // 构造双向指针
    cur.right = root;
    root.left = cur;
    // 移动cur
    cur = cur.right;

    dfs(root.right);
}
```

### 二叉树展开为链表(114-中等) 💯

给定一颗二叉树，将其展开为一条链表，展开后的单链表应该与二叉树先序遍历顺序相同。

![](/zimg/yuque-1777880837017-61.png)

[https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/description/](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/description/)

**迭代展开法**

这个题使用先序递归解法会比较难以理解，因为涉及指针的修改，**可以使用先序遍历的迭代法，在处理数据时进行指针的修改与转化**。

```
public void flatten(TreeNode root) {
    if (root == null) return;

    Stack<TreeNode> stack = new Stack<>();
    stack.push(root);
    // 遍历指针
    TreeNode cur = new TreeNode(0);

    while (!stack.isEmpty()) {
        TreeNode node = stack.pop();

        // 处理数据! 处理左右指针
        cur.right = node;
        cur.left = null;
        // cur往后移动
        cur = node;

        // 先压入右子树, 再压入左子树
        if (node.right != null) stack.push(node.right);
        if (node.left != null) stack.push(node.left);
    }
}
```

### 二叉树的序列化与反序列化(297-困难) 🔑

设计一个算法来实现二叉树的序列化与反序列化。这里不限定序列/反序列化算法执行逻辑，只需要保证一个二叉树可以被序列化为一个字符串并且将这个字符串反序列化为原始的树结构。

[https://leetcode-cn.com/problems/serialize-and-deserialize-binary-tree/](https://leetcode-cn.com/problems/serialize-and-deserialize-binary-tree/)

**层序构造法**

利用层序遍历的方式实现序列化与反序列化。层序遍历时将结点的所有叶子结点都会记录下来，包含 null 的结点，同时每个结点之间用特殊符号分隔。构造二叉树时，需要按照特殊符号分隔出不同的结点，注意需要对 null 和普通结点进行区分。

![](/zimg/yuque-1777880837034-62.png)

代码如下：

```
public String serialize(TreeNode root) {
    if (root == null) {
        return "";
    }
    StringBuilder builder = new StringBuilder();
    Queue<TreeNode> queue = new LinkedList<>();
    queue.add(root);

    // 层序遍历进行序列化
    while (!queue.isEmpty()) {
        TreeNode node = queue.poll();
        // 结点不为空则记录当前值并采用逗号分隔
        if (node != null) {
            builder.append(node.val).append(",");
            // 注意这里需要同时把左右子树添加到队列中, 有可能加进去的是null
            queue.add(node.left);
            queue.add(node.right);
        } else {
            // 结点为空序列化为null
            builder.append("null").append(",");
        }
    }
    // 删除最后一个分隔符逗号
    builder.deleteCharAt(builder.length() - 1);

    return builder.toString();
}

public TreeNode deserialize(String data) {
    if ("".equals(data)) {
        return null;
    }
    // 去掉前后括号并按照逗号切分为不同的值
    LinkedList<String> values = new LinkedList<>(Arrays.asList(data.split(",")));
    // 弹出头结点构造root
    TreeNode root = new TreeNode(Integer.parseInt(values.poll()));

    Queue<TreeNode> queue = new LinkedList<>();
    queue.add(root);

    // 逐渐遍历并构造树
    while (!queue.isEmpty()) {
        TreeNode node = queue.poll();
        // 构造左右子树, 注意这里无论子结点是否都会一直弹出元素, 代表构造过了一个结点
        // 构造后将结点放入队列中
        if (!values.peek().equals("null")) {
            node.left = new TreeNode(Integer.parseInt(values.peek()));
            queue.add(node.left);
        }
        values.poll();

        if (!values.peek().equals("null")) {
            node.right = new TreeNode(Integer.parseInt(values.peek()));
            queue.add(node.right);
        }
        values.poll();
    }
    return root;
}
```

### 二叉树的镜像(226-简单) 💯🔑

翻转一棵二叉树，得到二叉树的镜像。

**自底向上递归法**

其实就是**交换了每个结点的左右子树**。所以可以**用递归的方式自底向上进行交换**，这样先操作子节点再操作根节点就是典型的后序遍历。所以整体递归的架构是后序遍历。

public TreeNode invertTree(TreeNode root) {

        return null;
    }

    // 先保存一下左右的子树
    // 注意这里必须用临时结点,不然会把指针带到递归中引发错误
    TreeNode right = invertTree(root.right);
    TreeNode left = invertTree(root.left);

    // 然后交换当前根结点的左右子树
    root.left = right;
    root.right = left;

    return root;
}

### 合并二叉树(617-简单)

给定两个二叉树，想象当你将它们中的一个覆盖到另一个上时，两个二叉树的一些节点便会重叠。需要将他们合并为一个新的二叉树。合并的规则是如果两个节点重叠，那么将他们的值相加作为节点合并后的新值，否则不为 NULL 的节点将直接作为新二叉树的节点。

注意：合并必须从两个树的根节点开始。

![](/zimg/yuque-1777880837098-64.png)

[https://leetcode-cn.com/problems/merge-two-binary-trees](https://leetcode-cn.com/problems/merge-two-binary-trees)

**自底向上递归删除法**

**可以对这两棵树同时进行先序遍历，并将对应的节点进行合并**。在遍历时，如果两棵树的当前节点均不为空，就将它们的值进行相加，并对它们的左孩子和右孩子进行递归合并；如果其中有一棵树为空，那么直接返回另一颗树作为结果；如果两棵树均为空，此时返回任意一棵树均可(因为都是空)。

下面合并的时候把 `t1 t2` 的值直接合并到 t1 之上。

![](/zimg/yuque-1777880837120-65.gif)

代码如下：

```
public TreeNode mergeTrees(TreeNode root1, TreeNode root2) {
    // Base case:只要其中一个为null直接返回另一个即可
    if (root1 == null) {
        return root2;
    }

    if (root2 == null) {
        return root1;
    }

    // 两个结点都不为空则将值合并到t1树上
    root1.val = root1.val + root2.val;

    // 递归构造左右子树
    root1.left = mergeTrees(root1.left, root2.left);
    root1.right = mergeTrees(root1.right, root2.right);

    // 返回合并后的树
    return root1;
}
```

**层序遍历法**

首先把两棵树的根节点入栈，栈中的每个元素都会存放两个根节点，并且栈顶的元素表示当前需要处理的节点。在迭代的每一步中，取出栈顶的元素并把它移出栈，并将它们的值相加。随后分别考虑这两个节点的左孩子和右孩子，如果两个节点都有左孩子，那么就将左孩子入栈；如果只有一个节点有左孩子，那么将其作为第一个节点的左孩子，不入栈；如果都没有左孩子，那么不用做任何事情。对于右孩子同理。最后返回第一棵树的根节点作为答案。

```
public TreeNode mergeTrees(TreeNode root1, TreeNode root2) {
    // Base case
    if (root1 == null) {
        return root2;
    }

    // 两个结点作为一组进行层序遍历
    Queue<TreeNode[]> queue = new LinkedList<>();
    queue.add(new TreeNode[]{root1, root2});

    // 把值都合并到root1上
    while (!queue.isEmpty()) {
        TreeNode[] pair = queue.poll();
        // 两个结点都为空
        if (pair[0] == null || pair[1] == null) {
            continue;
        }

        // 合并值到一个结点上
        pair[0].val = pair[0].val + pair[1].val;

        // 如果root1为空则之间使用root2
        if (pair[0].left == null) {
            pair[0].left = pair[1].left;
        } else {
            queue.add(new TreeNode[]{pair[0].left, pair[1].left});
        }

        // 如果root1为空则之间使用root2
        if (pair[0].right == null) {
            pair[0].right = pair[1].right;
        } else {
            queue.add(new TreeNode[]{pair[0].right, pair[1].right});
        }
    }

    return root1;
}
```

### 删除给定值的叶子节点(1325-中等)

给你一棵以 root 为根的二叉树和一个整数 target，请删除所有值为 target 的 叶子节点。注意一旦删除值为 target 的叶子节点，它的父节点就可能变成叶子节点；如果新叶子节点的值恰好也是 target，那么这个节点也应该被删除。也就是需要重复此过程直到不能继续删除。

**自底向上递归删除法**

由于是删除子结点，**所以是自底向上，使用后序遍历方式进行删除**。**遇到叶子结点且是目标值则返回 null，否则返回自己**。

```
// 先处理叶子所以整个是自底向上的后序递归处理
public TreeNode removeLeafNodes(TreeNode root, int target) {
    if (root == null) {
        return null;
    }

    // 递归构造根结点的左右子树
    root.left = removeLeafNodes(root.left, target);
    root.right = removeLeafNodes(root.right, target);

    // 如果是子结点且刚好需要删除, 则返回null, 否则返回root本身
    if (root.left == null && root.right == null && root.val == target) {
        // 直接返回null代表删除
        return null;
    }

    // 这个结点不删除则直接返回
    return root;
}
```

**自底向上递归删除法**

其实这个题就是 [删除给定值的叶子节点(1325-中等)](https://www.yuque.com/daerwendecha/qwuv92/gmi10ot8m7wa1ugt#jceVA) 的变体，这里即 `target = 0`。整体依然是从叶子结点开始操作，所以是自底向上的后序递归。

```
// 自底向上的后序递归删除
public TreeNode pruneTree(TreeNode root) {
    if (root == null) {
        return null;
    }

    root.left = pruneTree(root.left);
    root.right = pruneTree(root.right);

    // 如果遇到子结点其值为0则删除此结点也就是返回null
    if (root.left == null && root.right == null && root.val == 0) {
        return null;
    }

    // 否则啥也不干
    return root;
}

### 删除二叉搜索树中的节点(450-中等)

给定一个二叉搜索树的根节点 root 和一个值 key，删除二叉搜索树中的 key 对应的节点，并保证二叉搜索树的性质不变。返回二叉搜索树(有可能被更新)的根节点的引用。示例：

![](/zimg/yuque-1777880837272-70.png)

[https://leetcode.cn/problems/delete-node-in-a-bst/description/](https://leetcode.cn/problems/delete-node-in-a-bst/description/)

**递归法**

删除二叉搜索树结点时有以下几种情况：

- 没找到删除的节点，遍历到空节点直接返回了。
- 如果根节点不是目标值，则根据根节点与目标值的大小关系，递归往左右子树寻找目标结点。
- 找到删除的节点后：

- 如果左右孩子都为空(叶子节点)，直接删除节点，返回 NULL 为根节点。
- 如果左右孩子只有一个，则直接删除目标结点，使用这个孩子结点部位并返回。
- 如果左右孩子节点都不为空，则寻找右子树的最小的结点来替换目标结点即可。因为根据二叉搜索树的性质，目标结点的右子树的最小结点就是中序遍历的下一个结点，而且肯定是一个叶子结点，替换后刚好符合条件。可以看看动图。

![](/zimg/yuque-1777880837287-71.gif)

动画中的二叉搜索树中，删除元素 7，那么删除节点(元素 7)的左孩子就是 5，删除节点(元素 7)的右子树的最左面节点是元素 8。将删除节点(元素 7)的左孩子放到删除节点(元素 7)的右子树的最左面节点(元素 8)的左孩子上，就是把 5 为根节点的子树移到了 8 的左孩子的位置。要删除的节点(元素 7)的右孩子(元素 9)为新的根节点。

代码如下：

```
public TreeNode deleteNode(TreeNode root, int key) {
    // 未找到元素直接返回
    if (root == null) {
        return null;
    }
    // 找到需要删除的元素
    if (root.val == key) {
        // 如果左子树为空则返回右子树
        if (root.left == null) {
            return root.right;
            // 如果右子树为空则返回左子树
        } else if (root.right == null) {
            return root.left;
            // 找到右子树中最小的结点来替换当前结点
        } else {
            // 切换到右子树并一直往左走也就是寻找这棵子二叉树的最小结点(最左下角)
            TreeNode cur = root.right;
            while (cur.left != null) {
                cur = cur.left;
            }
            // 此时cur就是需要替换的结点
            cur.left = root.left;
            root = root.right;
            return root;
        }
    }

    // 往左右子树递归寻找删除
    if (root.val > key) {
        root.left = deleteNode(root.left, key);
    }
    if (root.val < key) {
        root.right = deleteNode(root.right, key);
    }

    return root;
}
```

### 不同的二叉搜索树 II(95-中等)

给定一个整数 n，生成所有由 `1 ... n` 为节点所组成的 二叉搜索树。

```
输入:3   输出有五种搜索二叉树
   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3
```

[https://leetcode.cn/problems/unique-binary-search-trees-ii/description/](https://leetcode.cn/problems/unique-binary-search-trees-ii/description/)

**暴力回溯法**

根据二叉搜索树的特性，若某个子树的根节点为 root，那么 root 的左子树任意节点值均比 root.val 要小，root 的右子树任意节点值均比 root.val 要大。

因此，假设使用 `[left, right]` 连续段来构造二叉搜索树，并且选择了节点 t 作为二叉搜索树的根节点时，那么使用 `[left, t − 1]` 构造出来的二叉搜索树均可作为根节点 t 的左子树；同理，使用 `[t + 1, right]` 构造出来的二叉搜索树均可作为根节点 t 的右子树。

也就是说，可以设计递归函数 `List<TreeNode> dfs(int left, int right)`，含义为使用连续段 `[left, right]` 进行二叉搜索树构造，并返回相应集合。

最终答案则为 `dfs(1, n)`，起始可以枚举 `[1, n]` 范围内的的每个数 t 作为根节点，并递归 `dfs(left, t - 1)` 和 `dfs(t + 1, right)` 获取左右子树的集合 leftSubtrees 和 rightSubtrees，结合「乘法原理」，枚举任意左子树和任意右子树，即可得到 t 作为根节点的二叉搜索树方案集，枚举所有 t 后即可得到所有二叉搜索树的总集。

代码如下：

```
public List<TreeNode> generateTrees(int n) {
    if (n == 0) {
        return new ArrayList<>();
    }
    return generateTrees(1, n);
}

// 递归方法: 生成[left, right]区间内所有可能的二叉搜索树
private List<TreeNode> generateTrees(int left, int right) {
    List<TreeNode> allTrees = new ArrayList<>();
    if (left > right) {
        allTrees.add(null);
        return allTrees;
    }

    for (int i = left; i <= right; i++) {
        // 递归生成所有可能的左子树
        List<TreeNode> leftTrees = generateTrees(left, i - 1);
        // 递归生成所有可能的右子树
        List<TreeNode> rightTrees = generateTrees(i + 1, right);

        // 将左子树和右子树组合到根节点
        for (TreeNode leftTree : leftTrees) {
            for (TreeNode rightTree : rightTrees) {
                TreeNode root = new TreeNode(i);
                root.left = leftTree;
                root.right = rightTree;
                allTrees.add(root);
            }
        }
    }
    return allTrees;
}
```

### 前缀树(208-中等)

前缀树 是一种树形数据结构，用于高效地存储和检索字符串数据集中的键。这一数据结构有相当多的应用情景，例如自动补全和拼写检查。

请实现 Trie 类：

- `Trie()` 初始化前缀树对象。
- `void insert(String word)` 向前缀树中插入字符串 word。
- `boolean search(String word)` 如果字符串 word 在前缀树中，返回 true（即在检索之前已经插入）；否则返回 false。
- `boolean startsWith(String prefix)` 如果之前已经插入的字符串 word 的前缀之一为 prefix，返回 true；否则返回 false。

[https://leetcode.cn/problems/implement-trie-prefix-tree/description/](https://leetcode.cn/problems/implement-trie-prefix-tree/description/)

**题目题解**

准备一个 26 个字母的映射节点，一个标记当前字符是否是某个单词的结尾的标记变量 isEnd。

- `insert()`：拿到当前的字母数组 next，依次遍历传入的单词的字符。如果当前字符不在 next 中，生成一个新的节点，并将当前的节点指向新生成的节点。如果当前字符在 next 中，将该字符所在的节点指向当前的节点。在单词被遍历结束时，给当前所在的节点设置 isEnd 的标记，表示以当前字符结束可以形成一个字母。
- `search()` 和 `startsWith()`：遍历字母的所有字符，如果当前字符所在的节点不存在，返回 false，说明单词还没搜完，树已经搜索完了。不断将遍历到的节点赋值给当前节点。注意：`search()` 和 `startsWith()` 的区别需要判断当前节点是否是 isEnd，如果是的话，证明是搜索到了，但是 `startsWith()` 则不需要满足此条件。

因此这里定义一个结点类，其包含两个属性，**一个是 26 个字符的子结点数组，一个是是否为字符串结尾**。

代码如下：

```
// 结点类
class Node {
    Node[] children = new Node[26];
    boolean isEnd;
}

class Trie {
    // 根结点
    private Node root;

    // 初始化根结点
    public Trie() {
        root = new Node();
    }

    public void insert(String word) {
        Node cur = root;
        // 遍历单词的每个字符, 持续构造Trie树
        for (char c : word.toCharArray()) {
            c -= 'a';
            // 如果当前结点的孩子结点为空则新建一个结点
            if (cur.children[c] == null) {
                cur.children[c] = new Node();
            }
            // 移动到下一个结点
            cur = cur.children[c];
        }
        // 标记当前结点是一个单词的结尾
        cur.isEnd = true;
    }

    public boolean search(String word) {
        return find(word) == 2;
    }

    public boolean startsWith(String prefix) {
        return find(prefix) != 0;
    }

    private int find(String word) {
        // 从根结点开始查找
        Node cur = root;
        for (char c : word.toCharArray()) {
            c -= 'a';
            // 未找到当前字符则返回0
            if (cur.children[c] == null) {
                return 0;
            }
            cur = cur.children[c];
        }
        // 所有字符都找到了, 判断当前结点是否是一个单词的结尾
        return cur.isEnd ? 2 : 1;
    }
}
```
