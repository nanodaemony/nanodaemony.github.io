### 服务器初始化

```
# 更新软件
yum update -y

# 关闭防火墙
systemctl stop firewalld
systemctl disable firewalld

# 安装必要软件
yum install -y yum-utils
yum install -y device-mapper-persistent-data
yum install -y lvm2

yum install -y curl
yum install -y apt-transport-https ca-certificates curl software-properties-common gnupg2
```

安装 Docker

```
# 先删除旧版
yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-engine
# 使用官方安装脚本安装
curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun
# 重启docker
systemctl start docker

mkdir -p /etc/docker
# 配置 Docker daemon, 设置cgroupdriver为systemd
cat <<EOF | sudo tee /etc/docker/daemon.json
{
  "exec-opts": ["native.cgroupdriver=systemd"],
  "log-driver": "json-file",
  "log-opts": {
    "max-size": "100m"
  },
  "storage-driver": "overlay2"
}
EOF

# 重启 docker 后台服务
$ sudo systemctl daemon-reload
$ sudo systemctl restart docker
```

## 服务器搭建 Nginx

创建一个 `nginx.conf` 文件：

```
server {
  listen 80;
  server_name localhost;

  root /usr/share/nginx/html;
  index index.html;

  location / {
    try_files $uri $uri/ /index.html;
  }
}
```

运行 Nginx 容器并映射端口。使用 Docker 命令运行 Nginx 容器，并将容器的 80 端口映射到服务器的 8081 端口，确保本地的 `/root/note` 目录的内容被挂载到 Nginx 容器的静态文件目录中，并且将容器的 80 端口映射到服务器的 8081 端口。

```
docker run -d \
    --name vuepress-nginx \
    -p 8081:80 \
    -v /root/note:/usr/share/nginx/html \
    nginx
```

## MAC 新增命令

MAC 新增自己的命令行脚本工具：

1. 开发一个普通的脚本, 比如 `commit-code.sh`。
2. 复制到 `/usr/local/bin` 目录。
3. 去掉脚本后缀：`mv commit-code.sh commit-code`。
4. 赋予执行权限 `chmod 777 commit-code`。
5. 重启一下 Bash 即可使用这个脚本。
