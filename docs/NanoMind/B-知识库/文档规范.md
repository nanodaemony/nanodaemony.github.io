## 总结

技术学习必问：

- 怎么用? 先学会用，再学习原理。
- 使用场景是什么? 知道可以用在什么地方。
- 最佳实践?
- 工作中用到了吗? 工作中如果用到了，理解可能会更深。
- 是否存在什么问题? 是否有缺陷? 是否有替代的方案。
- 实现原理是啥?

笔记整理原则：

- 尽量减少标题，特别是无用的标题，比如：总结，开篇词这种。
- 重要的部分在对应的章节上打上星号。
- 使用英文的标点符号。
- Java 方法名后应该有 `()`，方法名加粗时将 `()` 也加粗。

## 路线

**整理已有的笔记资料**

- 已经完成的章节文件标记为 🌸。
- 重要章节的标题后需要加上 🌟。
- 如果一个大类也完成整理那就文件夹名上加上 🌸。
- **用不到的一定要删掉。没用的东西删掉。拒绝又臭又长的内容**。
- 过时的内容相关的细节也删掉，不过可以记录一下历史发展和淘汰原因。

**看书**

- 一定要先找找有没有对应的 EPUB 版本，方便复制出来内容。

**如何选择接下来应该学什么?**

- 对该内容有刚需，所做工作正好有涉及。
- 有用的帖子及时整理。

**谨记**

- 需要自己补充一下各个知识点之间的关联，发展脉络，理解比记忆更重要。

## 利用思源笔记导入 Epub

思源笔记安装导入插件。导入时先删除垃圾文件。

![](/zimg/%E6%96%87%E6%A1%A3%E8%A7%84%E8%8C%83-1775305320963.png)

导入到思源笔记后，选择导出 `Markdown.zip`：

![](/zimg/%E6%96%87%E6%A1%A3%E8%A7%84%E8%8C%83-1775305418792.png)

语雀进行导入，选择导入 Markdown 打包后的压缩文件：

![](/zimg/%E6%96%87%E6%A1%A3%E8%A7%84%E8%8C%83-1775305442167.png)

即可导入。

注意：对于极客时间的 Epub，有的导入之后图片上下有两行 HTML 的数据，需要先删掉再从思源导出 ZIP，不然语雀会因为这些 HTML 解析得格式混乱。

![](/zimg/%E6%96%87%E6%A1%A3%E8%A7%84%E8%8C%83-1775305463671.png)

## 批量导入极客时间

极客时间专栏如有有 Markdown 文件合集，只需要先**整个压缩为一个 ZIP 文件**，然后再导入即可。

注意：**HTML 格式的极客时间无法导入，导入后无法编辑**。
