## 项目简介

小方格是一个优雅的模块化工具集合应用，采用 Flutter 构建，支持 Android 和 iOS 平台。应用采用插件化架构，每个工具都是独立的模块，可快速注册和扩展。

### 基本信息

| 属性     | 值                                       |
| -------- | ---------------------------------------- |
| 项目名称 | LittleGrid（小方格）                     |
| 版本     | 1.0.0+1                                  |
| 开发语言 | Dart                                     |
| 框架     | Flutter >= 3.0.0                         |
| 支持平台 | Android, iOS, Web, Windows, macOS, Linux |
| 开源协议 | MIT                                      |

## 核心架构设计

### 工具模块化架构

小方格采用插件化架构，每个工具都是一个独立的模块，实现了 `ToolModule` 接口。

#### ToolModule 接口定义

```dart
abstract class ToolModule {
  String get id;                    // 唯一标识
  String get name;                  // 显示名称
  String get version;               // 版本号
  IconData get icon;                // 图标
  ToolCategory get category;        // 分类
  int get gridSize;                 // 格子大小
  Widget buildPage(BuildContext context);  // 构建页面
  ToolSettings? get settings;       // 设置（可选）
  Future<void> onInit() async {}    // 初始化
  Future<void> onDispose() async {} // 销毁
  void onEnter() {}                 // 进入
  void onExit() {}                  // 退出
}
```

#### 工具分类

```dart
enum ToolCategory {
  life,   // 生活实用
  game,   // 趣味随机
  calc,   // 计算工具
}
```

#### 工具注册机制

通过 `ToolRegistry` 统一管理所有工具：

```dart
// 注册工具
ToolRegistry.register(CoinTool());
ToolRegistry.register(DiceTool());
ToolRegistry.register(TodoTool());
// ...

// 获取所有工具
List<ToolModule> tools = ToolRegistry.getAll();

// 按分类获取
List<ToolModule> lifeTools = ToolRegistry.getByCategory(ToolCategory.life);
```

## 开发指南

### 添加新工具

1. 在 `app/lib/tools/` 下创建新工具目录
2. 实现 `ToolModule` 接口
3. 在 `main.dart` 中注册工具

```dart
// tools/my_tool/my_tool.dart
class MyTool implements ToolModule {
  @override
  String get id => 'my_tool';
  @override
  String get name => '我的工具';
  @override
  IconData get icon => Icons.extension;
  @override
  ToolCategory get category => ToolCategory.life;
  @override
  int get gridSize => 1;
  @override
  Widget buildPage(BuildContext context) => MyToolPage();
}

// main.dart
ToolRegistry.register(MyTool());
```
