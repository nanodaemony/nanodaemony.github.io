    # 5.知识库管理

    # 网站发布

    # 文件编译

    基本流程如下：

    • 打开思源笔记软件，执行 darwin 项目中的脚本，将思源笔记的所有数据导出 Markdown 文件到 Vuepress 的目录下，这里为 PangolinNote 项目。
    • 使用 VS Code 打开 PangolinNote 项目。打开根目录下面的部署说明文件。按照步骤操作。执行 npm run build​ 之后，就能在本地生成编译好的文件了。
    # 发布GitHub

    执行部署说明文件中的上传到 GitHub 的命令即可。注意，有可能本地提交了一个 git，但是 push 的时候没有网失败了，那么就找到 PangolinNote\docs\.vuepress\dist​ 目录，修改任何一个文件夹（相当于有个新的提交），查询执行一次上传即可。

    # 发布到云服务器

    前置条件：服务器安装 Docker，使用 Docker 安装 Nginx，配置文件如下：

    server {
    listen 80;
    server_name localhost;

    root /usr/share/nginx/html;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
