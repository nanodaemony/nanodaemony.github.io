    摄影基础

    # 1.摄影基础

    # 概述

    摄影的核心在于捕捉光线. The art of capture light.

    摄影是一个做减法的过程.

    # 曝光

    曝光就是图像的明暗程度, 正常称为适度曝光. 太暗就是曝光不足, 太亮就是过曝光.

    # 1.曝光三角

    设置相机调整曝光程度: 光圈, 快门速度, 感光度. 这三个称为曝光三角.

    适度曝光是光线和曝光三角设置的平衡.

    # (1)光圈

    光圈(通光孔径)是相机镜头里让光线进入的孔, 可以变大变小.

    光圈变大, 进的光就多. 因此增大光圈, 会使得图片变亮. 反之变暗. 因此在暗处拍照, 可以加大光圈.

    光圈档位/光圈值可以表示光圈的大小程度. 其值为: F/1, F/1.4, F/2, F/2.8, F/4​ 及以上等. ​F 后面的数值越小, 光圈越大​, 反之.

    不是所有镜头都能调节成任何光圈大小.

    景深: 指照片里有多少是成像清晰的. 比如浅的景深就是很近的距离是清晰的, 远处是模糊的.

    光圈的设置也会影响景深, 通常大光圈与小光圈相比, 大光圈有更浅的景深​ **, ** 小光圈的景深更深.

    • 如果希望拍摄远处的风景, 希望尽量多的元素都清晰, 需要用小光圈.
    • 如果是拍人像, 希望背景虚化, 就用大光圈.
    # (2)快门速度

    快门用于阻挡光纤到达传感器. 按下快门会让快门迅速打开, 然后迅速关闭, 让光线进入.

    快门速度就是快门打开到关闭的时间, 一般取值有 1/30s, 1/60s, 1/125s ... 1/8000s 等. 快门速度提高了, 进入快门的光线就会变少, 图片就会变暗. 快门速度慢的话, 可能就会导致光线过多, 引起过曝.

    **快门速度越快, 光线进入越少. ​如果拍摄的场景很暗, 那么可以降低快门速度. **

    **快门速度与运动模糊. 如果是拍运动的物体, 就需要提高快门速度, 来定格目标的动作. **

    "相机抖动"是由于聚齐相机拍照时, 手部产生的抖动引起. 尝试以 1/80s, 1/100s 的快门速度拍摄, 以减少相机抖动.

    # (3)感光度ISO

    ISO 指相机传感器对光的敏感度.

    **ISO 通过数码方式增加或者减少曝光. **​增加 ISO, 增加曝光. 降低 ISO, 减少曝光​ **. **

    ISO 等级的取值通常先从 ISO 100 开始, 然后双倍递增, 如 200, 400, 800...

    由于是通过数码方式影响曝光, 所以 ISO 过高会给图像带来数字噪点和颗粒.

    大多数入门级相机建议保持 ISO 1600 以下.

    **基础感光度(ISO): 相机最低感光度设置, 不会通过数码方式增加曝光. 因此这时拍出的图像通常品质较高, 数字噪点较少. **

    如果在室内或者较暗的场景时, 首先得降低快门速度, 并尽量开大光圈, 如果这还不能拍出合适曝光的图像, 就可以考虑增大 ISO.

    # (4)曝光三角

    调整三个参数达到最佳的曝光.

    光线较强的场景: 从基础感光度100开始, 接着调整光圈档位, 然后调节快门速度. 也可以调整光圈调节景深, 最后通过快门速度来满足曝光.

    一般: 先 ISO --> 光圈 --> 快门速度.

    # 2.测光
    # (1)测光计

    相机上一般都有"测光计"来表示当前图像的曝光水平, 一般到中间就是合适的曝光. 一般是一条线和标尺来表示. 初学者的目标是把这条线保持在中点. 但是测光计并不一定准确, 不一定需要完全按照测光计的指示, 因为很多摄影规则注定就是需要被打破的.

    # (2)测光模式

    大多数相机有三种测光模式: 点测光, 中心测光, 矩阵测光(综合评价测光).

    常见的单反都是矩阵测光(综合评价测光), 相机会从取景器的所有区域的光线中读取曝光, 并计算平均值. 在多样化照明情况下拍搜时可以使用矩阵测光.

    点测光: 相机只对一个焦点的光进行评估. 相机只通过特定点的曝光来决定是否均衡曝光. 拍搜大场景中高对比的小物体时适用点测光模式. 比如月亮, 雪地里的一只黑狗.

    中心测光: 相机读取画幅中点和周围环境的曝光值. 当主体将在取景器的中心时可以采用中心测光模式. 比如肖像.

    # 3.曝光补偿

    一般相机上有个小按钮, 有一个加号或者减号在上面, 可以通过它调节曝光补偿. 有的相机有一个独立的曝光补偿的刻度. 调节它, 可以让照片更亮或者更暗.

    光圈优先模式下, 曝光补偿会改变快门速度.

    快门优先模式下, 曝光补偿会改变光圈大小.

    程序自动模式下, 曝光补偿还是会改变快门速度.

    **在手动模式下, 曝光补偿的意义不大, 因为可以通过其他参数来调节曝光度. **

    # 4.曝光模式

    自动模式. 不知道怎么设置时用这个. 但是相机总是按照测光值进行拍照, 会导致有时候感光度较高, 然后很快的快门速度来拍照. 可能不能完美处理需要的场景.

    曝光模式通常有: PASM.

    • P: 程序模式. 相机基于选择的测光类型, 选择最佳的快门和光圈. 可以快速调整快门或光圈, 而相机会自动调节其他的参数来达到正常曝光. 它是光圈优先和快门优先的结合.
    • A: 光圈优先模式. 可以设置光圈, 然后相机会决定最佳曝光的快门速度.
    • S: 快门优先模式. 自己设置快门速度和感光度, 相机来根据测光值自动选择光圈. 当拍摄运动的物体时, 可以选择这个模式, 拍摄运动物体通常需要较快快门速度, 而可能来不及调整光圈, 此时让相机自动调节光圈即可.
    • M: 手动模式. 用户设置快门速度, 感光度, 光圈和闪光灯等全部参数. 手动模式下, 相机只会告诉你测光值, 而不会自动改任何参数. 可以在自动模式中使用曝光补偿来调整照片的亮度.
    # 5.动态范围

    动态范围定义: 照片中暗光与亮光间的比值.

    动态范围低的照片的话, 整个照片的曝光度都差不多, 没有特别突出的亮点或者暗点, 也就是说对比度较低.

    低动态范围--低对比度.

    高动态范围--高对比度.

    一般来说, 相机传感器越好, 动态范围就越高.

    用 RAW 格式拍搜可以获取更多信息, 更高的动态范围. JPEG 则不行.

    # 6.HDR摄影与多重曝光

    HDR功能: 在不同曝光下拍摄多张照片, 然后合成一张照片, 使得曝光更加均匀. 这个功能需要相机支持.

    多重曝光也是类似的做法, 只不过就是手动多拍几张照片, 用软件合成.

    用 HDR 或多重曝光时借助三脚架使得构图一致.

    # 7.曝光直方图

    多数相机会在拍照时或者拍照后显示直方图.

    直方图表是能够显示画面各部分曝光水平的图表. 图表左侧为阴影, 右侧为高光. 通过柱状的高低不同来显示照片的曝光水平. 想要均匀曝光的话, 得通过在阴影与高光之间均匀居中分布的直方图来实现.

    # 8.曝光档位

    光的"档位"是衡量进入相机的光线的一种方式.

    • 增加 1 档曝光=光线加倍
    • 降低 1 档曝光=光线减少一半

    光圈增加 1 个档位, 曝光量降低 1 个档位. 比如光圈从 F2 调到 F2.8, 等于降低 1 个曝光档位.

    # 构图

    # 1.三分构图法

    原理: 把摄像物放在取景框中部靠左或靠右, 将画面用两竖两横的线分开, 将物象放在靠近线条交叉处的位置.

    **把最大的注意力集中于照片的主体. **

    拍摄远处时, 地平线可以沿着三分线的上方或者下方的横线位置.

    # 2.负空间

    负空间构图: 在拍搜主体时, 主体非常突出, 周围或后面没有过多的干扰物(通常在画面中很小). 使用负空间可以使摄影主体看上去很渺小甚至更突出.

    负空间会把你的注意力吸引到照片的主体上, 而不受其他干扰.

    注意负空间并不意味着背景是完全空白的.

    # 3.引导线

    定义: 引导线就是画面中能够吸引观众注意力的线头. 用引导线可以吸引观众注意到图片的主要拍摄对象上.

    引导线可以把远处的景跟照片中的近景连接起来, 可以调整构图让这些线条指向某个地方.

    # 4.对称与平衡

    平衡可以指画面里视觉上平衡的物体, 也可以指画面里物体数量以及大小的平衡.

    对称是指照片的顶端和底部或左右两边看上去都很相似.

    # 5.透视与视角

    **焦距: 照相机传感器到镜头光学中心(光线聚集处)之间的距离. 以毫米为单位. **

    焦距越小, 镜头越宽.

    焦距越大, 景物显得越大, 或景深越短.

    大体上小于 35 mm 的镜头属于广角镜头, 大于 70mm 的镜头可以被视为远摄镜头.

    拍摄广阔的区域和风景, 广角镜头是不错的选择; 拍搜细节或者特写用远摄镜头就很好.

    透视基本上就是近处物体比远处物体显得更大. 使用广角镜头, 近物会显得更大一些, 看起来近物与背景隔得很远, 这个效果称为夸大透视.

    远摄镜头可以压缩透视, 让物体看起来挨得更近.

    下面是用广角和远摄镜头分别拍的两个构图类似的照片. 左图是站的远一些用长焦镜头拍搜的, 右图是站的近一些用广角镜头拍搜的. 可以看到它们的背景是不同的. 用长焦镜头远摄的照片中空间被压缩, 背景中的盆栽看起来比右边的大很多. 因为使用广角拍摄, 所有东西看起来都很小, 看起来会离人物更远一些, 所以看起来也更小一些.

    记住: 不要用广角怼很近拍人像, 照出来人可能会变形.

    视角是指: 从哪里拍照. 如从下往上拍, 贴近地面拍, 从楼顶往下拍等. 变换一下视角, 别总是将相机举到齐眼处拍摄.

    通常从上往下拍人像会好看一些, 可以让人物稍微低一点.

    # 6.背景

    选择一个好的背景非常重要.

    如果你的照片的主体是重点的时候, 需要强调主体的时候, 需要尽量消除背景中的干扰因素.

    • 需要看背景中是否有抢镜的光点或者亮色, 需要去掉这些元素.
    • 需要检查有没有线条从主体延伸出来. 比如人物后面的电线, 树枝等.
    • 关注地平线, 不要把地平线安排在人物头部的位置.

    尝试用空间和焦距区分主体和背景, 可以直接让主体远离背景, 或者调低光圈值, 形成一个虚化的背景. 虚化背景可以更加强调拍摄的主体, 减少背景的干扰.

    # 7.用层级创造景深

    为拍摄更多动态照片, 其中一个最有用的技巧就是在图像中增添"深度" . 为了创造"深度", 你需要在目标物前后设置多层空间.

    通常情况下, 一张照片需要三层. 图像第一层是前景, 通常聚焦; 然后是背景, 位于目标物的后面. 不是说只能有三层, 很多时候, 只会有目标物和背景; 有的时候, 只有浅浅一层景深来创造一定的"深度". 又或者你可以运用多层前景和背景. 无论如何, 都可以通过增添目标物与周围景观的空间来增添层级.

    背景和前景通过增添目标物周围有趣的材质与色彩来充实画面. 这些层级依据你的镜头与设置, 能够柔化并散焦, 或者使部分目标物周围 的景观更清晰. 照片层级最重要的一点就是确保运用围绕目标物的其他层级并使注意力集中于此. 你需要通过某种方式运用其他层级来增添图像的故事性. 有时, 这意味着在目标物前放置如铁栅栏这类的物体, 或者使自己或物体与背景分离.

    为图像增添"深度", 可使图像看起来更动态, 更专业. 但别忘了同时关注背景与前景, 目标物于空间的最佳位置, 以及为拍出优质照片而使层级间共同协作的角度.

    # 对焦和景深

    # 1.景深和光圈

    景深就是图片中具有可接受的清晰度或焦点的区域.

    景深较浅的照片呈现出的空间较小, 这意味着图片里面只聚焦了局部的景色, 也就是可能很多地方都是模糊的.

    呈现大部分的空间的是深景深, 这意味着图片在更大范围内聚焦, 大部分的区域是清晰的.

    有三种东西影响景深: 光圈, 拍摄物的距离, 镜头的焦距.

    • **光圈越大(数值越小), 聚焦区域越小, 景深越浅; 光圈越小(数值越大), 聚焦范围越大, 则景深越深. **
    • **物体离相机越近, 景深越浅. **
    • **焦距越大, 放大倍数越大, 景深越浅. **

    # 2.虚化的浅景深背景

    三种虚化背景的方法.

    • 降低光圈值, 调大光圈. 更大的光圈孔径, 能得到更浅的景深.
    • 远离背景.
    • 靠近拍摄物, 也可以使用远摄镜头.
    # 3.对焦与景深

    与调整景深相关的概念是对焦. 也就是调整相机镜头, 聚焦于画面中的重点.

    在多数情况下建议使用自动对焦(AF: Auto Focus).

    光线昏暗的环境可以使用手动对焦(MF: Manual Focus).

    **当拍摄人物时, 一般要聚焦于眼睛. **

    **如果不好聚焦, 或者聚焦效果不满意, 可以尝试调小光圈, 以获得更深的景深, 从而更容易对焦清晰. **

    拍摄远处的景物时, 可以调小光圈, 使得景深更深, 对焦更清晰.

    # 4.对焦点和自动对焦模式

    对焦点是相机取景器中的小点, 相机用它选择要对焦的对象. 一般来说, 对焦点越多, 对焦过程就越快速方便, 专业相机甚至能达到一百多个. 一般取景框中有很多个小方块, 就是指的对焦点, 可以切换对焦点, 对焦到不同的地方.

    一般办按快门时, 相机开始自动对焦.

    连续自动对焦可以用于拍摄移动的物体, 比如奔跑的小孩, 行驶的汽车等.

    # 5.如何清晰的拍摄运动物体

    一般来说, 设置快门优先模式, 然后设置较快的快门速度.

    如果想让运动者的背景模糊, 可以调低快门速度, 然后跟着运动者一起运动, 这样拍出来人物不会糊, 而背景会模糊, 是创意的做法.

    # 相机结构与设置

    # 1.文件类型

    一般支持 RAW 和 JPG 两种格式.

    RAW 就是指未压缩, 全品质的图像(大文件). 通常 RAW 的图片在相机里看起来色彩比较单一, 对比度低, 需要进行后期处理. 尼康的 RAW 图片格式为 .NEF.

    JPG 是压缩裁剪后的图片, 更加通用.

    推荐用 RAW 格式拍摄, 但一般都是两种一起保存.

    # 2.感光元件

    指相机感受与记录光的传感器.

    按照感光元件大小, 相机分为全画幅相机: 准专业相机领域. 半画幅相机(APS-C画幅): 比全画幅尺寸小. 多见于小型数码单反相机和无反相机. 分辨率较低.

    感光元件尺寸和焦距也有关系. 尺寸更小的感光元件"裁剪"画幅(拉近画面效果), 也就是会看起来更近.

    尺寸大的感光元件通常在低光环境下表现更好, 也更容易得到好的景深效果.

    始终盖着相机盖除非有镜头, 不要把感光元件暴露在空气中. 最好不要自己清洁, 找专业人士清洁.

    # 3.照片的分辨率

    分辨率即相机能捕捉到的信息量(像素), 是传感器中水平像素和垂直像素的乘积.

    像素越高, 照片质量, 细节越高.

    # 4.高速连拍模式

    一次性连续拍摄多张照片, 通常以每秒的帧数计算, 或以 FPS 为单位.

    拍摄活动或团体照片时, 使用连拍模式能保证拍到好的照片, 防止人闭眼或失误.

    连拍模式需要使用高速的存储卡, 不然可能无法支持.

    # 5.单反相机与无反相机

    单反相机(DSLR) 指的是数码单镜反光相机. 基本上, 单反相机里的一面镜子移开后, 才能将光线投射到感光元件. 也就是反光镜开合将感光元件暴露与光线中. 取下镜头后, 看到的是反光镜. 平时没有拍摄时, 取景器中的图像是从反光镜中看到的, 拍摄时, 会快速移开反光镜, 让光线照射到感光元件进行拍照. 单反的取景器能让你实时的看到一切事物.

    无反相机就是无反光镜的相机. 取下镜头后直接就能看到感光元件. 无反相机一般更小. 由于感光元件一直打开, 所以通常耗电更快. 无反相机比同类单反相机贵.

    # 白平衡与色彩

    # 1.色温与白平衡

    白平衡是相机将白色调适成基准白色的过程, 这样其他颜色可以以此为基准.

    大部分相机使用自动白平衡就很不错.

    调节白平衡可以调整照片的冷或暖色调.

    # 2.开尔文温度计

    开尔文温度图表如下:

    小数值的颜色更暖一些, 颜色偏橘. 色温越高更接近于蓝色.

    # 3.白平衡自动与手动模式

    多数相机都有预设白平衡模式, 能够匹配最常见的光源.

    白平衡一般是 "WB" 键. AWB 也即是自动白平衡.

    可以设置为 "多云" 模式或者 "阴影" 模式(直接日光较少, 整体更暖色调)等, 当然也有其他各种模式, 适应不同的场景.
