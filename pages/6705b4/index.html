    # 安装

    遇到 Node 高版本导致的 SSL 问题:

    参考: https://blog.csdn.net/itopit/article/details/127280592

    node build时高版本的 SSL 问题, 执行:

    NODE_OPTIONS=--openssl-legacy-provider
    1

    Windows 的命令:

    $env:NODE_OPTIONS="--openssl-legacy-provider"
    1

    # vue-press-vdoing

    这个主题不错, 开箱即用.

    这里面遇到过编译 OOM 的情况, 可以如下设置 package.json 中 max_old_space_size=10240​:

    "scripts": {
    "dev": "node --max_old_space_size=10240 ./node_modules/vuepress/cli.js dev docs",
    "build": "node --max_old_space_size=10240 ./node_modules/vuepress/cli.js build docs",
    "deploy": "bash deploy.sh",
    "editFm": "node utils/editFrontmatter.js",
    "baiduPush": "node utils/baiduPush.js https://xugaoyi.com && bash baiduPush.sh",
    "publish": "cd ./vdoing && npm publish && cd .. && yarn updateTheme",
    "updateTheme": "yarn remove vuepress-theme-vdoing && rm -rf node_modules && yarn && yarn add vuepress-theme-vdoing -D"
}
