    # 541-反转字符串 II🌸
    # 题目

    给定一个字符串 s 和一个整数 k, 从字符串开头算起, 每计数至 2k 个字符, 就反转这 2k 个字符中的前 k 个字符.

    如果剩余字符少于 k 个, 则将剩余字符全部反转. 如果剩余字符小于 2k 但大于或等于 k 个, 则反转前 k 个字符, 其余字符保持原样.

    示例:

    输入: s = "abcdefg", k = 2 输出: "bacdfeg"

    # 题解

    在遍历字符串的过程中, 让 i += (2 * k), i 每次移动 2 * k 就可以了, 然后判断是否需要有反转的区间. 因为要找的也就是每 2 * k 区间的起点, 这样写, 程序会高效很多.

    所以当需要固定规律一段一段去处理字符串的时候, 要想想在在 for 循环的表达式上做做文章.

    代码如下:

    public String reverseStr(String s, int k) {
    char[] arr = s.toCharArray();

    // 每次i增加2k, 并翻转2k这个区间的字符
    for (int i = 0; i < arr.length; i += 2 * k) {
        // 起点
        int left = i;
        // 终点
        int right = Math.min(i + k - 1, arr.length - 1);
        // 反转区间的字符
        while (left < right) {
            swap(arr, left++, right--);
        }
    }
    return new String(arr);
}

private void swap(char[] s, int i, int j) {
    char tmp = s[i];
    s[i] = s[j];
    s[j] = tmp;
}

    # 151-翻转字符串里的单词🌸
    # 题目

    给定一个字符串, 逐个翻转字符串中的每个单词.

    示例 1: 输入: "the sky is blue" 输出: "blue is sky the"

    示例 2: 输入: " hello world! " 输出: "world! hello" 解释: 输入字符串可以在前面或者后面包含多余的空格, 但是反转后的字符不能包括.

    示例 3: 输入: "a good example" 输出: "example good a" 解释: 如果两个单词间有多余的空格, 将反转后单词间的空格减少到只含一个.

    https://leetcode.cn/problems/reverse-words-in-a-string/

    # 从后往前遍历法

    使用 StringBuilder, 从后往前找, 找到一个单词就添加到 StringBuilder 中, 然后持续过滤中间的空格.

    public String reverseWords(String s) {
    // 移除两端的空格
    s = s.trim();
    // 从后往前遍历找到所有的单词
    int j = s.length() - 1, i = j;
    StringBuilder res = new StringBuilder();
    while (i >= 0) {
        // 寻找一个单词
        while (i >= 0 && s.charAt(i) != ' ') {
            i--;
        }
        // 添加找到的单词&加空格
        res.append(s, i + 1, j + 1).append(" ");
  
        // 寻找下一个单词的开头 注意这里有可能存在多个空格 所以用while
        while (i >= 0 && s.charAt(i) == ' ') {
            i--;
        }
        // 重置单词的末尾
        j = i;
    }
  
    return res.toString().trim();
}

    public void reverseString(char[] nums) {
    if (nums == null || nums.length < 2) {
        return;
    }
  
    int left = 0;
    int right = nums.length - 1;

    while (left < right) {
        swap(nums, left, right);
        left++;
        right--;
    }
}

private void swap(char[] nums, int i, int j) {
    char temp = nums[i];
    nums[i] = nums[j];
    nums[j] = temp;

    示例 1:

    输入: "A man, a plan, a canal: Panama"
输出: true
    1

    示例 2:

    输入: "race a car"
输出: false

    2

    https://leetcode-cn.com/problems/valid-palindrome/ (opens new window)

    # 双指针法

    直接在原字符串 s 上使用双指针. 在移动任意一个指针时, 需要不断地向另一指针的方向移动, 直到遇到一个字母或数字字符, 或者两指针重合为止. 也就是说每次将指针移到下一个字母字符或数字字符, 再判断这两个指针指向的字符是否相同.

    public boolean isPalindrome(String s) {
    if (s == null) {
        return false;
    }
  
    int left = 0, right = s.length() - 1;
    while (left < right) {
        // 跳过非字母和数字字符
        while (left < right && !Character.isLetterOrDigit(s.charAt(left))) {
            left++;
        }
        while (left < right && !Character.isLetterOrDigit(s.charAt(right))) {
            right--;
        }
        // 如果不相等则返回false
        if (Character.toLowerCase(s.charAt(left)) != Character.toLowerCase(s.charAt(right))) {
            return false;
        }
        left++;
        right--;
    }
  
    return true;
}

    • 所有英文字母(小写或大写)位置反转.

    示例 1:

    输入: s = "ab-cd"

    输入: s = "a-bC-dEf-ghIj"
    输出: "j-Ih-gfE-dCba"

    输入: s = "Test1ng-Leet=code-Q!"
    输出: "Qedo1ct-eeLg=ntse-T!"

    提示

    • 1 <= s.length <= 100
    • s 仅由 ASCII 值在范围 [33, 122] 的字符组成
    • s 不含 '"' 或 '\'

    https://leetcode.cn/problems/reverse-only-letters/description/

    # 双指针法

    直接使用左右指针, 分别找到字母之后进行交互即可. 与上一题基本一样的写法.

    public String reverseOnlyLetters(String s) {
    char[] chars = s.toCharArray();
    int left = 0, right = chars.length - 1;
    while (left < right) {
        // 找到左边的字母
        while (left < right && !Character.isLetter(chars[left])) {
            left++;
        }
        // 找到右边的字母
        while (left < right && !Character.isLetter(chars[right])) {
            right--;
        }
        // 交换
        swap(chars, left, right);
        // 移动指针
        left++;
        right--;
    }
    return new String(chars);
}

private void swap(char[] chars, int left, int right) {
    char tmp = chars[left];
    chars[left] = chars[right];
    chars[right] = tmp;
}

    # 辅助栈法

    将 s 中的所有字母单独存入栈中, 所以出栈等价于对字母反序操作. (或者可以用数组存储字母并反序数组.)

    然后, 遍历 s 的所有字符, 如果是字母就选择栈顶元素输出.

    public String reverseOnlyLetters(String str) {
    Stack<Character> letters = new Stack();
    // 将全部字母压栈
    for (char c: str.toCharArray()) {
        if (Character.isLetter(c)) {
            letters.push(c);
        }
    }
    // 构造结果
    StringBuilder ans = new StringBuilder();
    for (char c: str.toCharArray()) {
        // 如果当前是字母就从栈中弹出一个字母进行替换
        if (Character.isLetter(c)) {
            ans.append(letters.pop());
        } else {
            ans.append(c);
        }
    }
    return ans.toString();
}

    # 题目

    给你一个字符串 s, 字符串的「能量」定义为: 只包含一种字符的最长非空子字符串的长度. 请你返回字符串的能量.

    1
    2
    3
    4

    https://leetcode-cn.com/problems/consecutive-characters (opens new window)

    # 直接遍历法

    直接从左到右变遍历记录各个连续字符出现的次数, 遇到不同的字符就更新计数器.

    public int maxPower(String str) {
    if (str == null) {
        return 0;
    }
    // 至少有1个字符
    int res = 1;
    // 字符计数器
    int counter = 1;
    // 记录当前的字符
    char temp = str.charAt(0);
    for (int i = 1; i < str.length(); i++) {
        // 如果当前的字符与上一个相同
        if (str.charAt(i) == temp) {
            counter++;
        } else {
            // 不同就更新结果并重置计数器
            res = Math.max(res, counter);
            counter = 1;
            temp = str.charAt(i);
        }
    }
    // 最后一定要更新结果, 防止出现aaaaa这种记录
    res = Math.max(res, counter);
    return res;
}

    # 1408-数组中的字符串匹配(简单)
    # 题目

    给你一个字符串数组 words, 数组中的每个字符串都可以看作是一个单词. 请你按 任意 顺序返回 words 中是其他单词的子字符串的所有单词.

    输入: words = ["mass","as","hero","superhero"]
输出: ["as","hero"]
解释: "as" 是 "mass" 的子字符串, "hero" 是 "superhero" 的子字符串. ["hero","as"] 也是有效的答案.
    1
    2
    3

    https://leetcode-cn.com/problems/string-matching-in-an-array (opens new window)

    # 解题思路

    用 StringBuffer 把所有的单词都拼接起来, 用 ',' 隔开, 当在前面找 indexOf 得到的索引和从后面找 lastIndexOf 得到的索引假如是一致时, 意味着这个单词是独一无二的, 在其他单词也不包含, 假如不等就是在其他单词中也包含.

    public List<String> stringMatching(String[] words) {
    List<String> resList = new ArrayList<String>();
    StringBuffer buffer = new StringBuffer();
    // 将字符串全部拼接
    for (int i = 0; i < words.length; i++) {
        buffer.append(words[i]);
        buffer.append(",");
    }
    // 寻找某个单词的前后出现索引, 不相同则说明包含子串
    for (int i = 0; i < words.length; i++) {
        if (buffer.indexOf(words[i]) != buffer.lastIndexOf(words[i])) {
            resList.add(words[i]);
        }
    }
    return resList;
}

    # 409-构造最长回文串(简单)
    # 题目

    给定一个包含大写字母和小写字母的字符串, 找到通过这些字母构造成的最长的回文串. 在构造过程中, 请注意区分大小写. 比如"Aa"​不能当做一个回文字符串.

    回文串: 是一个正读和反读都一样的字符串, 比如“level”或者“noon”等等就是回文串.

    输入: "abccccdd" 输出: 7 解释:可以构造的最长的回文串是"dccaccd", 它的长度是 7.
    1
    # 次数统计法

    现在考虑一下可以构成回文串的两种情况:

    • 字符出现次数为双数的组合
    • 字符出现次数为偶数的组合+单个字符中出现次数最多且为奇数次的字符

    统计字符出现的次数即可, 双数才能构成回文. 因为允许中间一个数单独出现, 比如 “abcba”, 所以如果最后有字母落单, 总长度可以加 1. 首先将字符串转变为字符数组. 然后遍历该数组, 判断对应字符是否在 hashset 中, 如果不在就加进去, 如果在就让 count++, 然后移除该字符! 这样就能找到出现次数为双数的字符个数.

    public int longestPalindrome(String s) {
    if (s.isEmpty()) {
        return 0;
    }
    // 用于存放出现过的字符
    HashSet<Character> set = new HashSet<>();
    char[] chars = s.toCharArray();
    int count = 0;
  
    for (int i = 0; i < chars.length; i++) {
        // 如果hashset没有该字符就保存进去
        if (!set.contains(chars[i])) {
            set.add(chars[i]);
            // 如果有,就让count+2(说明找到了一个成对的字符), 然后把该字符移除
        } else {
            set.remove(chars[i]);
            count += 2;
        }
    }
  
    // 如果set为空,说明所有字符都是成对的, 直接返回count*2
    // 如果set不为空,说明至少有一个字符是单个的, 那么最后结果就是count*2+1
    return set.isEmpty() ? count : count + 1;
}

    https://leetcode-cn.com/problems/longest-palindromic-substring (opens new window)

    # 中心扩展算法

    面试记住这个是好理解的. 回文串一定是对称的, 所以可以每次循环选择一个中心, 先进行左右扩展, 找到与当前字符相同的字符作为窗口, 然后逐一向两边扩散, 判断左右字符是否相等即可.

    编辑图: https://www.processon.com/diagraming/66233b98fc41004d85095f4a (opens new window)

    这里理解这个窗口很重要, 对于一个字符串来说, 只要是相同的字符, 无论是奇数还是偶数个, 都是回文串, 所以这个就可以构成一个窗口.

    public String longestPalindrome(String str) {

    if (str == null || str.isEmpty()) return "";
    int strLen = str.length();

    // 记录最大长度回文串的起始位置和长度
    int maxStart = 0;
    int maxLen = 0;

    // 从左到右以每个元素为中心判断
    for (int center = 0; center < strLen; center++) {
        int curLen = 1;
        int left = center - 1;
        int right = center + 1;
        // 先进行左右扩展, 找到与当前字符相同的字符作为窗口, 然后逐一向两边扩散
        // 往左扩散-找到左边与当前中心字符相同的字符
        while (left >= 0 && str.charAt(left) <mark> str.charAt(center)) {
            curLen++;
            left--;
        }
        // 右边扩散-找到右边与当前中心字符相同的字符
        while (right < strLen && str.charAt(right) </mark> str.charAt(center)) {
            curLen++;
            right++;
        }

        // 以当前center为重复串的中心扩散
        while (left >= 0 && right < strLen && str.charAt(right) == str.charAt(left)) {
            // 每次长度+2
            curLen = curLen + 2;
            left--;
            right++;
        }
        // 如果发现最大长度就更新
        if (curLen > maxLen) {
            maxLen = curLen;
            maxStart = left;
        }
    }
    return str.substring(maxStart + 1, maxStart + maxLen + 1);
}

    # 动态规划

    优化: 中心扩散的方法, 其实做了很多重复计算. 动态规划就是为了减少重复计算的问题. 动态规划听起来很高大上. 其实说白了就是空间换时间, 将计算结果暂存起来, 避免重复计算.

    用一个 boolean dp[l][r] 表示字符串从 i 到 j 这段是否为回文. 试想如果 dp[l][r]=true, 要判断 dp[l-1][r+1] 是否为回文. 只需要判断字符串在 (l-1) 和 (r+1) 两个位置是否为相同的字符, 就减少了很多重复计算.
    进入正题, 动态规划关键是找到初始状态和状态转移方程.

    • 初始状态, l = r 时, 此时 dp[l][r]=true.
    • 状态转移方程, dp[l][r]=true 并且 (l-1) 和(r+1) 两个位置为相同的字符, 此时 dp[l-1][r+1]=true.
    public String longestPalindrome3(String str) {
    if (str == null || str.length() < 2) return str;
    // 字符串长度
    int strLen = str.length();
    int maxStart = 0;  // 最长回文串的起点
    int maxEnd = 0;    // 最长回文串的终点
    int maxLen = 1;    // 最长回文串的长度
    // dp数组
    boolean[][] dp = new boolean[strLen][strLen];

    for (int right = 1; right < strLen; right++) {
        for (int left = 0; left < right; left++) {
            // 状态转移方程
            if (str.charAt(left) == str.charAt(right) && (right - left <= 2 || dp[left + 1][right - 1])) {
                dp[left][right] = true;
                // 更新结果
                if (right - left + 1 > maxLen) {
                    maxLen = right - left + 1;
                    maxStart = left;
                    maxEnd = right;
                }
            }
        }
    }
    // 根据索引找到最长回文字符串
    return str.substring(maxStart, maxEnd + 1);
}

    # 中心扩展法

    由中心往两边扩散. 这里分为两种情况:

    (1) 以一个单独的元素作为中心往两边扩散, 能扩散则找到一个回文串, 同时这个元素自己也是回文串.

    (2) 以两个相同的相邻元素作为中心往两边扩散, 能扩散则找到一个回文串, 如果相邻的元素不相同, 则直接忽略, 否则这两个元素自己也构成回文串.

    代码如下:

    public int countSubstrings(String str) {
    if (str == null || str.isEmpty()) return 0;
    int strLen = str.length();

    // 记录最大长度回文串的起始位置和长度
    int res = 0;

    // 以一个元素为中心进行扩散
    for (int center = 0; center < strLen; center++) {
        // 当前元素也算一个回文串
        res++;
        int left = center - 1;
        int right = center + 1;
        // 以当前center为重复串的中心扩散, 能够扩散则说明找到一个回文串
        while (left >= 0 && right < strLen && str.charAt(right) == str.charAt(left)) {
            left--;
            right++;
            res++;
        }
    }

    // 以两个相邻元素为中心进行扩散
    for (int center = 0; center < strLen - 1; center++) {
        // 如果两个相邻元素不相等则不可能是回文串
        if (str.charAt(center) != str.charAt(center + 1)) {
            continue;
        }
    
        // 以两个相邻元素为中心往两边扩散 能扩散则找到一个回文串
        int left = center;
        int right = center + 1;
        while (left >= 0 && right < strLen && str.charAt(right) == str.charAt(left)) {
            left--;
            right++;
            res++;
        }
    }

    return res;
}

        counter++;
        start--;

    马拉车算法可以在线性时间内找出以任何位置为中心的最大回文串.

    # 动态规划

    状态: dp[i][j] 表示字符串 s 在 [i, j] 区间的子串是否是一个回文串.

    状态转移方程: 当 s[i] == s[j] && (j - i < 2 || dp[i + 1][j - 1])​ 时, dp[i][j] = true, 否则为 false.

    • 当只有一个字符时, 比如 a 自然是一个回文串.
    • 当有两个字符时, 如果是相等的, 比如 aa, 也是一个回文串.
    • 当有三个及以上字符时, 比如 ababa 这个字符记作串 1, 把两边的 a 去掉, 也就是 bab 记作串 2, 可以看出只要串 2 是一个回文串, 那么左右各多了一个 a 的串 1 必定也是回文串. 所以当 s[i]==s[j] 时, 自然要看 dp[i+1][j-1]​ 是不是一个回文串.
    public int countSubstrings2(String str) {
    // 动态规划法
    boolean[][] dp = new boolean[str.length()][str.length()];
    int ans = 0;
    for (int j = 0; j < str.length(); j++) {
        for (int i = 0; i <= j; i++) {
            if (str.charAt(i) == str.charAt(j) && (j - i < 2 || dp[i + 1][j - 1])) {
                dp[i][j] = true;
                ans++;
            }
        }
    }
    return ans;
}

    # 辗转相除法

    使用辗转相除法求最大公约数.

    public String gcdOfStrings(String str1, String str2) {
    // 假设str1是N个x, str2是M个x, 那么str1+str2肯定是等于str2+str1的. 
    if (!(str1 + str2).equals(str2 + str1)) {
        return "";
    }
    // 辗转相除法求最大公约数
    return str1.substring(0, gcd(str1.length(), str2.length()));
}

private int gcd(int a, int b) {
    if (b == 0) {
        return a;
    } else {
        return gcd(b, a % b);
    }
}

    # 解题思路

    定义两个变量, 对数字字符和字母字符计数, 如果数量相差 2 及以上, 直接不算了. 为了节省空间计数的两个变量拿来复用, 哪个数大, 哪个从零开始, 每次加 2, 自己走自己的, 互不干涉. 反正数字字符和字母字符最终都要放进去. 通过双指针将字符放入到字符数组中, 返回字符串.

    public String reformat(String str) {
    int letterCnt = 0, numberCnt = 0;
    char[] chars = str.toCharArray();
    // 统计数字与字母的个数
    for (char c : chars) {
        int a = Character.isDigit(c) ? numberCnt++ : letterCnt++;
    }
    // 两种数量相差超过1直接返回
    if (Math.abs(letterCnt - numberCnt) > 1) {
        return "";
    }

    // 哪个数量多则放到第一个位置
    int letterIndex = 0, numberIndex = 0;
    if (letterCnt > numberCnt) {
        numberIndex = 1;
    } else {
        letterIndex = 1;
    }

    // 遍历字符串重新填充
    for (char c : str.toCharArray()) {
        if (Character.isLetter(c)) {
            chars[letterIndex] = c;
            letterIndex += 2;
        } else {
            chars[numberIndex] = c;
            numberIndex += 2;
        }
    }

    return new String(chars);
}
