    # 明朝那些事

    总结

    • 作者：当年明月
    • 阅读时间：2022年7月
    • 评分：9.5分

    要是每个朝代都有这样的作品就好了。

    朱元璋-洪武

    • 元朝末年, 官逼民反, 朱元璋造反, 当了皇帝.
    • 与陈友谅大战.
    • 为了保护子孙皇位, 朱元璋杀了很多开国功臣.

    朱允炆-建文

    • 担心藩王造反, 开始削藩. 燕王朱棣不得不反.
    • 一直选用李景隆对抗朱棣, 奈何李景隆是个庸才, 耽误战机.
    • 朱棣攻入京城后烧掉皇宫失踪.

    朱棣-永乐

    • 朱元璋第四个儿子.
    • 朱棣军师姚广孝. 以靖难的名义造反.
    • 命令谢晋修书《永乐大典》, 谢晋号称永乐第一大才子.
    • 迁都北京. 原因: 自己是燕王一直住北京+住北方压制蒙古部落.
    • 设立内阁, 东厂, 巡抚三大机构.
    • 继承人之争: 朱高炽与朱高煦为了争夺太子之位而进行了权益斗争.
    • 当政时内阁的重要人物-"三杨": 杨士奇, 杨荣, 杨溥.
    • 评价: 是个好皇帝, 但不是个好人.

    朱高炽

    • 仁厚. 敢于废除朱棣的不合理政令.
    • 在位仅十个月就病逝.

    朱瞻基

    • 在位十年. 与朱高炽统治加起来开创盛世, 世称仁宣之治.
    • 刚登基朱高炽造反. 朱瞻基御驾亲征平反. 朱高炽落败. 做事不做绝, 彻头彻尾的丑角.
    • 好皇帝. 没啥不良嗜好. 只是喜欢斗蛐蛐.
    • 年仅38岁. 为自己5岁的儿子选了五位顾命大臣.

    朱祁镇-明英宗

    • 算是开明的皇帝.
    • 大太监王振崛起.
    • 于谦崭露头角.
    • 也先出兵攻打明朝. 王振说服朱祁镇御驾亲征. 但是王振过于垃圾. 被也先打败, 几乎全军覆没, 史称土木堡之战. 朱祁镇被也先俘虏.

    朱祁钰

    • 朱祁镇被抓之后, 临时选出的皇帝. 朱祁镇成为太上皇.
    • 也先带兵进攻北京. 于谦率众将士保卫了京城, 史称”京城保卫战“.
    • 也先同意送回朱祁镇, 但朱祁钰不想朱祁镇回国. 所以派了使臣也不打算接朱祁镇回国.
    • 后来派去的使臣杨善是接朱祁镇回国的重要功臣. 杨善能言善辩, 说服了也先派人将朱祁镇送回明朝.
    • 但是朱祁钰将朱祁镇囚禁起来, 共7年.
    • 朱祁钰因病身故.

    朱祁镇

    • 阴谋集团”还乡团“(徐有贞, 曹吉祥, 石亨等人)策划并实施”夺门之变“, 帮助朱祁镇复辟重新当上皇帝.
    • 徐有贞为报复于谦, 公报私仇, 利用朱祁镇害死了于谦.
    • ”还乡团“内部成员互相算计, 都倒台了.

    朱见深-成化

    • 在位 23 年.
    • 由于其父朱祁镇被也先俘虏, 回国后被朱祁钰囚禁, 朱见深从小由比他大17岁的万宫女带大.
    • 为于谦平反.
    • 万宫女成为万贵妃. 万贵妃设计除掉了所有怀孕妃子的孩子.
    • 朱见深临幸了一个宫女, 生下了朱佑樘. 万贵妃命令除掉朱佑樘, 但是手下却放过了朱佑樘. 朱佑樘在宫里吃百家饭长大, 直到五岁才与朱见深相认.
    • 在位期间, 不理朝政, 但也不昏庸. 评价: 懦弱, 对好人坏人都强硬不起来.
    • 万贵妃是朱见深的精神支柱, 万贵妃病逝后不久, 朱见深也去世.

    朱佑樘

    • 为人宽厚仁慈, 躬行节俭, 不近女色, 勤于政事, 重视司法, 大开言路, 努力扭转朝政腐败状况, 驱逐奸佞, 勤于政事, 励精图治, 任用王恕, 刘大夏等为人正直的大臣, 史称“弘治中兴”.
    • 从小经历了很多磨难, 励志当好皇帝.
    • 内阁成员: 李东阳, 谢迁等.
    • 评价: 好人+好皇帝.

    朱厚照

    • 明朝最会玩的皇帝.
    • 打太极刘瑾猖獗一时, 六部九卿集体上书要求处死刘瑾.
    • 刘瑾善于抓住朱厚照爱玩的特点, 非但没有被处理反而更强势. 六部九卿很多都被报复. 许多官员上书骂刘瑾, 其中包含王守仁. 王守仁被打, 并贬官到贵州陇场.
    • 刘瑾派人路上暗杀王守仁. 王守仁逃脱后见了父亲之后, 决定去贵州赴任.
    • 王守仁父亲王华, 是状元出身.
    • 王阳明一直学习程朱理学. 参悟格物致知的道理. 后陇场悟道, 领悟知行合一的智慧. 体会到”去人欲望“是不对的, 人欲永远存在.
    • 杨一清为太监张永出谋划策, 成功打倒刘瑾.
    • 唐寅唐伯虎是同期的大才子. 科举考试中陷入”作弊事件“被处罚永远不允许再参与考试.
    • 宁王朱宸濠叛乱. 朱宸濠买通朝廷内部许多官员, 秘密准备造反.
    • 王守仁时任江西巡抚, 决定平反. 但是王守仁开始没有兵, 于是通过自己的计谋, 抓住朱宸濠内心的弱点, 最后平反.
    • 朱厚照以平宁王之乱为借口, 借机代表下江南. 途中四处游玩, 所到之处民不聊生. 南巡图中游玩落水后生病, 后驾崩.

    朱厚熜-嘉靖

    • 朱厚照没有儿子, 杨廷和安排朱厚熜当了皇帝.
    • 朱厚熜由于是藩王当皇帝, 杨廷和主张朱厚熜认朱厚照为父, 但是朱厚熜不同意, 主张认自己的父母. 双方就此展开博弈, 史称”大礼议事件“. 这实际是权利的博弈. 最后杨廷和输了, 退出历史舞台.
    • 嘉靖喜欢修道.
    • 大奸臣严嵩崛起, 依靠其奸诈狡猾当上了内阁首府.
    • 徐阶崛起, 入阁.
    • 嘉靖年间, 倭寇猖獗. 两大头领为汪直和徐海. 胡宗宪奉命打击倭寇, 徐文长(徐渭)受胡宗宪赏识, 作为打击倭寇的军师.
    • 戚继光辅佐胡宗宪抗倭. 戚继光会带兵, 组织了戚家军. 后成功抗倭.
    • 徐阶隐忍了十几年, 最终扳倒了严嵩. 严嵩的第一智囊是其子严世蕃, 十分聪明狡诈. 徐阶也设计打败严世蕃. 徐阶后成为首辅. 徐阶任用了很多优秀的人才, 如张居正.
    • 明代第一清官海瑞为人十分清廉, 还敢写奏书直接骂嘉靖, 后直接入狱, 嘉靖死后被徐阶放出.
    • 嘉靖后期徐阶成为实际的掌权人, 权倾天下. 徐阶任命张居正, 高拱为裕王(后为隆庆)的老师.

    # 软技能-代码之外的生存指南

    总结

    • 作者: 约翰 * 森梅兹
    • 阅读时间: 2023年3月
    • 个人评分: 9.0分

    评语: 一本非技术书籍, 其实不只是给程序员看的, 各种行业都很适用. 部分章节并不是很需要看.

    读书笔记

    • 如果你想成为一名出色的软件开发人员, 就得学会高效地与他人相处(即使写代码是你工作当中最为享受的一部分). 作为一个软件开发人员, 你的工作就是与人打交道(其实几乎所有的职业都是这样).

    • 专业化很重要. 有大量的软件开发人员并没有具体的专业方向. 更细分的专业领域常常比全能更受欢迎.

    • 承担责任. 在任何公司里能让你脱颖而出的最重要法宝就是承担更多的责任. 金钱总是追随着责任. 有任何机会去承担更多责任时, 承担起来!

    • 引人注目. 如果你一直默默无闻, 你的成就不为人知的话, 即使你是团队中最聪明, 最努力, 最出色的开发人员, 那也一文不名. 如果找不到方法让你的老板或高层管理人员知道你在做什么, 那你的所有努力都是徒劳的. 技术上也是, 如果没有宣传自己, 自己再牛逼别人也不会知道.

    • 自学. 不要只学软件开发. 如果你把目标设定为更高级别的岗位甚至是行政岗位, 你还需要学习领导力, 管理和商科的相关知识. 千万不要忘记分享自己学到的东西, 这是实现"引人注目"的一种方式.

    • 追求品质, 完善自我. 专业人士是通过持续不断的自我完善达到自己所追求的高品质的. 如果你也想成为专业人士, 就要致力于持续不断地完善自己的技能, 学习更多与专业相关的匠艺, 确保自己制订并执行了学习计划, 拓展自己的技能, 学习新东西, 这将有助于你做得更好. 不要沾沾自喜, 永远对努力成为更好的自己心存渴望.

    • 请勿陷入对技术的狂热之中. 如果你能让自己不成为某种技术的信徒, 你会在职业生涯之路上走得更远.

    • 伟大的音乐家和超级巨星之间的真正区别无非就是营销而已. 程序员也需要自我营销. 你也许是世界上最有天赋的软件开发人员, 但是如果没有人知道你的存在, 你也只不过是浮云一片. 营销方式: 演讲, 博客等.

    • 作为一名软件开发人员, 可以用来推销自己的最佳媒介之一就是博客. 我坚信每一个在乎自己职业生涯的软件开发人员都应该投资创建一个博客. 博客是推销自己的一种既廉价又简单的方式, 对于让自己声名鹊起极具价值. 成功的博客每天能够吸引数百甚至数千的访客, 这可以给你带来很多机会, 从工作岗位到咨询项目, 甚至是产品销售的目标客户. 打造成功博客的最大秘诀有且仅有一个——持之以恒. 我和许多成功的博主都谈论过, 他们都有一个共同点——写了很多博客. 我认识的一些最成功的博主每天都要写博客, 而且已经坚持了许多年.

    • 关于学习. 学习知识的最好方式就是立即将其用于实践, 即使你还不知道自己在做什么.

    • 一切始于专注.

    • 产量多只表明生产效率高, 只有完成正确的工作才会成为高效的人.

    • 番茄工作法的基本思路: 你规划出打算一天之内完成的工作, 然后设置一个时长25分钟的定时器, 去完成计划中的第一项任务; 在这25分钟之内, 你只专注于这一项任务, 心无旁骛. 一旦有干扰, 可以用各种方法屏蔽掉干扰, 但是通常你要努力保证自己完全不被打扰. 总之, 你不希望自己的专注的工作状态被打断. 番茄工作法只有被当作估算和评估工作的工具使用时, 才能发挥它的真正威力. 通过跟踪自己在一天内完成了多少个番茄钟, 以及为每天要完成的番茄钟的数量设定目标, 你瞬间有能力去真正评估自己每日工作的努力程度, 也能知晓自己的工作能力.

    • 自我控制是一门自我激励的艺术, 而自我激励的核心则是自我责任感. 如果你想不再依赖别人的影响, 获得稳定, 可预测的结果, 你就要学会对自己负责.

    • 培养习惯: 刷新你的代码

    • 成就我们的恰恰就是那些不断重复做的事情. 因此, 优秀不是一种行为, 而是一种习惯.

    • 为什么你需要跑步. 跑步并不是增强心脏, 增加肺活量的唯一方法, 任何形式的运动都能做到这一点, 但是跑步却是最容易的. 它简单到无论你在哪里只需要出门跑就行了.

    # 曾国藩传

    总结

    • 作者: 张宏杰
    • 阅读时间: 2023年5月
    • 个人评分: 9.0分

    读书笔记

    • 尚拙.
    • 读不懂上一句, 不读下一句. 不读完一本书, 不摸下一本书.
    • 立德立功立言三不朽, 为师为将为相一完人.
    • 三十岁立志做王阳明这样的圣人.
    • 进入翰林院考试都很顺利. 得到了道光皇帝的赏识. 意气风发. 后咸丰继位, 软弱无能. 初入朝廷, 锋芒毕露, 直言批咸丰, 得罪了皇帝, 同时又因为上书说朝廷腐败得罪了大部分官员. 人人唾弃.
    • 当了十三年京官后, 曾国藩对咸丰灰心, 觉得咸丰不行. 后咸丰派曾国藩去江西当主考官, 途中得知母亲病逝, 后回湖南守孝.
    • 太平天国起义崛起, 已经进入湖北. 咸丰命曾国藩在湖南操练, 保卫乡里. 曾国藩极其强势, 变身“曾剃头”, 遇到犯罪行为格杀勿论. 同时做事过于雷厉风行不管他人, 认为其他官员都是废物. 各种行为也影响了其他官员的灰色收入, 动了别人的蛋糕, 因此被多数官员厌恶.
    • 太平天国是整个中国传统文化的敌人, 洪秀全企图毁掉一切书籍, 寺庙, 古文物建筑等. 为了挽救人民与文化, 曾国藩组建湘军.
    • 组建湘军后, 曾国藩并不急于让军队上战场, 而是稳扎稳打的训练. 后来打太平军十分顺利.
    • 朝中有人给咸丰说曾国藩能轻易建立强大的湘军, 如果要谋反也是可能的. 咸丰听信. 后来不断给曾国藩穿小鞋, 并且不让曾国藩拥有实权. 加之咸丰没有谋略, 胡乱指挥, 导致贻误战机.
    • 胡林翼是曾国藩的好友, 战场上互相配合.
    • 湘军的战术: 结硬寨, 打呆仗.
    • 咸丰没有战略头脑, 胡乱指挥, 经常延误战机. 并且一直不给曾国藩实权.
    • 曾国藩与好友胡林翼配合, 取得了不少胜利
    • 曾国藩的弟弟曾国荃是著名将领. 带兵收复南京, 至此太平天国几乎瓦解. 然而曾国荃被朝廷厌恶. 曾国藩深知自己和弟弟一个带兵一个有权, 肯定得不到朝廷的信任. 因此撤裁湘军, 让弟弟回家待命. 这种自剪羽翼的行为让慈禧十分满意.
    • 左宗棠对曾国藩有"瑜亮情结", 因此经常与曾国藩作对. 曾国藩死后, 两人才算和解, 左宗棠自愧不如曾国藩.
    • 李鸿章是曾国藩的弟子. 曾国藩极力培养李鸿章, 后李鸿章接替了曾国藩成为晚清的外交, 军事的实际掌权人.
    • 曾国藩为股权大局, 舍弃自己的名声, 处理了棘手的 "天津教案".

    # Kubernetes实战

    总结

    • 阅读时间: 2024年2月
    • 个人评分: 9.0分

    评语: 这个书有点老了, 貌似是 2018 年的. 很多技术都可能已经过时了, 但是整体来说, 并没有太大的影响, 因为本源和思想是类似的.

    本书的一个宗旨是: 先知道是什么, 再了解为什么及其原理. 这个模式非常好, 前七八章都只是介绍怎么用, 建立认知, 后面才开始升级和介绍原理.
